## 법인의 종류

한국 상법은 회사를 **5가지** 형태로 규정합니다. 사원(출자자)의 책임 범위에 따라 **인적회사**와 **물적회사**로 나뉩니다.

### 인적회사

| 형태 | 구성 | 책임 | 특징 |
|------|------|------|------|
| **합명회사** | 2인 이상 무한책임사원 | 회사 채무에 직접·연대·무한 책임 | 별도 의사기관·감사기관 없음. 사원 각자가 대표권 보유 |
| **합자회사** | 무한책임사원 + 유한책임사원 각 1인 이상 | 무한사원은 무한책임, 유한사원은 출자액 한도 | 현실적으로 매우 드문 형태 |

### 물적회사

| 형태 | 구성 | 책임 | 특징 |
|------|------|------|------|
| **주식회사** | 주주 (1인 이상) | 주식 인수가액 한도 | 가장 보편적. 주식 발행으로 자본조달 용이. 외부감사 대상 가능 |
| **유한회사** | 유한책임사원 (1인 이상) | 출자금액 한도 | 외부감사 의무 없음. 소규모 사업에 적합 |
| **유한책임회사** | 유한책임사원 (1인 이상) | 출자금액 한도 | 내부는 조합 성격. 외국계·사모펀드·벤처에서 선호 |

> **실무 선택**: 가족법인은 대부분 **주식회사** 또는 **유한회사**로 설립합니다. 투자 유치가 목적이면 주식회사, 외부 공시 부담을 줄이려면 유한회사가 유리합니다.

## 가족법인이란

본인이 대표이사를 맡고, **주주 구성원을 가족**으로 하는 법인입니다. 법적으로 별도 유형은 아니지만, 세법상 "특수관계인이 50% 초과 지분을 보유한 법인"으로 분류되어 별도 과세 규정이 적용됩니다.

### 일반적인 구조

```
대표이사: 본인
사내이사: 배우자 또는 자녀
주주: 본인 + 배우자 + 자녀 (지분 분배)
```

- 자본금 10억 원 미만 소규모 법인은 **감사 선임 의무 없음**
- 설립등기 시 주식 미보유 이사 또는 감사 1인이 설립경과 조사보고서 작성 필요

## 개인사업자 vs 법인사업자 비교

### 세율 비교

| 구분 | 과세표준 구간 | 세율 |
|------|-------------|------|
| **개인사업자** (소득세) | 1,400만 원 이하 | 6% |
| | 1,400만~5,000만 원 | 15% |
| | 5,000만~8,800만 원 | 24% |
| | 8,800만~1.5억 원 | 35% |
| | 1.5억~3억 원 | 38% |
| | 3억~5억 원 | 40% |
| | 5억~10억 원 | 42% |
| | 10억 원 초과 | 45% |
| **법인사업자** (법인세, 2026년~) | 2억 원 이하 | 10% |
| | 2억~200억 원 | 20% |
| | 200억~3,000억 원 | 22% |
| | 3,000억 원 초과 | 25% |

**핵심 차이**: 과세표준 2억 원 기준으로 개인사업자는 38% 세율, 법인은 10% 세율 → **약 5,600만 원** 세액 차이가 발생합니다.

### 종합 비교

| 항목 | 개인사업자 | 법인사업자 |
|------|-----------|-----------|
| **설립 절차** | 사업자등록만으로 완료 | 정관 작성 → 공증 → 등기 → 사업자등록 |
| **설립 비용** | 거의 없음 | 50만~100만 원 (등록세, 공증, 법무사 수수료) |
| **세율** | 6~45% (누진) | 10~25% |
| **대표자 급여** | 비용 인정 불가 | 비용 처리 가능 |
| **대표자 퇴직금** | 비용 인정 불가 | 비용 처리 가능 |
| **책임 범위** | 무한책임 | 출자금 한도 유한책임 |
| **대외 신용도** | 상대적으로 낮음 | 높음 (투자유치, 대출에 유리) |
| **회계 처리** | 간편장부 가능 | 복식부기 필수 |
| **외부 감사** | 없음 | 자산 120억 원 이상 시 의무 |

## 법인이 유리한 케이스

### 1. 연소득 5,000만 원 이상인 경우

개인소득세율이 24% 이상 구간에 진입하면, 법인세 10% 대비 절세 효과가 본격화됩니다. **연소득 1억 원 이상**에서는 법인 전환이 거의 확실히 유리합니다.

### 2. 대표자 급여·퇴직금 비용 처리

개인사업자는 대표자 인건비를 경비로 인정받지 못하지만, 법인은 대표이사 급여·상여금·퇴직금 모두 **법인 비용**으로 처리 가능합니다.

예시:
- 대표이사 연봉 6,000만 원 설정 → 법인 과세소득 6,000만 원 감소
- 대표이사는 근로소득으로 분리 과세 (근로소득공제 적용)

### 3. 가족법인의 소득 분산 효과

가족 구성원에게 급여를 지급하면 소득을 분산할 수 있습니다.

| 구조 | 소득 | 적용 세율 |
|------|------|----------|
| 개인사업자 1인 | 2억 원 전체 | 38% |
| 가족법인 (4인 급여 분산) | 각 5,000만 원 | 각 15% |

> **주의**: 실제 근무 실체가 없으면 세무조사 시 부인될 수 있습니다. 가족 구성원이 실제로 업무를 수행해야 합니다.

### 4. 배당을 통한 소득 환류

2026년부터 배당소득 분리과세 제도가 시행됩니다.

| 배당소득 구간 | 분리과세 세율 (지방세 포함) |
|-------------|------------------------|
| 2,000만 원 이하 | 15.4% |
| 2,000만~3억 원 | 22% |
| 3억~50억 원 | 27.5% |
| 50억 원 초과 | 33% |

### 5. 유한책임으로 리스크 차단

사업 실패 시 개인사업자는 개인 재산 전체로 채무를 부담하지만, 법인은 출자금 한도 내에서만 책임을 집니다.

## 법인 유지 관리 비용

### 필수 비용

| 항목 | 비용 (연간 기준) | 비고 |
|------|----------------|------|
| **세무 기장료** | 180만~240만 원 (월 15만~20만 원) | 매출 1억 원 미만 기준 |
| **세무조정료** (법인세 신고) | 100만~200만 원 | 매출 규모에 따라 변동 |
| **4대 보험** | 급여의 약 10% (사업주 부담분) | 대표이사 포함 |
| **등기 변경 비용** | 건당 10만~30만 원 | 임원 변경, 자본금 변경 등 |

### 정기 의무사항

| 시기 | 내용 |
|------|------|
| **매 분기** | 부가가치세 신고 (1·4·7·10월) |
| **매년 3월** | 법인세 신고·납부 (12월 결산 기준) |
| **매년** | 주주총회 개최, 재무제표 승인 |
| **변경 시** | 등기 변경 (임원, 본점, 자본금 등) |

### 개인사업자 유지 비용과 비교

| 항목 | 개인사업자 | 법인 |
|------|-----------|------|
| 세무 대리 비용 | 연 100만~150만 원 | 연 280만~440만 원 |
| 등기 관리 | 없음 | 연 수십만 원 |
| 회계 복잡도 | 간편장부 가능 | 복식부기 필수 |
| **총 관리 비용 차이** | - | **연 150만~300만 원 추가** |

## 절세 전략

### 비용 처리 최적화

법인의 핵심 절세는 **적법한 비용을 최대한 경비 처리**하는 것입니다.

**비용 처리 가능 항목:**
- 임원·직원 급여, 상여금, 퇴직금
- 업무용 차량 유지비 (연 1,500만 원 한도, 업무용으로 입증 필요)
- 접대비 (기본 3,600만 원 + 매출액 비례 추가)
- 복리후생비 (식대, 경조사비, 체육비 등)
- 업무 관련 교육비, 출장비

### 적격증빙 확보

세금계산서, 신용카드 매출전표, 현금영수증 등 **적격증빙**을 반드시 확보해야 합니다. 증빙 없는 비용은 인정되지 않아 법인세가 증가합니다.

### 가수금·가지급금 관리

| 항목 | 문제점 |
|------|--------|
| **가지급금** (법인 → 대표 개인) | 비용 불인정 + 인정이자 과세 |
| **가수금** (대표 개인 → 법인) | 부채비율 증가 → 대출 심사 불리 |

법인 자금과 개인 자금을 철저히 분리 관리해야 합니다.

### 임원 보수 규정 정비

대표이사·임원의 급여와 상여금을 비용 처리하려면 **정관에서 위임한 임원 보수규정**을 사전에 마련해야 합니다. 규정 없이 지급하면 세무조사 시 비용 인정이 거부될 수 있습니다.

## 가족법인 설립 시 유의사항

### 1. 소규모 부동산 임대법인 세율 인상

2025년부터 다음 조건을 **모두** 충족하는 법인은 법인세율 9% → **19%**로 인상됩니다.

- 특수관계인 지분 50% 초과
- 주요 사업이 부동산 임대업 또는 임대 수익이 매출의 50% 초과
- 상시 근로자 5명 미만

### 2. 특정법인 거래를 통한 증여의제

상속세 및 증여세법 제45조의5에 따라, 자녀 주주에게 **1억 원 이상**의 거래 혜택이 발생하면 **증여세가 부과**됩니다.

2025년 개정으로 **자본거래**도 적용 대상에 포함되어, 가족법인 내 자본 거래 시 각별한 주의가 필요합니다.

### 3. 미성년 자녀 지분 보유 리스크

소득이 없는 미성년 자녀에게 과도한 지분을 부여하면 세무당국이 **증여로 간주**하여 과세할 수 있습니다. 자금 출처 소명이 핵심입니다.

### 4. 과밀억제권역 취득세 중과

서울·과천·성남 등 과밀억제권역에 본점을 둔 법인이 **5년 이내** 해당 권역 내 부동산을 취득하면 취득세 **9.4%**가 적용됩니다.

비과밀권역에 형식적으로 본점을 두는 것은 실질과세 원칙에 따라 부인될 수 있습니다.

### 5. 신설 법인 대출 제한

재무제표 이력이 없는 신설 법인은 은행 대출 심사에서 매우 불리합니다. 충분한 자기자본 확보 후 설립하는 것이 현실적입니다.

## 법인 설립이 유리한 케이스 요약

| 케이스 | 이유 | 예상 절세 효과 |
|--------|------|--------------|
| **연소득 1억 원 이상** | 소득세 35% vs 법인세 10% | 연 1,000만 원 이상 |
| **가족 구성원이 실제 근무 가능** | 급여 분산으로 소득세 구간 낮춤 | 연 수백만~수천만 원 |
| **대표자 퇴직금 적립 필요** | 법인만 대표 퇴직금 비용 처리 가능 | 퇴직 시 수천만 원 |
| **사업 리스크가 큰 업종** | 유한책임으로 개인 재산 보호 | 리스크 차단 |
| **부동산·금융 자산 보유** | 법인 명의 취득 시 세율 이점 | 케이스별 상이 |

### 개인사업자가 유리한 케이스

| 케이스 | 이유 |
|--------|------|
| **연소득 5,000만 원 이하** | 세율 차이 미미, 관리 비용이 절세분 상회 |
| **1인 프리랜서** | 행정 부담 대비 이점 적음 |
| **단기 사업** | 법인 설립·청산 비용이 비효율적 |

## 체크리스트: 법인 설립 전 확인사항

- [ ] 연소득 규모가 법인세율 이점을 넘는가? (최소 5,000만 원 이상)
- [ ] 가족 구성원의 실제 근무 가능 여부 확인
- [ ] 연간 관리 비용(300만~400만 원) 대비 절세 효과가 큰가?
- [ ] 자금 출처 소명이 가능한가? (특히 자녀 지분)
- [ ] 세무사·법무사와 사전 상담 완료 여부
- [ ] 임원 보수규정, 정관 정비 계획
- [ ] 과밀억제권역 해당 여부 확인 (부동산 취득 계획 시)
