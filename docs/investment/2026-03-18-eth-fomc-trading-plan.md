### Triple Witching 변동성 해석 (3/20 금)

FOMC 직후(3/19) + Triple Witching(3/20)이 연속으로 겹치는 구간. **리밸런싱 타이밍에 직접 영향**.

| 3/20 시장 상황 | 해석 | 리밸런싱 액션 |
|---------------|------|-------------|
| **FOMC 방향 유지 + 거래량 감소** | 만기 소화 순조, 방향 확정 | 즉시 리밸런싱 실행 (초과분 매도) |
| **FOMC 반대 방향 스파이크** | 만기 정리 물량의 일시적 역류 — 노이즈 | 당일 거래 금지, **3/21~22 주말 대기** 후 실행 |
| **FOMC 방향 + 급격 가속** (숏/롱 스퀴즈) | 청산 캐스케이드 — <span>$</span>2,400↑ 시 숏 <span>$</span>10.6억, <span>$</span>2,210↓ 시 롱 <span>$</span>13.9억 | 스퀴즈 종료 대기 후 실행 (보통 4~8시간 내 소진) |
| **양방향 난타** (상하 5%+ 진폭) | 전형적 Triple Witching 패턴 — 방향 없음 | **월요일(3/23)까지 대기**, 변동성 소진 후 리밸런싱 |

**핵심 원칙**: Triple Witching 당일(3/20)은 **관찰만**. 리밸런싱은 변동성 소진 확인 후 **3/21~23에 실행**.
