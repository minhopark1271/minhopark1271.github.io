# 주식 발행 형태 완전 비교: 보통주부터 RCPS까지
{:.no_toc}

회사가 발행하는 주식은 단순한 보통주만 있는 것이 아닙니다. 의결권, 배당 우선권, 상환권, 전환권 등 다양한 조건이 결합되어 여러 종류의 주식이 만들어집니다. 이 글에서는 모든 주식 발행 형태를 체계적으로 정리하고, 특히 스타트업 투자에서 가장 많이 사용되는 RCPS(상환전환우선주)를 중심으로 각 유형의 차이를 비교합니다.

## 주식 발행 형태 전체 구조

주식 발행 형태는 크게 **주식형**과 **채권형(메자닌)**으로 나뉘며, 주식형 안에서 부여되는 권리 조합에 따라 세분화됩니다.

```
주식 발행 형태
├── 주식형
│   ├── 보통주 (Common Stock)
│   └── 우선주 (Preferred Stock)
│       ├── (순수) 우선주
│       ├── 전환우선주 (CPS)
│       ├── 상환우선주 (RPS)
│       └── 상환전환우선주 (RCPS)  ← 스타트업 투자 핵심
│
└── 메자닌 (채권+주식 혼합)
    ├── 전환사채 (CB)
    └── 신주인수권부사채 (BW)
```

---

## 보통주 (Common Stock)

회사가 발행하는 **가장 기본적인 주식** 형태입니다. 주식시장에서 일반 투자자가 거래하는 주식이 바로 보통주입니다.

| 항목 | 내용 |
|------|------|
| **의결권** | 1주 1의결권 (경영 참여 가능) |
| **배당** | 받을 수 있으나 보장되지 않음 |
| **잔여재산 분배** | 우선주 이후 후순위 |
| **상환권** | 없음 |
| **전환권** | 없음 |

**활용**: 일반적인 주식 투자, 엔젤 투자 초기 단계, 창업자 지분

**특징**: 회사가 성장하면 주가 상승으로 수익을 얻지만, 배당이나 청산 시에는 우선주보다 불리합니다. 별도의 상환 의무가 없어 회사 입장에서 자금 부담이 가장 적습니다.

---

## 우선주 (Preferred Stock)

보통주 대비 **배당과 잔여재산 분배에서 우선권**을 가지는 주식입니다. 대신 의결권이 제한되거나 없습니다.

| 항목 | 내용 |
|------|------|
| **의결권** | 없거나 제한적 |
| **배당** | 보통주보다 우선 지급 (배당률 정관에 명시) |
| **잔여재산 분배** | 보통주보다 우선 분배 |
| **상환권** | 없음 |
| **전환권** | 없음 |

**활용**: 안정적 배당을 추구하는 투자자, 대기업 배당주 투자

**특징**: 경영에 참여하지 않으면서 안정적인 수익을 원하는 투자자에게 적합합니다. 회사 입장에서는 의결권을 부여하지 않아 경영권 방어에 유리합니다.

---

## 전환우선주 (CPS, Convertible Preferred Stock)

우선주에 **전환권**이 추가된 형태입니다. 일정 조건 하에서 보통주로 전환할 수 있습니다.

| 항목 | 내용 |
|------|------|
| **의결권** | 전환 전: 없거나 제한 / 전환 후: 보통주와 동일 |
| **배당** | 우선 배당 |
| **잔여재산 분배** | 우선 분배 |
| **상환권** | 없음 |
| **전환권** | **있음** — 보통주로 전환 가능 |

**전환권의 핵심**:
- 우선주 1주를 보통주 N주로 전환 (전환비율은 계약으로 결정)
- 회사가 성장하면 보통주로 전환하여 시세 차익 실현
- 전환 후에는 의결권도 획득

**활용**: 회사 성장에 베팅하면서도 배당 우선권을 유지하려는 투자자

**제한**: 전체 우선주 발행의 25% 이내로 발행 가능 (상법상 제한)

---

## 상환우선주 (RPS, Redeemable Preferred Stock)

우선주에 **상환권**이 추가된 형태입니다. 일정 기간 후 회사에 주식을 되팔아 투자금을 회수할 수 있습니다.

| 항목 | 내용 |
|------|------|
| **의결권** | 없거나 제한적 |
| **배당** | 우선 배당 |
| **잔여재산 분배** | 우선 분배 |
| **상환권** | **있음** — 투자금 + 이자 회수 가능 |
| **전환권** | 없음 |

**상환권의 핵심**:
- 정해진 기간(보통 3~5년) 후 주식을 발행가 + 이자로 상환 청구
- **중요**: 회사의 이익잉여금(배당 가능 이익)으로만 상환 가능
- 이익잉여금이 부족하면 실질적으로 상환 불가

**활용**: 회사 전망이 불확실할 때 투자금 회수 안전장치로 활용

**한계**: 청산 시 변제 순위가 채권보다 낮아, 실제 투자금 회수 보장이 완전하지는 않습니다.

---

## 상환전환우선주 (RCPS, Redeemable Convertible Preferred Stock)

**상환권 + 전환권 + 우선권**을 모두 가진 주식으로, 현재 스타트업 투자에서 **가장 많이 사용**되는 투자 방식입니다.

| 항목 | 내용 |
|------|------|
| **의결권** | 비상장 벤처에서는 의결권 부여가 일반적 |
| **배당** | 우선 배당 |
| **잔여재산 분배** | 우선 분배 |
| **상환권** | **있음** |
| **전환권** | **있음** |

### 투자자에게 유리한 이유

RCPS는 투자자에게 **두 가지 선택지**를 동시에 제공합니다:

- **회사가 잘 될 때** → 전환권 행사 → 보통주로 전환하여 시세 차익 극대화
- **회사가 안 될 때** → 상환권 행사 → 투자금 + 이자 회수 (원금 방어)

즉, "밑져도 본전(상환), 잘 되면 대박(전환)"이라는 구조입니다.

### 발행 시 정관 필수 조항

RCPS를 발행하려면 정관에 다음 두 가지가 반드시 포함되어야 합니다:

1. **종류주식 발행 근거 조항** — 우선주 발행이 가능하다는 근거
2. **제3자배정 발행 근거 조항** — 기존 주주 외에게 주식을 발행할 수 있는 근거

정관에 해당 조항이 없으면 정관 개정 후 유상증자 등기를 진행해야 합니다.

### 주요 계약 조건

| 조건 | 설명 |
|------|------|
| **전환비율** | 우선주 1주당 보통주 몇 주로 전환할지 (예: 1:1.1) |
| **전환가격 조정(리픽싱)** | 다운라운드 시 전환비율을 유리하게 재조정 |
| **상환가액** | 발행가 + 약정 이자율 (보통 연 1~8%) |
| **상환 청구 기간** | 통상 발행 후 3~5년 이후부터 가능 |
| **배당률** | 우선 배당 비율 (정관에 명시) |

### 회계 처리상 주의점

RCPS는 상환권이 있어 부채의 성격을 동시에 가집니다. K-IFRS(한국채택국제회계기준) 적용 시 자본이 아닌 **부채로 분류**될 수 있어, 재무제표상 부채비율이 높아지는 효과가 있습니다.

## 메자닌 증권: 전환사채(CB)와 신주인수권부사채(BW)

주식과 채권의 중간 성격을 가진 **메자닌(Mezzanine) 증권**도 자금 조달에 자주 사용됩니다.

### 전환사채 (CB, Convertible Bond)

**채권 + 전환권**의 결합입니다.

| 항목 | 내용 |
|------|------|
| **기본 성격** | 채권 (확정 이자 수취 + 만기 상환) |
| **전환권** | 일정 조건에서 주식으로 전환 가능 |
| **전환 시점** | 발행일로부터 보통 3개월 이후 |
| **상환 의무** | 만기 시 원리금 상환 의무 (RCPS보다 부담 큼) |

**RCPS와의 차이**: CB는 채권이므로 **만기 시 상환 의무가 법적으로 확정**되어 있습니다. 반면 RCPS는 이익잉여금이 있어야만 상환이 가능하므로 회사 부담이 상대적으로 적습니다.

### 신주인수권부사채 (BW, Bond with Warrant)

**채권 + 신주인수권**의 결합입니다.

| 항목 | 내용 |
|------|------|
| **기본 성격** | 채권 (정기 이자 + 만기 상환) |
| **신주인수권** | 미래에 새 주식을 정해진 가격으로 인수할 권리 |
| **CB와의 차이** | 채권이 그대로 유지 + 별도로 신주 인수 가능 |

**핵심 차이**: CB는 전환 시 채권이 소멸하지만, BW는 채권이 유지되면서 추가로 주식을 인수할 수 있습니다.

## 전체 비교표

| 항목 | 보통주 | 우선주 | CPS | RPS | **RCPS** | CB | BW |
|------|--------|--------|-----|-----|----------|----|----|
| **의결권** | O | X | X→O | X | O* | X | X |
| **배당 우선** | X | O | O | O | O | 이자 | 이자 |
| **잔여재산 우선** | X | O | O | O | O | 채권자 | 채권자 |
| **전환권** | X | X | O | X | **O** | O | 별도 |
| **상환권** | X | X | X | O | **O** | 만기상환 | 만기상환 |
| **회사 부담** | 최소 | 배당 | 배당 | 상환 | 중간 | 높음 | 높음 |
| **투자자 유연성** | 낮음 | 낮음 | 중간 | 중간 | **높음** | 중간 | 중간 |
| **주요 활용** | 일반투자 | 배당투자 | 성장베팅 | 안전장치 | **VC투자** | 자금조달 | 자금조달 |

\* 비상장 벤처 기업에서는 RCPS에도 의결권을 부여하는 것이 일반적

## RCPS가 스타트업 투자의 표준이 된 이유

스타트업 VC 투자에서 RCPS가 사실상 표준으로 자리 잡은 이유를 정리하면 다음과 같습니다.

**투자자(VC) 관점**:
- 상환권으로 **하방 리스크 제한** (원금 + 이자 회수 가능)
- 전환권으로 **상방 수익 극대화** (보통주 전환 후 IPO/M&A 시 차익 실현)
- 배당 우선권과 잔여재산 분배 우선권으로 **이중 안전장치**
- 상황에 따라 최적의 회수 전략 선택 가능

**창업자(회사) 관점**:
- CB/BW 대비 **상환 부담이 적음** (이익잉여금이 있을 때만 상환)
- 초기에 현금 유출 없이 자금 조달 가능
- 전환 시 부채 감소, 자본 증가로 **재무구조 개선**

실제로 VC 투자에서 상환권이 행사되는 비율은 약 6% 수준(2020년 기준)에 불과합니다. 대부분의 경우 IPO나 M&A를 통해 전환권을 행사하여 회수하므로, 상환권은 실질적인 안전장치 역할을 합니다.

## 투자 시 체크포인트

RCPS로 투자를 받거나 투자할 때 반드시 확인해야 할 핵심 조건입니다.

1. **전환비율과 리픽싱 조건**: 후속 투자(다운라운드) 시 전환비율이 어떻게 조정되는지
2. **상환 조건**: 상환 청구 가능 시점, 상환가액 산정 방식, 이자율
3. **우선 배당률**: 배당 기준과 우선 배당 비율
4. **의결권 유무**: 비상장사에서는 의결권 부여 여부가 경영권에 직결
5. **독소 조항**: 동반매도청구권(Drag-along), 우선매수권(ROFR), 희석방지(Anti-dilution) 등

---

## 요약

- **보통주**: 가장 기본, 의결권 O, 배당·청산 후순위
- **우선주**: 의결권 X, 배당·청산 우선권
- **전환우선주(CPS)**: 우선주 + 보통주 전환권
- **상환우선주(RPS)**: 우선주 + 투자금 상환권
- **상환전환우선주(RCPS)**: 우선주 + 전환권 + 상환권 → 스타트업 VC 투자의 핵심
- **전환사채(CB)**: 채권 + 전환권, 만기 상환 의무
- **신주인수권부사채(BW)**: 채권 + 신주인수권, 채권 유지

투자 형태를 선택할 때는 **투자자의 리스크 허용 범위**, **회사의 자금 부담 능력**, **향후 회수 전략**을 종합적으로 고려해야 합니다.
