# RSI + MA 배열 활용 장세 분류 & 매매 케이스
{:.no_toc}

## 공통 관측 프레임워크

매매 판단 전, 아래 두 축을 항상 동시에 확인합니다.

### MA 배열 체크포인트

| 항목 | 확인 내용 |
|------|-----------|
| **정렬** | 단기 > 중기 > 장기 (상승 추세) / 반대 (하락 추세) |
| **기울기** | 장기 MA가 우상향인지 → "진짜 추세"의 핵심 조건 |
| **이격** | 가격이 단기 MA에서 얼마나 떨어졌는지 → 과열/되돌림 위험 판단 |

### RSI 체크포인트

| 항목 | 확인 내용 |
|------|-----------|
| **50선 위치** | 상승 압력 우위 (50 위) vs 하락 압력 우위 (50 아래) |
| **추세장 밴드** | 상승장에선 RSI 바닥이 주로 40 근처, 하락장에선 천장이 60 근처 |
| **다이버전스/실패스윙** | 모멘텀 구조 변화의 조기 신호 |

> **핵심 원칙**: MA는 "어느 방향의 장인지(레짐)"를 알려주고, RSI는 "그 장 안에서 지금 어느 위치인지(타이밍)"를 알려줍니다. 둘을 분리해서 쓰면 절반짜리 정보가 됩니다.

## 케이스별 기대값 순위 요약

승률과 손익비(R:R) 기준으로 유리한 구간 순서입니다.

| 순위 | 케이스 | 예상 승률 | 손익비 | 핵심 근거 |
|------|--------|-----------|--------|-----------|
| **1위** | Case 1 — 정상 상승 추세 | 60~70% | 2:1 이상 | 추세 추종 중 가장 일관성 높음. 추적 손절로 수익 극대화 가능 |
| **2위** | Case 4 — 정상 하락 추세 | 55~65% | 2:1 이상 | Case 1과 대칭 구조. 암호화폐는 급반등 위험으로 SL 관리가 소폭 까다로움 |
| **3위** | Case 7 — 돌파 장 | 50~60% | 3:1 이상 | 승률은 중간이지만 레인지 높이 = 측정 가능한 목표로 R:R 우수 |
| **4위** | Case 6 — 레인지/박스 | 60~70% | 1:1~1.5:1 | 극단 구간 승률은 높지만 박스 폭이 목표의 한계 — R:R이 구조적으로 제한됨 |
| **5위** | Case 5-B — 진짜 바닥 전환 | 40~55% | 3:1 이상 | R:R은 최상급이나 5-A(함정)와의 구분이 어려워 실전 승률이 낮아짐 |
| **6위** | Case 3 — 상승→전환 초기 | 35~45% | 2:1 이상 | 이론상 R:R은 좋지만 추세 재개 시 손절 빈번. "SL이 생명" 구간 |
| **7위** | Case 2 — 과열/피로 | — | 불리 | 신규 진입 금지 구간. 기존 보유자의 분할 익절 타이밍으로만 유효 |

```
EV(기대값) 순서:

Case 1 ≈ Case 4  >  Case 7  >  Case 6  >  Case 5-B  >  Case 3  >>  Case 2
   (추세 추종)       (돌파)      (레인지)    (반전)         (전환)     (진입 금지)
```

## 케이스 1. 정상 상승 추세

> **"추세 추종의 교과서" — 가장 높은 확률의 롱 구간**

| 구분 | 조건 |
|------|------|
| MA | 단기 > 중기 > 장기 정렬 + 장기 MA 우상향 |
| 가격 | 단기/중기 MA 위에서 눌림 후 재상승 |
| RSI | 50선 위 유지, 조정 때 40~50에서 지지 후 반등 |

### 진입 시점

- **1차 진입**: 눌림이 중기 MA(20 or 60) 부근에서 멈추고 캔들이 양전환 확인
- **2차 추가**: 고점 돌파 + RSI가 50~60 이상 재확인 ("엔진 재점화" 신호)

### TP/SL

| 구분 | 조건 |
|------|------|
| **SL** | 눌림 저점 이탈 *또는* 중기 MA 종가 이탈 + RSI 50 하향 **동시** 충족 |
| **TP 1** | 직전 스윙 고점 부근 분할 청산 (30~50%) |
| **TP 2** | RSI 70~80에서 꺾이는데 가격이 신고점 못 내면 (약한 다이버전스) 추가 정리 |
| **TP 3** | 추세 유지형이면 중기 MA 기반 **추적 손절**이 수익 극대화에 유리 |

## 케이스 2. 상승 추세의 과열/피로

> **"방향은 살아있지만, 기대수익/리스크 비율이 나빠진 상태"**

| 구분 | 조건 |
|------|------|
| MA | 정렬은 상승이지만, 가격이 단기 MA에서 과도하게 이격 |
| RSI | 75~85 고레벨 반복 + 베어리시 다이버전스 출현 (가격 HH, RSI LH) |

**중요 해석**: 추세가 꺾인 "확정"이 아니라, 지속 확률은 있지만 **신규 진입의 매력이 크게 줄어든 상태**입니다.

### 실전 운영

| 포지션 | 대응 |
|--------|------|
| **신규 진입** | 보수적 대기 (눌림 기다리기) — 추격 매수 금지 구간 |
| **기존 보유자** | 이격/RSI 과열 구간에서 분할 익절 시작 |
| **SL 조정** | 단기 MA 이탈이 아닌 **중기 MA 이탈**을 트리거로 (과열 구간은 단기 흔들림이 극심) |

## 케이스 3. 상승 추세 → 분배/전환 초기 신호

> **"경고에서 확인으로 — 단계적으로 읽어야 실수를 줄인다"**

전환은 한 번에 오지 않습니다. **4단계로 순서대로 확인**해야 합니다.

| 단계 | 신호 |
|------|------|
| 1단계 | RSI 베어리시 다이버전스 / 실패스윙 조짐 |
| 2단계 | 가격 고점 돌파 실패 (더블탑 유사 구조) |
| 3단계 | 단기 MA 기울기 둔화 → 단기/중기 데드크로스 가능성 |
| 4단계 | **RSI 50선 하향 이탈** ← "전환 확인"의 가장 중요한 신호 |

### 진입 시점

- **숏/헤지**: RSI 실패스윙 완성 + 50선 하향 + 단기 < 중기 첫 확인 시
- **기존 롱 보유자**: "새 롱 중단 + 보유분 리스크 축소"가 먼저. 바로 숏 전환은 성급함

### TP/SL

전환 매매에서는 **SL이 생명**입니다.

| 구분 | 조건 |
|------|------|
| **SL** | 마지막 구조적 고점 상향 돌파 시 즉시 손절 |
| **TP** | 장기 MA(120/200) 또는 이전 박스 하단까지 단계적 목표 |

## 케이스 4. 정상 하락 추세

> **"추세 추종 vs 역추세 반등매매를 반드시 구분해야 한다"**

| 구분 | 조건 |
|------|------|
| MA | 단기 < 중기 < 장기 정렬 + 장기 MA 하향 |
| RSI | 50선 아래 유지, 반등 때 60 근처에서 막히고 재하락 |

### 전략 분기

| 전략 | 진입 조건 |
|------|-----------|
| **추세 추종 (숏/현금 비중 확대)** | 반등이 중기 MA에서 막힐 때 진입 |
| **역추세 바닥잡기** | RSI 30 이하만으로는 **절대 부족**. 케이스 5-B 조건 모두 충족 시에만 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **숏 TP** | 이전 저점 / RSI 20~30 과매도 구간에서 분할 청산 |
| **숏 SL** | "중기 MA 회복 + RSI 50 회복" 동시 조건 — **레짐 변화 기반 청산** |

## 케이스 5. 하락 추세 속 반등 — 함정 vs 바닥 전환

> **"가장 헷갈리는 구간 — RSI+MA 조합이 가장 빛나는 케이스"**

### 5-A. 데드캣 바운스 (반등 함정)

| 항목 | 상태 |
|------|------|
| MA | 장기 하향, 배열 여전히 하락 정렬 |
| RSI | 반등해도 50을 못 넘고 55~60에서 꺾임 |
| 가격 | 중기 MA(60) 아래에서 반등 종료 |

→ **"추세 내 조정" 가능성 높음**. 롱 진입 자제.

### 5-B. 진짜 바닥 전환 후보

| 항목 | 상태 |
|------|------|
| RSI | 불리시 다이버전스 (가격 LL, RSI HL) + 이후 **50선 상향 회복** |
| MA | 단기 MA가 평평→상향 전환, 가격이 중기 MA를 회복하고 지지 |
| 배열 | 완전 전환 전이라도 **"단기 > 중기"가 먼저** 발생 |

→ 이 조합이 나오면 **"진짜 전환 후보"로 격상**해서 관찰합니다.

### TP/SL (전환 롱은 SL을 타이트하게)

| 구분 | 조건 |
|------|------|
| **SL** | 다이버전스를 만든 마지막 저점 이탈 시 즉시 손절 |
| **TP 1차** | 중기/장기 MA 구간 (첫 저항) |
| **TP 이후** | MA 배열이 상승 전환되면 케이스 1 (추세 추종)으로 운영 방식 변경 |

## 케이스 6. 레인지/박스 장

> **"RSI 단독으로 가장 잘 먹히는 구간 — MA 크로스는 잡음으로 취급"**

| 구분 | 조건 |
|------|------|
| MA | 장기 MA 평평, 단기/중기 MA가 서로 엉킴 (배열이 자주 뒤집힘) |
| RSI | 40~60 또는 30~70 사이 왕복 반복 |

### 진입 시점

| 방향 | 조건 |
|------|------|
| **박스 하단 매수** | RSI 30~40 + 가격이 단기 MA 회복 확인 |
| **박스 상단 매도** | RSI 60~70 + 가격이 단기 MA 이탈 확인 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **TP** | 박스 반대편 + RSI가 50 도달 시 일부 청산 (욕심 금지) |
| **SL** | 박스 종가 이탈 + RSI가 50을 **한 방향으로 지속 유지** (레짐 변화 신호) |

> 레인지 장에서 MA 크로스 신호는 잡음이 많습니다. **RSI 평균 회귀 전략이 우선**입니다.

## 케이스 7. 돌파 장 (레인지 → 추세 레짐 전환)

> **"레인지 탈출 — 리테스트 확인이 핵심"**

### 돌파 판별 조건

| 단계 | 상태 |
|------|------|
| **돌파 전** | MA 평평/엉킴 + RSI 중립권 (45~55) |
| **돌파 발생** | 가격이 중기/장기 MA 위에 안착 + RSI가 50 돌파 후 60~70 테스트 |
| **결정적 확인** | 눌림(리테스트)에서 **RSI가 40~50에서 지지** ← 핵심 |

### 진입 시점

| 스타일 | 조건 |
|--------|------|
| **보수적** | 돌파 후 첫 눌림 (리테스트)에서 중기 MA 지지 확인 후 진입 |
| **공격적** | 돌파 캔들 + 거래량 조건 충족 시 즉시 진입 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **SL** | 리테스트 실패 (돌파 레벨 하향 이탈) + RSI 50 하향 동시 |
| **TP 1차** | 직전 레인지 높이만큼 측정 목표 |
| **TP 이후** | 돌파 확인되면 케이스 1 (정상 상승 추세) 운영으로 전환 |

## TP/SL 규칙화 요약

| 방식 | 적합 케이스 | 설명 |
|------|------------|------|
| **구조 기반** (스윙 저점/고점 + RSI 50 확인) | 케이스 1, 3, 5 | 추세 매매에서 가장 일관적 |
| **MA 기반** (중기 MA 종가 이탈 + RSI 밴드) | 케이스 1, 4, 7 | 휩소 감소, 추세 유지 조건화 |
| **과열 기반** (RSI 70/30 + 이격) | 케이스 2, 6 | **TP에만 유리**, SL 단독 사용 비추 |

## 레짐 분류 빠른 판단 트리

```
장기 MA 방향은?
├─ 우상향 → 상승 레짐
│       RSI 50 위? → 케이스 1 (추세 추종)
│       RSI 과열 + 다이버전스? → 케이스 2/3 (과열/전환 주의)
│
├─ 하향 → 하락 레짐
│       반등 RSI 50 못 넘음? → 케이스 5-A (함정)
│       불리시 다이버전스 + RSI 50 회복? → 케이스 5-B (전환 후보)
│
└─ 횡보/평평 → 레인지 레짐
        RSI 왕복? → 케이스 6 (레인지 매매)
        RSI 50 돌파 + MA 정렬 시작? → 케이스 7 (돌파 대기)
```

## 데이트레이딩 타임프레임 구성 상세 (v3)

위 7가지 케이스를 크립토 데이트레이딩에 적용할 때의 타임프레임 구성입니다. BTC 일봉 757캔들(2024-02~2026-02) 실증 분석을 반영한 v3 구조입니다.

> **v2 → v3 주요 변경점**
> - 단일 4H EMA(21)/EMA(55) → **비대칭 이중 필터**
>   - 상승 레짐: Daily EMA(21)/EMA(55) (느린 필터, R:R 평균 5.84x, 수익률 평균 +7.0%)
>   - 하락 레짐: 4H EMA(21)/EMA(55) = Daily EMA(4)/EMA(9) (빠른 필터, 음수비율 63%)
> - 필터 선정 기준을 EV(기대값=평균) 중심으로 전환 (BTC 일봉 757캔들 실증 분석)
> - 하락 레짐 운용 지침 현실화 (BTC bull bias 반영, 사이즈 축소 기본)
> - 보조 필터 1H EMA(200) → 4H EMA(84) 변경 (Daily EMA(14) 등가, 두 레짐 필터 대역 중간으로 조정)
> - 기존 OI/펀딩비 보조 필터, 30m 실행 레이어, 리스크 관리 유지

### 기본 구조

```
[레짐 판단]
  상승 레짐 필터  (Daily EMA21/55)    → 느린 필터, R:R 평균 5.84x, 수익률 +7.0%, 평균 13.1일
  하락 레짐 필터  (4H EMA21/55)       → 빠른 필터, 음수비율 63%, R:R 평균 3.20x, 평균 4.2일
  레인지 판단     (양 필터 모두 SIDE)  → 방향성 없음
     ↓
  [4H EMA(84) 보조 필터]             → 매크로 방향 일치 확인 (Daily EMA14 등가, v2 대비 조정)
     ↓
  타이밍          (4H RSI14)          → 진입 위치 판단 (기존 유지)
     ↓
  실행            (30m 가격 구조)      → 정확한 진입가 (기존 유지)
```

### 비대칭 이중 필터 도입 근거

**α 변환 공식**: `α_daily = 1 - (1 - α_4h)^6`
- 4H EMA(21) → Daily EMA(≈4), 4H EMA(55) → Daily EMA(≈9)

BTC 일봉 757캔들(2024-02~2026-02) 실증 분석([Bitcoin Optimal Regime Index](../../../obsidian_vault/private/projects/trading_bot/docs/bitcoin-optimal-regime-index))에서 4가지 필터를 비교한 결과:

### 레이어 1. 레짐 + 타이밍 판단

v3에서는 상승과 하락 레짐에 **서로 다른 속도의 MA 필터**를 적용합니다. 단일 MA 쌍으로 양방향을 커버하면 한쪽을 필연적으로 희생하기 때문입니다 (비대칭 필터 도입 근거는 위 섹션 참조).

이 비대칭 설계는 BTC에서 역사적으로 반복 관찰된 가격 패턴에 근거합니다. **상승은 완만하고 장기적**으로 진행되는 경향이 있어, 빠른 필터를 적용하면 중간 눌림마다 레짐이 해제되어 대형 랠리 구간을 놓치게 됩니다. 반면 **하락은 급격하고 단기적**으로 진행된 후 강한 반등이 빠르게 뒤따르는 경향이 있어, 느린 필터를 적용하면 반등이 시작된 후에도 하락 레짐이 유지되어 불필요한 리스크에 노출됩니다.

> **설계 전제**: 이 비대칭 구조는 과거 관찰에 근거하며 미래를 보장하지 않습니다. 상승이 단기·급등형으로 변하거나 하락이 장기·완만형으로 전환되는 패턴 변화가 누적 관찰될 경우, 필터 속도의 비대칭 방향 자체를 재검토해야 합니다.

#### 상승 레짐 판단: Daily EMA(21) / EMA(55)

**지표 선정 근거**:

| 지표 | 시간 환산 | 역할 |
|------|----------|------|
| Daily EMA(21) | 21일 ≈ 3주 | 중기 추세. 눌림/반등의 1차 지지/저항 |
| Daily EMA(55) | 55일 ≈ 11주 | 장기 추세. 레짐 방향의 기준선 |

- 실증 근거 (EV 관점): UP R:R 평균 5.84x, 수익률 평균 +7.0% (4가지 중 압도적 1위). 중앙값도 R:R 2.49x, 양수비율 67%로 1위. 평균 13.1일 → RSI 전략 운용 충분
- v2의 4H EMA(21)/EMA(55)로 판단 시 R:R 평균 3.04x/중앙값 1.06x, 수익률 +3.6%, 4.5일 → EV 절반 수준이며 구간도 짧음
- "큰 구간을 놓치지 않는" 전략적 의미: EMA21/55 UP의 평균 최대상승 +11.1%는 대형 랠리 구간 포착 능력이 가장 높음

**상승 레짐 조건** (Strict 3-State):

```
close > Daily_EMA(21) > Daily_EMA(55)
+ Daily_EMA(55) 기울기 > 0
```

- 허가: **롱 진입만 탐색**
- 금지: 숏 진입 (RSI 과매수여도 숏 신규 진입 금지)
- 특징: 평균 13.1일 구간 — 4H RSI가 눌림→반등 사이클을 1~2회 완성할 수 있는 시간

#### 하락 레짐 판단: 4H EMA(21) / EMA(55)

**지표 선정 근거**:

| 지표 | 시간 환산 | 역할 |
|------|----------|------|
| 4H EMA(21) | 84시간 ≈ 3.5일 | 단기 추세. 반등/눌림의 1차 저항/지지 |
| 4H EMA(55) | 220시간 ≈ 9.2일 | 중기 추세. 하락 레짐 방향의 기준선 |

- 4H EMA(21)/EMA(55) = Daily EMA(4)/EMA(9) 등가 (α 변환)
- 실증 근거: DOWN 음수비율 63% (4가지 중 유일하게 실용적), R:R 평균 3.20x, 수익률 평균 -2.8%
- Daily EMA(21)/EMA(55) DOWN은 음수비율 38% → 62%의 DOWN 구간에서 오히려 상승하여 사용 불가
- 빠른 필터의 단점: 구간이 짧고(평균 4.2일) 개별 수익 여력이 작음 → 비용 효율 확인 필요
- DOWN 필터 선택 근거: "음수비율(방향 적중률)이 EV에 선행합니다. 평균 R:R이 높아도 37%의 구간이 비용(수수료+슬리피지)을 지불한 후 역방향 이동하면 EV가 크게 훼손됩니다."

**하락 레짐 조건** (Strict 3-State):

```
close < 4H_EMA(21) < 4H_EMA(55)
+ 4H_EMA(55) 기울기 < 0
```

- 허가: **숏 진입만 탐색** (현물이면 신규 매수 금지)
- 금지: 롱 진입

> **BTC bull bias 주의사항 (역사적 패턴 기반)**: BTC는 역사적으로 상승 편향이 강했으며, 이 결과 가장 빠른 필터(EMA4/9)조차 음수비율이 63%에 불과합니다. 이는 과거 반복 관찰된 패턴이며, 시장 구조 변화(기관 보유 비중, 매크로 상관관계 등)에 따라 향후 달라질 수 있습니다. **이 패턴이 지속되는 한**, 하락 레짐 진입은 상승 레짐보다 구조적으로 불리하므로 **포지션 사이즈 기본 축소(50~75%)를 적용**합니다. DOWN 레짐 필터는 적극적 숏보다 **롱 회피 용도**로 더 현실적입니다.
>
> **패턴 변화 감지**: 하락 레짐 평균 지속 기간이 역사적 평균(4.2일)보다 현저히 길어지거나, DOWN 음수비율이 롤링 기준으로 지속적으로 75% 이상으로 상승하는 경우, bull bias 가정을 재검토하고 숏 운용 방침을 재조정합니다.

#### 레인지 레짐 판단

**레인지 조건**: 상승 레짐(Daily 기준)도 하락 레짐(4H 기준)도 아닌 상태

| 조건 | 설명 |
|------|------|
| Daily EMA(21)/EMA(55) 정렬 미충족 | UP이 아님 |
| AND 4H EMA(21)/EMA(55) 정렬 미충족 | DOWN이 아님 |

- 허가: 박스 하단 롱 (사이즈 50% 축소) / 박스 상단 숏 (사이즈 30% 축소). 상단 숏은 bull bias 환경에서 상방 돌파 위험이 하방 이탈보다 역사적으로 높은 경향이 있어 추가 축소
- 권장: **거래 자제** (방향성 미확정, EMA 레짐 필터가 사실상 무용지물인 시간대)

> **SIDE 구간 진입 타이밍 인사이트** (실증): 실증 분석에서 SIDE 구간의 종가 기준 선행 이동은 모든 필터에서 0%입니다. 레짐 신호 발생 시 즉시 진입해도 "늦는" 손해는 없습니다. 다만 고저 기준 선행 이동(27~38%)은 SIDE 구간 자체의 변동성이며 구조적으로 제거 불가능합니다.

#### 전환 감시

비대칭 필터에 따른 전환 규칙:

| 전환 | 조건 | 설명 |
|------|------|------|
| 상승 → SIDE | Daily EMA(21)이 Daily EMA(55) 하향 크로스 | 상승 레짐 해제 |
| SIDE → 하락 | 4H EMA(21)이 4H EMA(55) 하향 크로스 + close < 4H EMA(21) | 하락 레짐 활성화 |
| 하락 → SIDE | 4H EMA(21)이 4H EMA(55) 상향 크로스 | 하락 레짐 해제 |
| SIDE → 상승 | Daily EMA(21)이 Daily EMA(55) 상향 크로스 + close > Daily EMA(21) | 상승 레짐 활성화 |

- 두 필터의 속도 차이로 "상승 해제 → 하락 활성화" 사이에 **SIDE 버퍼 구간이 자연 발생**
- 크로스 발생 후 12시간 이내는 방향 미확정 → 거래량을 줄이거나 **관망**
- **SIDE → 하락 전환 추가 확인**: 4H 기준 하향 크로스는 단순 조정에서도 발생하는 경향이 있습니다. 4H 봉 2개 이상 EMA(21) 하방 마감이 확인되거나 거래량이 동반될 경우 신뢰도가 높아집니다. 단일 크로스 직후에는 사이즈를 절반으로 줄이거나 관망을 우선합니다.
- **급락(-5% 이상) 예외**: 하락 레짐 즉시 활성화 대신 **1개 4H 봉 추가 확인** 후 전환 (급락 후 V자 반등이 빈번한 경향 있음). 단, -10% 이상의 극단적 급락은 즉시 전환으로 간주
- 급등(+5% 이상) 시 12시간 확인 없이 상승 레짐 전환으로 간주 가능

#### 레짐 신뢰도 보조 필터: 4H EMA(84) + OI + 펀딩비

VWMA가 아닌, 조작 불가능한 크립토 고유 데이터를 사용합니다.

> **VWMA를 쓰지 않는 이유**: 크립토 거래소 거래량의 30~70%는 워시트레이딩+아비트라지+HFT 마켓메이커 활동. 펀딩비 정산(8시간마다) 전후 거래량이 3~5배 급증하며, 이는 방향성과 무관한 순수 아비트라지 활동. VWMA는 "노이즈의 이동평균"을 계산하는 셈이 됨

**4H EMA(84) — 매크로 방향 확인**

4H EMA(84) ≈ 336시간 ≈ 14일 룩백 (Daily EMA(14) 등가). 두 레짐 필터 대역의 기하 평균에 위치하여 독립적 보조 확인선으로 활용합니다.

| 레짐 | 가격 vs EMA(84) | 신뢰도 | 포지션 사이즈 |
|------|---------------------|--------|--------------|
| 상승 (Daily) | 가격 > EMA(84) | **고신뢰** | 풀 사이즈 |
| 상승 (Daily) | 가격 < EMA(84) | **저신뢰** | 하프 사이즈 또는 대기 |
| 하락 (4H) | 가격 < EMA(84) | **고신뢰** | **축소 사이즈** (BTC bull bias) |
| 하락 (4H) | 가격 > EMA(84) | **저신뢰** | 대기 |

> 4H EMA(84)은 "독립 레짐 판단 지표"가 아니라 레짐의 "신뢰도 부스터"입니다. Daily EMA(14) 등가로, 하락 레짐 필터(Daily EMA 4/9)와 상승 레짐 필터(Daily EMA 21/55) 사이의 기하 평균(√(9×21)≈14일)에 위치하여 양쪽 모두에 독립적인 정보를 제공합니다.

**OI(미결제약정) 변화 — 추세 확신도**

OI는 워시트레이딩에 의해 인위적으로 부풀려지기 어려운 데이터입니다 (동일 계정 내 매수/매도는 포지션이 상쇄되고, 다른 계정 간이라도 추가 증거금 필요).

| 가격 방향 | OI 변화 | 해석 | 활용 |
|----------|---------|------|------|
| 가격↑ | OI↑ | 신규 롱 진입 → **진짜 강세** | 풀 사이즈 허용 |
| 가격↑ | OI↓ | 숏 커버링 → 약한 강세 (지속성 의문) | 사이즈 50% 제한 |
| 가격↓ | OI↑ | 신규 숏 진입 → **진짜 약세** | 풀 사이즈 허용 |
| 가격↓ | OI↓ | 롱 청산 → 약한 약세 (바닥 근접 가능) | 신규 숏 자제 |

**펀딩비 — 과열/과냉 필터**

| 펀딩비 | 상승 레짐 | 하락 레짐 |
|--------|----------|----------|
| < -0.02% | 역추세 숏 과열 → 숏 스퀴즈 가능, 롱 유리 | 정상 숏 |
| -0.02% ~ +0.03% | **정상 롱** (가장 유리한 구간) | **정상 숏** |
| > +0.03% | 고위험 롱 (과열, 사이즈 축소) | 역추세 롱 고려 가능 |
| > +0.05% | **롱 진입 금지** (연환산 비용 >54%) | 역추세 롱 신호 강화 |

> 펀딩비는 크립토 선물 시장에서 유일하게 시장 포지셔닝을 직접적으로 보여주는 데이터입니다. 같은 "EMA 강세 레짐"이라도 펀딩비 +0.01% vs +0.08%는 전혀 다른 리스크 프로파일입니다.

#### 연속 레짐 판단

알고리즘이 상승 레짐(Daily EMA21/55)과 하락 레짐(4H EMA21/55)을 실시간 모니터링하여 "롱만/숏만/레인지/관망" 레짐을 자동 분류합니다. 레짐 변화 감지 시 하위 TF 신호를 즉시 재평가합니다. **하위 TF 신호가 먼저 잡혀도 레짐 조건을 충족하지 않으면 무시합니다.** 다만 EMA 기반 레짐은 외생적 이벤트(규제 뉴스, 거래소 사건, ETF 결정 등)에 대해 구조적으로 후행하므로, 주요 이벤트 캘린더 전후에는 **EMA 신호를 무시하고 관망**합니다.

#### 타이밍 판단: 4H RSI(14) + 방향 전환 필터

**RSI 설정**: RSI(14) 단일 사용 (4시간봉 기준, 56시간 룩백)

> **RSI(9)를 쓰지 않는 이유**: RSI(9)는 RSI(14) 대비 과매수/과매도 진입 빈도가 +60~80% 증가하며, 추가 신호의 상당수가 위양성입니다. 같은 4H TF에서 EMA 크로스오버와 RSI의 상호정보량(mutual information)이 높아, RSI(9)의 "빠른 신호"는 대부분 EMA가 이미 포착한 정보의 중복입니다. RSI(14)에 **방향 전환 필터**를 추가하면 불필요한 신호 증가 없이 타이밍 정밀도를 확보할 수 있습니다.

**방향 전환 필터**: `RSI[현재] > RSI[1봉 전]` (롱) 또는 `RSI[현재] < RSI[1봉 전]` (숏)

RSI가 진입 구간에 도달한 것만으로 진입하지 않고, **RSI가 바닥을 찍고 반등하기 시작한 시점**(또는 천장을 찍고 하락하기 시작한 시점)에만 진입합니다.

> **상승 레짐 구간 길이와 RSI 운용**: Daily 기준 상승 레짐은 평균 13.1일(약 79봉)로, RSI 눌림→반등 사이클을 1~2회 운용할 여유가 있습니다.

| 항목 | RSI 구간 | 해석 |
|------|----------|------|
| **진입 탐색** | 30~45 | 추세 내 눌림이 "지지권"에 도달한 상태 |
| **진입 확인** | RSI 방향 전환 (하락→상승) + 40 이상 회복 | "모멘텀 재점화" 신호 |
| **1차 익절** | 65~70 (40~50% 물량) | 부분 청산 후 잔여분을 추적 손절로 전환. 상승이 장기 지속되는 경향에 대비해 전량 청산 지양 |
| **추적 손절 전환** | 1차 익절 이후 잔여분 | SL을 직전 완성 30m 스윙 저점으로 추적. RSI 75~80 도달 또는 Daily EMA21 이격 과대 시 청산 |
| **손절** | RSI 30 이탈 (종가 기준) | 추세 내 눌림이 아닌 추세 약화 |
| **추가 진입** | 50~60 유지 + RSI 방향 상승 + 가격 신고점 돌파 | 모멘텀이 살아있는 상태에서의 추가 |

> **하락 레짐 구간 길이 주의**: 4H 기준 하락 레짐은 평균 4.2일(약 25봉)로, 4H RSI가 진입→반등→반락 사이클을 1회 완성하기에 빠듯합니다. 진입 후 RSI가 즉시 반응하지 않으면 해당 신호를 포기합니다.

##### 하락 레짐에서 4시간봉 RSI 활용

| 항목 | RSI 구간 | 해석 |
|------|----------|------|
| **진입 탐색** | 55~70 | 하락 추세 내 반등이 "저항권"에 도달한 상태 |
| **진입 확인** | RSI 방향 전환 (상승→하락) + 60 이하 하락 | 반등 실패 확인, 하락 재개 신호 |
| **1차 익절** | 40~45 (50~60% 물량) | 급락 후 이 구간에서 단기 반등이 발생하는 경향. 상승 레짐 대비 조기·높은 비율로 실현 |
| **2차 익절** | 30~35 (잔여) | RSI 40 이하에서 단봉 급반등(30m 강한 양봉 + 거래량 급증) 출현 시 즉시 청산 |
| **손절** | RSI 60 돌파 (종가 기준) | 하락 레짐 약화 신호. 기존 70에서 하향 조정 — 급반등이 빈발하는 경향 반영. 단, 백테스트로 민감도 검증 필요 |
| **조기 종료** | 진입 후 2봉(8시간) 내 RSI 하락 미확인 | 해당 신호 포기. 즉시 반응하지 않는 신호는 하락 레짐 잔여 구간(평균 25봉) 고려 시 신뢰도 낮음 |

##### 레인지 레짐에서 4시간봉 RSI 활용

| 항목 | 조건 |
|------|------|
| **박스 하단 매수** | RSI 30~40 진입 + RSI 방향 전환 (하락→상승) 확인 |
| **박스 상단 매도** | RSI 60~70 진입 + RSI 방향 전환 (상승→하락) 확인 |
| **TP** | RSI 50 도달 시 절반 청산, 반대편 박스 도달 시 나머지 |
| **SL** | 박스 이탈 종가 확정 시 즉시 손절 |

> 레인지 중간(RSI 45~55)에서 신규 진입 금지. 크로스나 50선 돌파만으로 진입하면 잡음에 당하기 쉽습니다.

##### 4시간봉 RSI 다이버전스 활용

| 유형 | 조건 | 활용 |
|------|------|------|
| **베어리시 다이버전스 조짐** | 가격 HH + RSI LH (1회) | 상승 레짐에서 **부분 TP 트리거** (30~40% 청산). 추적 손절 유지. 상승 레짐에서 다이버전스 이후 오버슈팅 후 재상승하는 경향이 있어 단순 출현 시 전량 청산 금지 |
| **베어리시 다이버전스 확정** | RSI LH 2회 연속 + 가격 신고점 | 잔여 포지션 전량 청산. 신규 롱 진입 금지 |
| **불리시 다이버전스** | 가격 LL + RSI HL | 하락 레짐에서 **숏 과반 커버 신호** (50% 이상 즉시 청산). 잔여 숏 유지 시 SL을 직전 30m 스윙 고점으로 즉시 재설정 |
| **불리시 다이버전스 + 거래량 급증** | 불리시 다이버전스 + 해당 봉 거래량 20봉 평균의 2배 이상 | 숏 스퀴즈 가능성 상승. 잔여 숏 전량 청산. 4H EMA로 레짐 재확인 후 재진입 검토 |

> 다이버전스는 단독 진입 신호로 쓰지 않습니다. 상승 레짐에서는 "추세 강도 약화의 경고", 하락 레짐에서는 "급반등 위험 상승의 경고"입니다. 어느 경우든 레짐 전환의 "확인"이 아닌 "경고"로만 취급합니다.

### 레이어 2. 실행 판단 — 30분봉 가격 구조

30분봉은 딱 한 가지만 담당합니다: **"4시간봉 RSI가 진입 조건을 충족했을 때, 정확히 어느 가격에서 들어가는가"**

> **15m가 아닌 30m을 사용하는 이유**: 15m SNR ≈ -14dB (변동의 ~96%가 노이즈). N=3 스윙 포인트는 45분 컨텍스트로, 랜덤워크에서도 3봉마다 발생하며 스탑 헌팅의 직접적 표적. 30m N=5는 2.5시간 컨텍스트로 노이즈 극값 비율이 유의미하게 감소하고, 15m SL(~0.5%)의 주당 10~25회 스탑아웃 문제를 완화

> **캔들 패턴이 아닌 가격 구조를 사용하는 이유**: 전통 TA 캔들 패턴(핀바, 장대양봉)은 크립토에서 코드화하기 모호하고 ("얼마나 긴 꼬리가 핀바인가?"), 봇 매매 비중이 높은 시장에서 신뢰도가 학술적으로 일관되게 입증되지 않음. 스윙 포인트 돌파는 이진(yes/no) 판단으로 명확하게 코드화 가능

30분봉에서 보지 않아야 할 것:
- 레짐 판단 (4H EMA가 담당)
- RSI 신호 (4H RSI가 담당)
- 매크로 방향 (4H EMA(84)이 담당)

#### 스윙 포인트 정의 (N=5)

```
스윙 고점: 중앙 봉의 고가가 양쪽 각 5봉의 고가보다 높은 지점
          → 2.5시간(5×30m) 컨텍스트
스윙 저점: 중앙 봉의 저가가 양쪽 각 5봉의 저가보다 낮은 지점

스탑 헌팅 방지 버퍼:
breakout_buffer = ATR(14, timeframe='30m') × 0.1
```

#### 롱 진입 실행 (상승 레짐 + 4H RSI 30~45 지지 + RSI 방향 전환)

| 조건 | 설명 |
|------|------|
| 30m 종가 > 직전 스윙 고점 + breakout_buffer | 스윙 고점 돌파 확인 (위크 돌파 무시) |
| breakout_buffer | ATR(14) × **0.08** (상승 레짐 적용). 완만한 상승 경향에서 작은 버퍼로 포착률 개선 |
| SL: 직전 30m 스윙 저점 - ATR(14) × 0.5 | 구조적 SL |
| 진입 R:R ≥ 1.5:1 | 최소 R:R 미충족 시 진입 포기 |
| 동일 4H 신호 내 최대 재진입: **2회** | 연속 실패 시 해당 4H 신호 포기 |

#### 숏 진입 실행 (하락 레짐 + 4H RSI 55~70 저항 + RSI 방향 전환)

| 조건 | 설명 |
|------|------|
| 30m 종가 < 직전 스윙 저점 - breakout_buffer | 스윙 저점 이탈 확인 (위크 이탈 무시) |
| breakout_buffer | ATR(14) × **0.15** (하락 레짐 적용). 급락 레짐에서 스탑 헌팅 방어 강화 |
| SL: 직전 30m 스윙 고점 + ATR(14) × **0.7** | 구조적 SL — 급반등 경향으로 허용 오차 확대. SL 확대로 R:R 미충족 시 진입 포기 |
| 진입 R:R ≥ 1.5:1 | 최소 R:R 미충족 시 진입 포기 |
| 동일 4H 신호 내 최대 재진입: **1회** | 하락 레짐 평균 25봉(4.2일)으로 2회 재시도 시간이 부족한 경향 |

### SL/TP 설정 — 레이어 조합 기준

#### SL 설정 원칙

| 유형 | 롱 (상승 레짐) | 숏 (하락 레짐) |
|------|---------------|---------------|
| **구조적 SL** (주된 기준) | 직전 30m 스윙저점 - ATR × 0.5 | 직전 30m 스윙고점 + ATR × 0.5 |
| **RSI 기반 SL** | 4H RSI 30 이탈 (지지 실패) | 4H RSI 70 돌파 (저항 실패) |
| **레짐 기반 SL** (최우선) | **Daily EMA(21)** 종가 이탈 + Daily EMA 크로스 확인 → 전량 청산 | 4H EMA(21) 종가 회복 + 4H EMA 크로스 확인 → 전량 청산 |

우선순위: 구조적 SL 또는 RSI SL 중 **먼저 도달하는 쪽** 적용. EMA 레짐 전환은 어느 것보다 우선하여 즉시 전량 청산.

> **레짐별 SL 비대칭 설계 의도**: 상승 레짐 SL은 Daily EMA(21/55) 기준을 사용하여 **느리게 반응** — 완만한 상승 중 일시적 눌림에서 조기 청산되지 않도록 허용. 하락 레짐 SL은 4H EMA(21/55) 기준을 사용하여 **빠르게 반응** — 급반등 발생 시 숏 손실을 신속히 제한. 진입 필터와 동일한 타임프레임 기준을 공유하여 전략 내 일관성을 유지합니다.

> **하락 레짐 타임아웃 강화**: 하락 레짐 구간이 짧은 경향이 있으므로(역사적 평균 4.2일), 타임아웃을 상승 레짐보다 현저히 짧게 설정합니다. 진입 후 **24~48시간** 내 TP/SL 미도달 시 시장가 청산을 고려합니다. 수치는 백테스트로 최적화 필요.

#### TP 설정 원칙

**상승 레짐 롱 기준**

| 단계 | 조건 |
|------|------|
| **1차 TP** (40~50% 물량) | 4H RSI 65~70 도달 → 부분 청산 후 SL을 진입가로 이동 (본전 SL). 잔여분은 추적 손절 전환 |
| **2차 TP** (나머지 — 추적 손절 운용) | 다음 중 **먼저** 도달: (1) 최고가 대비 -2×ATR(4H) 추적 손절 이탈 / (2) 4H RSI 베어리시 다이버전스 **확정** (단순 조짐 1회에서 전량 청산 금지 — 오버슈팅 후 재상승 경향 있음) / (3) **타임아웃** (진입 후 약 5~7일, 백테스트로 최적값 도출 필요) |

**하락 레짐 숏 기준**

| 단계 | 조건 |
|------|------|
| **1차 TP** (50~60% 물량) | 4H RSI 40~45 도달 → 과반 즉시 청산. 급락 후 이 구간에서 단기 반등이 발생하는 경향 |
| **2차 TP** (잔여) | 다음 중 **먼저** 도달: (1) RSI 30~35 / (2) 4H RSI 불리시 다이버전스 출현 / (3) **타임아웃** (진입 후 **24~48시간**, 상승 레짐 대비 현저히 짧게 설정 — 하락이 단기에 집중되는 경향 반영) |

> **타임아웃 청산 원칙**: 진입 후 TP/SL 미도달 시 알고리즘이 자동 청산합니다. 상승 레짐 타임아웃(5~7일)과 하락 레짐 타임아웃(24~48시간)의 비율은 역사적 평균 레짐 지속 기간 비율(13.1일 vs 4.2일)에 근거하며, 패턴 변화 시 재검토가 필요합니다. RSI 방향성 소멸 기반 청산(RSI가 진입 후 N봉 이상 50 부근에서 횡보 시 청산)도 검토 대상입니다. TP/SL은 진입 시 자동 설정됩니다.

### 리스크 관리 — 필수 인프라

> 전략 로직보다 리스크 관리가 먼저입니다. 아래 항목 없이 라이브 투입은 금지합니다.

#### 필수 리스크 파라미터

| 항목 | 권장 수치 | 설명 |
|------|----------|------|
| **트레이드당 리스크** | 계좌의 0.5~1% | SL까지의 거리 기반 포지션 사이징 |
| **일일 최대 손실** | 계좌의 1.5~3% | 한도 도달 시 당일 자동 중단 |
| **주간 최대 손실** | 계좌의 5~7% | 한도 도달 시 주간 자동 중단 |
| **월간 최대 드로다운** | 계좌의 10~15% | 전략 재검토 트리거 |
| **일일 최대 거래** | 3~5회 | 과잉매매 방지 |
| **연속 손실 정지** | 3연패 → 4시간 쿨다운 | 틸트/연쇄 손실 방지 |
| **동일 신호 재진입** | 최대 2회 | 같은 4H RSI 신호에서의 30m 실행 실패 시 |
| **동시 포지션** | 최대 2개 | 상관 리스크 관리 |
| **슬리피지 가정** | 편도 0.05~0.15% | 백테스트에 반드시 반영 |

#### 포지션 사이징

```
risk_amount = account_balance × risk_per_trade (0.5~1%)
position_size = risk_amount / (entry_price - sl_price)

# 1단계: 레짐 방향 기반 배수 (역사적 패턴 반영)
# BTC는 하락이 단기·급락형이고 급반등이 빈발하는 경향이 있음
# 이 경향이 지속되는 한, 숏 포지션은 구조적으로 축소 시작
if regime_direction == "BULL":
    regime_multiplier = 1.0   # 상승 레짐: 풀 사이즈 기준
elif regime_direction == "BEAR":
    regime_multiplier = 0.6   # 하락 레짐: 기본 40% 축소 (급반등 리스크, 역사적 패턴 기반)
else:  # RANGE
    regime_multiplier = 0.5

# 2단계: 신뢰도 기반 세부 조절 (레짐 배수 위에 적용)
if confidence == "HIGH":    # 4H 레짐 + 4H EMA(84) 일치 + OI 확인
    confidence_multiplier = 1.0
elif confidence == "LOW":   # 4H 레짐 확인, 보조 지표 불일치
    confidence_multiplier = 0.5

# 최종 사이즈
final_size = position_size × regime_multiplier × confidence_multiplier
# 참고: regime_multiplier는 역사적 경향에 근거한 조정값. 패턴 변화 감지 시 재검토 필요
```

| 항목 | 설명 |
|------|------|
| **변동성 스케일링** | ATR 기반 포지션 사이즈 동적 조절 (변동성↑ → 사이즈↓) |
| **유동성 필터** | 오더북 depth가 기준 이하 시 사이즈 축소 또는 진입 금지 |

### 핵심 규칙 체크리스트

**레짐 확인:**
- 상승 레짐: Daily EMA(21)/EMA(55) 정렬을 모니터링 중인가
- 하락 레짐: 4H EMA(21)/EMA(55) 정렬을 모니터링 중인가
- 현재 레짐 분류(롱/숏/레인지/관망)가 올바르게 업데이트됐는가
- 하락 레짐 진입 시 레짐 배수(0.6)와 신뢰도 배수가 2단계로 적용됐는가
- 하락 레짐 구간이 짧은 경향(역사적 평균 4.2일)을 감안하여 타임아웃을 단축했는가 (24~48시간)
- 레짐 전환 시 하위 TF 신호가 재평가되는가
- 주요 이벤트(FOMC, CPI, ETF 등) 전후에 EMA 신호를 무시하고 관망하는가

**신뢰도 확인:**
- 가격이 4H EMA(84) 방향과 일치하는가
- OI 변화가 추세를 지지하는가 (가격↑+OI↑ or 가격↓+OI↑)
- 펀딩비가 과열 수준(|FR| > 0.05%)이 아닌가

**타이밍 확인:**
- 4H RSI(14)가 레짐에 맞는 진입 구간에 있는가
- RSI **방향 전환**이 확인됐는가 (단순 구간 도달이 아님)
- 다이버전스 여부를 체크했는가

**실행 확인:**
- 30m 스윙 포인트 돌파가 종가 기준으로 확인됐는가
- TP/SL이 진입 시 자동 설정되는가
- 진입 R:R이 1.5:1 이상인가
- 동일 신호 내 재진입 횟수가 2회를 초과하지 않는가

**리스크 확인:**
- 일일 손실 한도에 도달하지 않았는가
- 연속 손실 쿨다운이 해제됐는가
- 동시 포지션이 최대 2개를 초과하지 않는가
- 주말/저유동성 시간대에 사이즈가 축소됐는가

**BTC 역사적 패턴 확인 (숏 포지션):**
- (진입 전) 하락 레짐에서도 급반등 경향을 감안하여 사이즈를 0.6 배수로 적용했는가
- (진입 후) 급반등 감지 트리거(4H 2봉 이내 +1.5% 역행 + 30m RSI 50 돌파) 모니터링 중인가
- (진입 후) 4H EMA(21) 종가 상향 돌파 발생 시 레짐 재평가 준비가 됐는가
- (패턴 점검) 최근 하락 레짐 평균 지속 기간이 역사적 평균(~4.2일)과 유사한가 — 크게 벗어날 경우 파라미터 재검토 필요

**금지 사항:**
- 레짐 필터(상승: Daily EMA21/55, 하락: 4H EMA21/55)와 반대 방향으로 진입하려 하지 않는가
- 4H RSI 조건 미충족 상태에서 30m만 보고 진입하지 않는가
- 펀딩비 > +0.05% 상태에서 롱 신규 진입을 시도하지 않는가
- 리스크 한도 초과 상태에서 거래를 시도하지 않는가
- BTC의 역사적 상승 편향을 이유로 유효한 하락 레짐 숏 신호 자체를 무시하지 않는가 (bull bias는 사이즈 축소와 빠른 TP로 반영하며, 신호 무시의 이유가 되어선 안 됨)
- 패턴 변화 신호(하락 레짐 장기화, 급반등 빈도 이상 변화 등)를 무시하고 기존 파라미터를 고집하지 않는가
