## Atlassian 회사 개요

| 항목 | 내용 |
|------|------|
| 설립 | 2002년, 호주 시드니 |
| 창업자 | Mike Cannon-Brookes, Scott Farquhar (UNSW 동기) |

2002년 대학 졸업 후 두 창업자가 시작한 회사로, 소프트웨어 개발팀과 프로젝트 관리를 위한 협업 도구 전문 기업입니다.

## 제품군 구조

Atlassian은 크게 **프로젝트 관리**, **문서 협업**, **코드 관리**, **커뮤니케이션**, **AI/자동화** 영역의 제품을 운영합니다.

### 핵심 제품

| 제품 | 출시 | 용도 | 비고 |
|------|------|------|------|
| **Jira** | 2002 | 이슈 트래킹, 프로젝트 관리 | 플래그십 제품. 칸반/스크럼 보드 |
| **Confluence** | 2004 | 위키/문서 협업 | 팀 지식 베이스 |
| **Bitbucket** | 인수 | Git 코드 저장소 | GitHub/GitLab 대안 |
| **Trello** | 2017 인수 | 경량 칸반보드 | 간단한 프로젝트 관리 |
| **Loom** | 2023 인수 (<span>$</span>975M) | 비동기 영상 메시징 | 화면 녹화/공유 |
| **Rovo** | 2024 출시 | AI 검색/자동화 | Find, Learn, Act 3축 |

### 서비스 관리 제품

| 제품 | 용도 |
|------|------|
| **Jira Service Management (JSM)** | IT 서비스 데스크, ITSM |
| **Customer Service Management** | 고객 지원 관리 |
| **Atlassian Guard** | SSO, 보안, ID 관리 |
| **Atlassian Analytics** | 크로스 제품 분석/대시보드 |

### 제품 번들 (Collection)

Atlassian은 개별 제품 외에 **번들 패키지**를 제공합니다:

- **Work Collection**: Rovo + Jira + Confluence + Loom
- **Service Collection**: JSM + Customer Service Management + Assets + Rovo

## 칸반보드 도구별 요금제 비교 (10인 팀 기준)

10인 이내 소규모 팀에서 칸반보드를 활용할 수 있는 주요 도구의 요금제를 비교합니다.

### 1. Jira <small>(2026-03 조회)</small>

| 플랜 | 월 요금(/인) | 10인 월 비용 | 사용자 제한 | 주요 기능 |
|------|-------------|-------------|------------|----------|
| **Free** | <span>$</span>0 | **<span>$</span>0** | 10명 | 칸반/스크럼 보드, 타임라인, 100 자동화/월, 2GB |
| **Standard** | ~<span>$</span>7.91 | ~<span>$</span>79 | 100,000명 | 권한 관리, 1,700 자동화/월, 250GB, Rovo AI |
| **Premium** | ~<span>$</span>14.54 | ~<span>$</span>145 | 100,000명 | 무제한 저장, Advanced Roadmaps, 99.9% SLA |
| **Enterprise** | 별도 문의 | - | 800명+ | 무제한 자동화, 멀티 사이트, 99.95% SLA |

**10인 팀 추천**: Free 플랜이 정확히 10명까지 지원. 칸반보드, 타임라인, 기본 자동화 포함. 무료로 충분히 사용 가능.

### 2. Trello <small>(2026-03 조회)</small>

| 플랜 | 월 요금(/인) | 10인 월 비용 | 주요 기능 |
|------|-------------|-------------|----------|
| **Free** | <span>$</span>0 | **<span>$</span>0** | 무제한 카드, 워크스페이스당 10 보드, 250 자동화/월 |
| **Standard** | <span>$</span>5 | <span>$</span>50 | 무제한 보드, 고급 체크리스트, 커스텀 필드 |
| **Premium** | <span>$</span>10 (연간) / <span>$</span>12.50 (월간) | <span>$</span>100~<span>$</span>125 | 타임라인/캘린더/대시보드 뷰, Atlassian AI |
| **Enterprise** | <span>$</span>17.50 (연간) | - | 50명+ 필수, SSO, SCIM, 조직 거버넌스 |

**10인 팀 추천**: Free로 시작 → 보드 10개 초과 시 Standard(<span>$</span>50/월). Jira보다 단순한 칸반 운영에 적합.

### 3. Confluence <small>(2026-03 조회)</small>

| 플랜 | 월 요금(/인) | 10인 월 비용 | 주요 기능 |
|------|-------------|-------------|----------|
| **Free** | <span>$</span>0 | **<span>$</span>0** | 10명, 기본 위키/문서 |
| **Standard** | ~<span>$</span>5.42~<span>$</span>6.40 | ~<span>$</span>54~<span>$</span>64 | 250GB, 업무시간 지원 |
| **Premium** | ~<span>$</span>10.44~<span>$</span>12.30 | ~<span>$</span>104~<span>$</span>123 | 무제한 저장, 1,000 자동화/인/월, 24/7 지원 |
| **Enterprise** | 별도 문의 | - | 전용 분석, 고급 보안 |

**참고**: Confluence는 칸반보드 도구가 아닌 **문서 협업** 도구. Jira/Trello와 함께 사용하면 효과적.

## 비-Atlassian 대안 비교

Atlassian 제품 외에도 칸반보드를 지원하는 도구가 있습니다.

### 4. Linear <small>(2026-03 조회)</small>

| 플랜 | 월 요금(/인) | 10인 월 비용 | 주요 기능 |
|------|-------------|-------------|----------|
| **Free** | <span>$</span>0 | **<span>$</span>0** | 무제한 멤버, 250 활성 이슈, 전체 API/연동 |
| **Standard** | <span>$</span>8 | <span>$</span>80 | 무제한 이슈, 파일 업로드, 게스트, 비공개 팀 |
| **Plus** | <span>$</span>14 | <span>$</span>140 | Linear Insights, 데이터 분석, SLA, 우선 지원 |

**10인 팀 추천**: 개발팀이라면 Free로 시작. 이슈 250개 한도만 주의. 깔끔한 UX가 강점.

### 5. Notion <small>(2026-03 조회)</small>

| 플랜 | 월 요금(/인) | 10인 월 비용 | 주요 기능 |
|------|-------------|-------------|----------|
| **Free** | <span>$</span>0 | <span>$</span>0 | 제한적 블록/업로드 |
| **Plus** | <span>$</span>10 (연간) / <span>$</span>12 (월간) | <span>$</span>100~<span>$</span>120 | 무제한 파일, 30일 버전, 무제한 게스트 |
| **Business** | <span>$</span>18 (연간) / <span>$</span>20 (월간) | <span>$</span>180~<span>$</span>200 | SAML SSO, 비공개 팀스페이스, 고급 권한 |

**10인 팀 추천**: 칸반 + 문서를 한 곳에서 관리하고 싶다면. Plus(<span>$</span>100/월)부터 실용적.

### 6. 플로우 (Flow) - 국산 <small>(2026-03 조회)</small>

[flow.team](https://flow.team) · 운영사: 마드라스체크(주) · 2015년 출시

메신저 + 프로젝트 관리를 결합한 **한국형 올인원 협업툴**. 누적 48만 팀, 55개국 이용(2024 기준). 클라우드(SaaS)와 구축형(On-Premise) 모두 지원.

| 플랜 | 월 요금 | 10인 월 비용 | 주요 기능 |
|------|--------|-------------|----------|
| **게스트** | 무료 | - | 초대받은 프로젝트 참여 (제한적) |
| **비즈니스 베이직 (월간)** | 10인 100,000원 (초과 시 1인당 10,000원) | **100,000원** | 프로젝트, 메신저, 업무관리, 파일함, 화상회의, 간트차트 |
| **비즈니스 베이직 (연간)** | 10인 80,000원/월 (초과 시 1인당 8,000원) | **80,000원** | 동일 기능, 20% 할인 |
| **엔터프라이즈 (구축형)** | 별도 문의 | - | On-Premise, 맞춤 구축 |

**10인 팀 추천**: 한국어 인터페이스와 별도 교육 없이 바로 사용 가능한 점이 강점. 메신저 중심 협업에 적합. 다만 무료 플랜이 없어 해외 도구 대비 초기 비용 발생. 30일 무료 체험 가능. 정부지원 바우처 활용 시 최대 80% 할인.

## 추천 시나리오

### 비용 최소화가 최우선
→ **Jira Free** 또는 **Trello Free**. 10인 이하에서 무료로 핵심 칸반 기능 사용 가능.

### 소프트웨어 개발팀
→ **Jira Free** (스프린트, 백로그 관리 필요 시) 또는 **Linear Free** (깔끔한 UX 선호 시).

### 비개발 팀 / 마케팅 / 운영
→ **Trello Free~Standard**. 직관적 드래그앤드롭 칸반, 학습 비용 최소.

### 문서 + 프로젝트 통합
→ **Notion Plus** + Jira/Linear 연동. 문서와 이슈를 한 곳에서 관리.

### Atlassian 올인
→ **Jira Free + Confluence Free**. 둘 다 10명 무료. 이슈 관리 + 문서 협업을 무료로 구성 가능.

## 참고 사항

> **가격 기준 시점**: 본 포스트의 모든 요금 정보는 **2026년 3월** 조사 기준입니다. 각 서비스의 최신 가격은 공식 웹사이트에서 확인하시기 바랍니다.

- Atlassian은 연간 결제 시 15~20% 할인을 제공합니다.
- Atlassian의 **Maximum Quantity Billing (MQB)**: 월간 구독 시 해당 청구 주기 내 최대 사용자 수 기준으로 과금. 인원 변동이 잦은 팀은 주의가 필요합니다.
- 무료 플랜에서 유료 전환 시, 연간 결제를 선택하면 비용을 크게 절약할 수 있습니다.
