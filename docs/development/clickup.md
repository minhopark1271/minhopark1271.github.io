## 개요

ClickUp은 **"하나의 앱으로 모든 것을 대체한다"**는 컨셉의 올인원 프로젝트 관리 플랫폼입니다. Jira(태스크 관리) + Notion(문서) + Slack(채팅) + Loom(화면 녹화)의 역할을 단일 도구로 통합하는 것을 목표로 합니다.

핵심 특징:
- **15+ 뷰**: Kanban, Gantt, Calendar, Table, Mind Map 등 다양한 시각화
- **ClickUp Brain**: GPT-5, Claude, o3 등 복수 AI 모델 전환 지원
- **100+ 자동화 템플릿**: 자연어로 워크플로우 생성 가능
- **1,000+ 통합**: GitHub, Slack, Jira, Salesforce 등 외부 도구 연동

## 핵심 기능

### 태스크 & 프로젝트 관리

ClickUp의 기본 단위는 **태스크(Task)**이며, 계층 구조로 조직됩니다:

```
Workspace → Space → Folder → List → Task → Subtask (최대 7단계)
```

- **커스텀 상태**: `To Do → In Progress → Review → Done` 등 자유 정의
- **우선순위**: Urgent / High / Normal / Low 4단계
- **스프린트**: 포인트 추정, 스프린트 관리 기본 내장
- **종속성(Dependencies)**: 태스크 간 선후 관계 설정
- **반복 태스크**: 주기적 작업 자동 생성

### 뷰 (Views)

동일한 데이터를 팀원 역할에 맞게 다른 형태로 볼 수 있습니다.

| 뷰 | 용도 | 적합 대상 |
|---|---|---|
| **List** | 전통적 태스크 목록 | 전체 |
| **Board** (Kanban) | 상태별 카드 이동 | 개발팀, 디자인팀 |
| **Gantt** | 타임라인·일정·종속성 시각화 | PM |
| **Calendar** | 달력 기반 일정 관리 | 마케팅, 콘텐츠 |
| **Table** | 스프레드시트 형태 | 데이터 정리 |
| **Workload** | 팀원별 업무량 시각화 | 리소스 관리 |
| **Timeline** | 수평 타임라인 | 로드맵 |
| **Mind Map** | 브레인스토밍 | 기획 |
| **Whiteboard** | 자유 형태 협업 보드 | 디자인, 기획 |
| **Map** | 위치 기반 태스크 | 현장 업무 |

### ClickUp Brain (AI)

ClickUp의 AI 기능으로, 경쟁 도구 대비 **복수 AI 모델 선택**이 가능합니다.

- **Super Agents**: 자율적으로 태스크 위임, 이벤트 업데이트, 문서 생성
- **AI Notetaker**: 미팅 자동 녹음 → 전사 → 요약 → 액션 아이템 추출
- **Enterprise Search**: 워크스페이스 전체를 AI로 검색
- **AI Fields**: 태스크 필드 자동 채움 (요약, 분류 등)
- **Talk to Text**: 음성 → 텍스트 변환

지원 모델: GPT-5, Claude Opus, o3, o1-mini 등

### 문서 & 지식 관리

| 기능 | 설명 |
|---|---|
| **Docs** | 태스크에 직접 연결된 협업 문서 (실시간 공동 편집) |
| **Wiki** | 사내 지식·SOP 중앙화 |
| **Notepad** | 빠른 메모 (개인용) |

### 협업 도구

- **Chat**: MS Teams 대체 가능한 내장 채팅 (목적별 채널 생성)
- **SyncUps**: 화상/음성 통화 + 화면 공유 + 자동 전사 + AI 요약
- **Clips**: 화면 녹화 (Loom 대체)
- **Assign Comments**: 코멘트를 태스크로 변환
- **Email 통합**: 이메일을 ClickUp 내에서 관리

### 자동화

**100+ 사전 구축 템플릿** 외에 자연어로 커스텀 워크플로우를 생성할 수 있습니다.

예시:
- "태스크가 `Review` 상태가 되면 → QA 팀에 할당 → Slack 알림 전송"
- "PR이 머지되면 → 연결된 태스크 상태를 `Done`으로 변경"

지원 요소:
- 조건 로직 (if/then)
- 다단계 워크플로우
- 트리거 (상태 변경, 날짜, 필드 변경 등)
- 액션 (할당, 알림, 상태 변경, 이메일 등)

### 대시보드 & 리포팅

**60+ 위젯 타입**으로 커스텀 대시보드를 구성할 수 있습니다:

- 번다운/번업 차트
- 속도 차트 (Velocity)
- 팀 워크로드
- 시간 추적 보고서
- AI Cards (활동 요약)

### 통합 (Integration)

1,000+ 네이티브 통합 지원, Zapier를 통해 7,000+ 앱 추가 연결 가능:

- **개발**: GitHub, GitLab, Bitbucket
- **커뮤니케이션**: Slack, MS Teams, Discord
- **디자인**: Figma
- **CRM**: HubSpot, Salesforce
- **기존 PM**: Jira (마이그레이션 지원)
- **개발자 도구**: API, Webhooks, MCP

## 경쟁 도구 비교

### ClickUp vs Notion vs Jira

| 항목 | ClickUp | Notion | Jira |
|---|---|---|---|
| **핵심 강점** | 올인원 PM + 협업 | 문서·지식 관리 | 소프트웨어 개발·Agile |
| **태스크 관리** | 매우 강력 | 보통 (DB 기반) | 강력 (개발 특화) |
| **문서** | 양호 | 최강 | 약함 (Confluence 별도) |
| **AI** | 멀티 모델 선택 | 단일 모델 | Atlassian Intelligence |
| **커스터마이징** | 최고 | 높음 | 중간 |
| **학습 곡선** | 가파름 | 완만 | 중간 |
| **통합** | 1,000+ (Zapier 7,000+) | 제한적 | Atlassian 에코시스템 |
| **적합 대상** | 팀 협업 전반 | 개인/소규모 팀 | 개발팀 |
| **가격** | $10/유저/월~ | $10/유저/월~ | $7.75/유저/월~ |

### 선택 기준

- **여러 도구를 하나로 통합하고 싶다** → ClickUp
- **문서·위키 중심으로 가볍게 쓰고 싶다** → Notion
- **소프트웨어 개발팀 전용이 필요하다** → Jira
- **개인 생산성 도구가 필요하다** → Notion

## 활용 예시

### 개발팀 스프린트 관리

1. **Board 뷰**에서 Kanban 운영 (To Do → In Progress → Review → Done)
2. **GitHub 통합**으로 커밋/PR을 태스크에 자동 연결
3. **자동화**: PR 머지 시 → 태스크 상태 `Done`으로 변경 → Slack 알림
4. **Gantt 뷰**로 릴리즈 일정 관리

### 크로스팀 프로젝트

1. 마케팅팀은 **Calendar 뷰**, 개발팀은 **Board 뷰**, PM은 **Gantt 뷰**로 동일 데이터 열람
2. **SyncUps**로 주간 미팅 → AI가 자동 요약 → 액션 아이템 태스크 생성
3. **Workload 뷰**로 팀원별 업무 부하 확인 후 리밸런싱

### 1인 개발자/소규모 팀

1. **무료 플랜**으로 시작 (무제한 태스크/유저)
2. **List 뷰** + **Docs**로 간단한 프로젝트 관리
3. **Brain**으로 태스크 설명 자동 생성, 문서 초안 작성

## 요약

ClickUp은 기능의 폭과 커스터마이징 자유도에서 최고 수준의 올인원 플랫폼입니다. "도구 하나로 모든 팀의 협업을 통합하겠다"는 명확한 비전을 가지고 있으며, 무료 플랜도 관대합니다.

다만 **기능이 많은 만큼 초기 학습 곡선이 가파르고**, 대규모 워크스페이스에서 성능 이슈가 보고됩니다. 단일 목적(문서 전용, 개발 이슈 전용)으로는 Notion이나 Jira가 더 적합할 수 있습니다.

가장 빛을 발하는 상황: **여러 직군이 협업하는 팀이 Jira + Notion + Slack + Loom 등 파편화된 도구를 하나로 통합하고 싶을 때**.
