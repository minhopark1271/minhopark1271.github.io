## 일일 요약

오늘 새롭게 부각된 가장 큰 이슈는 이혜훈 중소벤처기업부 장관 후보자의 인사청문회가 야당의 보이콧으로 파행된 점입니다. 국민의힘은 이 후보자의 자료 제출이 15%에 불과할 정도로 부실하다는 점을 명분으로 청문회를 거부하며, 그를 ‘범법 행위자’로 규정하고 수사기관 조사를 촉구했습니다 <a href="#u6X43cvje8Q">[1]</a>, <a href="#eB0MFw0BSCU">[17]</a>, <a href="#BbyxFQOfY4s">[27]</a>, <a href="#TvtWuu77k0c">[33]</a>. 다수의 영상은 이 후보자를 둘러싼 아들의 특혜성 병역, 위장 미혼을 통한 부동산 부정 청약, 보좌진 갑질 등 수많은 의혹을 집중 조명하며 검증의 필요성을 강조했습니다 <a href="#4pJZxQsICpE">[2]</a>, <a href="#awc-UnMcQ7I">[22]</a>, <a href="#nI3S2Qbx0R4">[31]</a>. 이에 이 후보자는 자료 제출률은 75%에 달하며, 30~40년 전 자료는 제출이 불가능하다고 직접 반박했습니다 <a href="#1Q4QoSJrqHk">[20]</a>.

이번 주 처음 등장한 또 다른 논란은 추미애 의원의 ‘경기도 2등 시민’ 발언입니다. 추 의원이 경기도민을 ‘서울 경쟁에서 밀려난 아류 시민’으로 칭한 것이 알려지며, 지역 비하이자 선민의식의 발로라는 거센 비판에 직면했습니다 <a href="#JVEPEa2PIGM">[8]</a>. 이 발언은 과거 ‘이부망천’ 논란을 소환하며, 향후 경기도지사 당내 경선에서 경쟁자들에게 계속 공격받을 치명적인 ‘정치적 먹잇감’이 될 것이라는 전망이 나왔습니다 <a href="#dIbxd1hAq6Y">[10]</a>.

더불어민주당 내부에서는 정청래 대표가 추진하는 **1인 1표제** 도입을 두고 당권파와 비당권파 간의 갈등이 표면화되는 새로운 양상이 나타났습니다. 당원 주권 강화라는 원칙에는 공감대가 있지만 <a href="#VN1O2VLACFo">[4]</a>, 추진 시점과 방식, 그리고 당대표 연임과 관련된 이해충돌 문제 등이 지적되며 내부 이견이 표출되었습니다 <a href="#Fakx7K6yhT4">[6]</a>, <a href="#eB0MFw0BSCU">[17]</a>, <a href="#nI3S2Qbx0R4">[31]</a>.

지난주부터 이어진 국민의힘의 극심한 내분은 오늘 장동혁 대표의 단식과 한동훈 전 대표의 사과를 중심으로 더욱 격화되었습니다. 장동혁 대표는 ‘통일교 게이트’와 ‘공천뇌물 카르텔’ 의혹에 대한 **쌍특검** 수용을 요구하며 닷새째 단식을 이어갔습니다 <a href="#u6X43cvje8Q">[1]</a>, <a href="#2BBjX3MZs_w">[19]</a>, <a href="#KsZK1FJE5To">[35]</a>. 이는 대여 투쟁의 명분을 갖지만, 한동훈 징계 논란으로 인한 리더십 위기를 돌파하고 당 장악력을 강화하려는 국면 전환용 정치적 행보라는 분석이 지배적입니다 <a href="#Fakx7K6yhT4">[6]</a>, <a href="#Ezj6n8x0hv8">[13]</a>, <a href="#DfqL0vMVIaY">[18]</a>.

이와 맞물려 한동훈 전 대표는 ‘당게 사건’에 대해 영상 메시지로 처음 공식 사과했으나, ‘송구’라는 표현과 함께 ‘명백한 조작이자 정치 보복’이라고 주장해 진정성 논란에 휩싸였습니다 <a href="#YaR3Efwqo9E">[12]</a>, <a href="#RJaXPYESxD0">[16]</a>, <a href="#86Ciug3pqdw">[25]</a>. AI 분석 결과 역시 해당 메시지는 사과보다 항변에 치우친 **정치적 입장문**이라는 평가가 나왔으며 <a href="#86Ciug3pqdw">[25]</a>, 향후 그의 선택지로 무소속 출마 등이 거론되었습니다 <a href="#YaR3Efwqo9E">[12]</a>.

지난주 내내 핵심 쟁점이었던 정부의 **검찰개혁안** 논란 역시 계속해서 이어졌습니다. 특히 검찰에 **보완수사권**을 부여하는 것은 수사·기소 분리 원칙을 훼손하고 검찰 권력을 유지시켜주는 **꼼수**라는 비판이 여전히 제기되었습니다 <a href="#dQTl7g52u28">[7]</a>, <a href="#fcfb3fvgEW4">[9]</a>. 수사권에 집착하는 것은 일부 ‘올드보이’ 검사들이며, 대다수 현직 검사들은 직접 수사를 원치 않는다는 내부 분위기도 전달되었습니다 <a href="#fcfb3fvgEW4">[9]</a>.

지난 16일 있었던 윤석열 전 대통령의 1심 징역 5년 선고에 대한 정치적 파장 분석도 여전히 비중 있게 다뤄졌습니다. 특히 재판부가 공수처의 수사권과 영장 집행의 **적법성**을 인정함으로써 윤 전 대통령 측의 핵심 방어 논리를 무력화시킨 것이 향후 내란 재판에 매우 치명적으로 작용할 것이라는 분석이 나왔습니다 <a href="#eB0MFw0BSCU">[17]</a>, <a href="#_Elmri13ixA">[21]</a>.

민주당 공천 헌금 의혹과 관련해서는 김병기, 강선우, 김경 세 인물이 서로 다른 진술을 하며 **각자도생**하는 모습이 집중 조명되었습니다 <a href="#Ch3HUS0x3YY">[11]</a>. 한편, 지난주부터 언급된 **북한 무인기** 이슈는 무인기를 날린 인물들이 윤석열 정부 대통령실 출신이며 국군 **정보사령부**와 수차례 접촉했다는 의혹이 제기되며 새로운 국면을 맞았습니다 <a href="#xkj8ZSANs_4">[3]</a>, <a href="#4fp8ktytvao">[14]</a>, <a href="#mx_Kc75dG0A">[29]</a>.

국제 정세에서는 **오늘 처음으로** 미국이 그린란드 병합 문제로 유럽과 충돌하며 **무역 전쟁** 위기가 고조되는 상황이 다뤄졌습니다 <a href="#v4M7-kNJQ4Y">[5]</a>, <a href="#W3xyF8ADpBY">[34]</a>. **계속해서 지적되는** 고환율 문제와 관련해서는, 정부의 확장 재정이 근본 원인이며 국민연금을 동원해 방어하려 한다는 비판이 제기되었습니다 <a href="#kO59ve9bxxM">[24]</a>, <a href="#sWcyosw5tZw">[28]</a>. 또한 BTS 부산 콘서트 소식에 숙박업소들이 기존 예약을 일방적으로 취소하며 **바가지 요금**을 책정하는 고질적인 문제가 또다시 도마 위에 올랐습니다 <a href="#YxKSTK3o0Pw">[30]</a>.
