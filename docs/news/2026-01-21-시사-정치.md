오늘 처음으로 열린 **이재명 대통령의 신년 기자회견** 역시 핵심 이슈였습니다. 역대 최장 시간인 2시간 52분 동안 진행된 회견에서 <a href="#2x6yxkzAz8k">[5]</a>, 대통령은 여러 현안에 대한 입장을 밝혔습니다. 특히 지난주부터 불거진 **신천지의 조직적 정당 가입** 의혹에 대해, 종교의 정치 개입은 **'나라 망하는 길'**이라며 검경 합동 수사를 지시했고 특검 추진 의사까지 강력하게 피력했습니다 <a href="#CSVPqU7-Cek">[3]</a>, <a href="#N0Ngjdg8yDU">[34]</a>, <a href="#iobj2TSZ54s">[43]</a>. 이와 함께 **코스피가 사상 최고치를 경신**한 것에 대해서는, 인위적 부양이 아닌 저평가 해소를 통한 **정상화 과정**이라 진단하며 주가조작 근절 의지를 밝혔습니다 <a href="#zJ8syUZolKI">[4]</a>, <a href="#8G3UPAw59q0">[37]</a>.

지난주 내내 최대 쟁점이었던 **검찰개혁안** 논란은 대통령의 입장 표명으로 새로운 국면을 맞았습니다. 이 대통령은 수사·기소 분리라는 대원칙은 유지하되, 국민 불편 해소를 위해 **예외적인 보완수사권**이 필요할 수 있다는 실용적 관점을 제시해 주목받았습니다 <a href="#2x6yxkzAz8k">[5]</a>, <a href="#-WsaMV2fu9M">[9]</a>, <a href="#Q1V6z2c27gM">[38]</a>. 또한 **이혜훈 후보자** 거취 문제에 대해서는 청문회가 무산된 점을 아쉬워하며, 본인 해명을 들어봐야 한다는 원칙 아래 시간을 갖고 신중히 판단하겠다고 밝혔습니다 <a href="#8XlNM6PKXps">[33]</a>, <a href="#yoNGzCAjwxw">[42]</a>.

계속해서 이어지는 국민의힘 내부 갈등은 **장동혁 대표의 단식**을 중심으로 더욱 격화되었습니다. 장 대표는 쌍특검 수용을 요구하며 **7일째 단식**을 이어갔고, 건강이 악화돼 산소 발생기까지 착용하는 등 위중한 상태에 이르렀습니다 <a href="#vk3rESrPWdg">[11]</a>, <a href="#durQvFHVV6E">[25]</a>. 하지만 이는 한동훈 제명 논란으로 인한 리더십 위기를 돌파하려는 **정치적 행위**라는 비판이 여전히 제기되고 있으며 <a href="#xvHovtoZfEc">[6]</a>, 한동훈 전 대표 측은 방문할 명분이 없다며 선을 그었습니다 <a href="#Ds_gVKs25f0">[7]</a>.

경제 분야에서는 연일 지적되는 **고환율** 문제가 대통령의 발언으로 다시 도마 위에 올랐습니다. 대통령이 "대책이 있으면 이미 했겠죠"라고 발언한 것을 두고 무책임하다는 비판이 제기되었으며, 정부의 확장 재정이 원화 가치 하락의 근본 원인이라는 지적이 계속되었습니다 <a href="#eRM4_jHRKak">[13]</a>, <a href="#Agp2kcl1EQQ">[22]</a>, <a href="#02VBXl5wkn8">[46]</a>.
