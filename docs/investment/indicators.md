## 이동평균 지표

이동평균은 일정 기간 동안의 가격 평균을 계산하여 가격의 추세를 파악하는 가장 기본적인 기술적 지표입니다.

### 단순이동평균 SMA

Simple Moving Average는 일정 기간 동안의 가격을 단순히 산술평균한 값입니다.

계산식:

```
SMA = (P1 + P2 + ... + Pn) / n
```

여기서:
- P = 각 기간의 가격 (보통 종가)
- n = 기간 (예: 5일, 20일, 200일)

의미:
- 가격이 이동평균선 위에 있으면 상승 추세
- 가격이 이동평균선 아래에 있으면 하락 추세
- 단기 이동평균이 장기 이동평균을 상향 돌파하면 골든크로스 (매수 신호)
- 단기 이동평균이 장기 이동평균을 하향 돌파하면 데드크로스 (매도 신호)

활용:
- SMA(20): 단기 추세 파악
- SMA(50): 중기 추세 파악
- SMA(200): 장기 추세 파악 및 주요 지지/저항선

### 지수이동평균 EMA

Exponential Moving Average는 최근 가격에 더 많은 가중치를 부여한 이동평균입니다.

계산식:

```
EMA_today = (Price_today × k) + (EMA_yesterday × (1 - k))

k = 2 / (n + 1)
```

여기서:
- k = 평활 계수 (smoothing factor)
- n = 기간

의미:
- SMA보다 최근 가격 변화에 민감하게 반응
- 추세 전환을 더 빠르게 포착
- 단, 허위 신호 (whipsaw)도 더 많이 발생

활용:
- EMA(12), EMA(26): MACD 계산에 사용
- 단기 트레이딩에서 SMA보다 선호됨

### 거래량가중이동평균 VWMA

VWMA(100)을 추천한다함.

계산식:

```
VWMA = (Σ(Price_i × Volume_i)) / (Σ Volume_i)
```

여기서:
- Price_i = 각 기간의 종가(또는 지정한 가격)
- Volume_i = 해당 기간의 거래량
- Σ = 선택한 n기간 동안의 합

의미:
- 거래량이 많은 캔들의 가격에 더 높은 비중을 두어, 실제 시장 참여 강도가 반영된 평균 가격을 산출
- SMA보다 추세의 신뢰도를 높이는 데 유용

## 모멘텀 지표

모멘텀 지표는 가격의 변화 속도와 강도를 측정하여 추세의 힘과 전환점을 파악합니다.

### 상대강도지수 RSI

Relative Strength Index는 일정 기간 동안의 가격 상승폭과 하락폭의 비율로 과매수/과매도 상태를 판단하는 지표입니다.

계산식:

```
RSI = 100 - (100 / (1 + RS))

RS = 평균 상승폭 / 평균 하락폭

평균 상승폭 = n일간 상승한 날들의 평균 상승폭
평균 하락폭 = n일간 하락한 날들의 평균 하락폭
```

여기서:
- n = 기간 (일반적으로 14일)

의미:
- RSI 값은 0에서 100 사이
- RSI > 70: 과매수 구간 (매도 고려)
- RSI < 30: 과매도 구간 (매수 고려)
- RSI = 50: 중립

활용:
- 박스권 장세에서 효과적
- 추세장에서는 70/30 기준이 아닌 80/20 기준 사용
- RSI 다이버전스: 가격은 고점 갱신하나 RSI는 고점 못 갱신하면 약세 전환 신호

### MACD

Moving Average Convergence Divergence는 두 개의 이동평균선 간 차이를 이용하여 추세와 모멘텀을 동시에 파악하는 지표입니다.

계산식:

```
MACD Line = EMA(12) - EMA(26)
Signal Line = EMA(MACD Line, 9)
Histogram = MACD Line - Signal Line
```

여기서:
- EMA(12): 12일 지수이동평균
- EMA(26): 26일 지수이동평균
- EMA(MACD Line, 9): MACD 값의 9일 지수이동평균

의미:
- MACD Line > 0: 단기 추세가 장기 추세보다 강함 (상승 추세)
- MACD Line < 0: 단기 추세가 장기 추세보다 약함 (하락 추세)
- MACD Line이 Signal Line을 상향 돌파: 매수 신호
- MACD Line이 Signal Line을 하향 돌파: 매도 신호

활용:
- Histogram의 크기: 추세의 강도 표시
- Histogram이 0선을 통과: 추세 전환 신호
- MACD 다이버전스: 가격과 MACD의 방향이 반대면 추세 약화 신호

### 스토캐스틱

Stochastic Oscillator는 일정 기간의 가격 범위 내에서 현재 가격의 상대적 위치를 백분율로 나타낸 지표입니다.

계산식:

```
%K = ((C - Ln) / (Hn - Ln)) × 100

%D = %K의 m일 이동평균
```

여기서:
- C = 현재 종가
- Ln = n일간 최저가
- Hn = n일간 최고가
- n = 기간 (일반적으로 14일)
- m = 평활 기간 (일반적으로 3일)

의미:
- 값은 0에서 100 사이
- %K > 80: 과매수 구간
- %K < 20: 과매도 구간
- %K가 %D를 상향 돌파: 매수 신호
- %K가 %D를 하향 돌파: 매도 신호

활용:
- 단기 매매에 효과적
- 20/80선 이탈 후 재진입 시점 포착
- 다이버전스: 가격은 고점 갱신하나 스토캐스틱은 고점 못 갱신하면 반전 신호

## 변동성 지표

변동성 지표는 가격의 변동 폭을 측정하여 시장의 위험도와 변동성을 파악합니다.

### 볼린저밴드

Bollinger Bands는 이동평균선을 중심으로 표준편차를 이용한 상하한선을 그어 가격의 변동 범위를 표시하는 지표입니다.

계산식:

```
중심선 (Middle Band) = SMA(20)
상단밴드 (Upper Band) = 중심선 + (2 × 표준편차)
하단밴드 (Lower Band) = 중심선 - (2 × 표준편차)

표준편차 = sqrt(sum((Pi - SMA)^2) / n)
```

여기서:
- n = 기간 (일반적으로 20일)
- 표준편차 배수 (일반적으로 2배)

의미:
- 가격의 약 95%가 밴드 내에 위치
- 밴드 폭 확대: 변동성 증가
- 밴드 폭 축소 (squeeze): 변동성 감소, 큰 움직임 임박
- 가격이 상단 밴드 터치: 과매수 상태
- 가격이 하단 밴드 터치: 과매도 상태

활용:
- 박스권: 상단에서 매도, 하단에서 매수
- 추세장: 밴드 돌파 후 추세 추종
- 밴드 워킹 (Band Walking): 강한 추세에서 가격이 상단 또는 하단 밴드를 따라 움직임

### 평균실제범위 ATR

Average True Range는 가격의 실제 변동폭을 측정하여 변동성을 수치화한 지표입니다.

계산식:

```
True Range (TR) = max(다음 3가지 중)
1. 당일 고가 - 당일 저가
2. |당일 고가 - 전일 종가|
3. |당일 저가 - 전일 종가|

ATR = TR의 n일 이동평균
```

여기서:
- n = 기간 (일반적으로 14일)

의미:
- ATR 값이 클수록 변동성이 큼
- ATR 값이 작을수록 변동성이 작음
- 방향성을 나타내지 않고 변동성만 측정

활용:
- 손절가 설정: 진입가 ± (ATR × 배수)
- 포지션 사이징: 변동성이 클수록 포지션 축소
- 시장 상태 판단: ATR 급등은 변동성 확대 신호
- 브레이크아웃 필터: ATR이 작을 때 돌파 신호에 주의

## 거래량 지표

거래량 지표는 가격과 함께 거래량을 분석하여 추세의 신뢰도를 평가합니다.

### 온밸런스볼륨 OBV

On-Balance Volume은 가격 상승/하락에 따라 거래량을 누적하여 자금의 흐름을 파악하는 지표입니다.

계산식:

```
OBV_today = OBV_yesterday + Volume (종가가 상승한 경우)
OBV_today = OBV_yesterday - Volume (종가가 하락한 경우)
OBV_today = OBV_yesterday (종가가 보합인 경우)
```

여기서:
- 초기 OBV = 첫날 거래량

의미:
- OBV 상승: 매수세 우세 (자금 유입)
- OBV 하락: 매도세 우세 (자금 유출)
- 가격과 OBV가 같은 방향: 추세 확인
- 가격과 OBV가 반대 방향 (다이버전스): 추세 전환 신호

활용:
- 가격 상승 + OBV 상승: 상승 추세 확인
- 가격 하락 + OBV 하락: 하락 추세 확인
- 가격 신고가 + OBV 신고가 미달: 상승 추세 약화 경고
- 돌파 확인: 가격 돌파 시 OBV도 함께 돌파하면 신뢰도 상승

### 거래량가중평균가 VWAP

Volume-Weighted Average Price는 거래량을 가중치로 하여 계산한 평균 가격으로 기관 투자자들의 매매 기준선으로 활용됩니다.

계산식:

```
VWAP = sum(Typical Price × Volume) / sum(Volume)

Typical Price = (High + Low + Close) / 3
```

여기서:
- High = 고가
- Low = 저가
- Close = 종가
- Volume = 거래량
- sum = 누적 합계 (보통 당일 시작부터)

의미:
- 현재 가격 > VWAP: 매수세 우세 (강세)
- 현재 가격 < VWAP: 매도세 우세 (약세)
- VWAP는 동적 지지/저항선 역할

활용:
- 기관의 평균 매수가 추정
- 장중 매매 기준선: VWAP 위에서 매수, 아래에서 매도
- 체결 벤치마크: 기관들이 VWAP 대비 체결가 평가
- 일중 매매: 시가부터 누적 계산하여 당일만 유효

일별 리셋:
- 실전에서는 매일 자정에 계산을 새로 시작
- 장 시작 시 VWAP = 시가
- 거래가 진행되며 누적 평균 업데이트

### Session VWAP, Rolling VWAP

Session VWAP:
- 하루(또는 특정 세션) 시작부터 누적된 거래량과 가격을 이용해 계산한 VWAP입니다.
- 시간의 경과에 따라 안정적인 기준선을 제공하며, 기관 투자자의 매매 벤치마크로 자주 사용됩니다.

Rolling VWAP:
- 최근 n개의 캔들 또는 일정 기간 내 데이터만 포함하여 계산한 VWAP입니다.
- 단기 변동성에 민감하게 반응하며, 단타 및 스캘핑 전략에 유용합니다.

## 지표 조합 활용

### 추세 확인 조합

이동평균 + MACD:
- 가격이 SMA(20) 위 + MACD > Signal: 강한 상승 추세
- 가격이 SMA(20) 아래 + MACD < Signal: 강한 하락 추세

### 과매수 과매도 조합

RSI + 스토캐스틱:
- RSI > 70 + %K > 80: 강한 과매수 (매도 고려)
- RSI < 30 + %K < 20: 강한 과매도 (매수 고려)

### 변동성 돌파 조합

볼린저밴드 + ATR:
- 밴드 폭 축소 + ATR 하락: 큰 움직임 임박
- 밴드 돌파 + ATR 상승: 추세 돌파 확인

### 거래량 확인 조합

가격 + OBV + VWAP:
- 가격 신고가 + OBV 신고가 + VWAP 돌파: 강한 상승 신호
- 가격 하락 + OBV 하락 + VWAP 이탈: 강한 하락 신호

## 지표 사용 시 유의사항

### 후행성

모든 기술적 지표는 과거 데이터 기반이므로 후행성을 가집니다. 지표가 신호를 줄 때는 이미 가격이 어느 정도 움직인 후입니다.

### 허위 신호

박스권이나 변동성이 큰 시장에서는 지표가 자주 오류 신호를 발생시킵니다. 단일 지표만으로 판단하지 않고 여러 지표를 조합하여 확인해야 합니다.

### 시장 상황별 적용

추세장과 박스권에서 효과적인 지표가 다릅니다:
- 추세장: 이동평균, MACD 효과적
- 박스권: RSI, 스토캐스틱, 볼린저밴드 효과적

### 기간 설정

지표의 기간 설정에 따라 민감도가 달라집니다:
- 단기 (5-14): 빠른 반응, 허위 신호 많음
- 중기 (20-50): 균형잡힌 신호
- 장기 (100-200): 느린 반응, 신뢰도 높음

자신의 투자 스타일에 맞는 기간 설정이 중요합니다.
