# RSI + MA 배열 활용 장세 분류 & 매매 케이스
{:.no_toc}

## 공통 관측 프레임워크

매매 판단 전, 아래 두 축을 항상 동시에 확인합니다.

### MA 배열 체크포인트

| 항목 | 확인 내용 |
|------|-----------|
| **정렬** | 단기 > 중기 > 장기 (상승 추세) / 반대 (하락 추세) |
| **기울기** | 장기 MA가 우상향인지 → "진짜 추세"의 핵심 조건 |
| **이격** | 가격이 단기 MA에서 얼마나 떨어졌는지 → 과열/되돌림 위험 판단 |

### RSI 체크포인트

| 항목 | 확인 내용 |
|------|-----------|
| **50선 위치** | 상승 압력 우위 (50 위) vs 하락 압력 우위 (50 아래) |
| **추세장 밴드** | 상승장에선 RSI 바닥이 주로 40 근처, 하락장에선 천장이 60 근처 |
| **다이버전스/실패스윙** | 모멘텀 구조 변화의 조기 신호 |

> **핵심 원칙**: MA는 "어느 방향의 장인지(레짐)"를 알려주고, RSI는 "그 장 안에서 지금 어느 위치인지(타이밍)"를 알려줍니다. 둘을 분리해서 쓰면 절반짜리 정보가 됩니다.

## 케이스별 기대값 순위 요약

승률과 손익비(R:R) 기준으로 유리한 구간 순서입니다.

| 순위 | 케이스 | 예상 승률 | 손익비 | 핵심 근거 |
|------|--------|-----------|--------|-----------|
| **1위** | Case 1 — 정상 상승 추세 | 60~70% | 2:1 이상 | 추세 추종 중 가장 일관성 높음. 추적 손절로 수익 극대화 가능 |
| **2위** | Case 4 — 정상 하락 추세 | 55~65% | 2:1 이상 | Case 1과 대칭 구조. 암호화폐는 급반등 위험으로 SL 관리가 소폭 까다로움 |
| **3위** | Case 7 — 돌파 장 | 50~60% | 3:1 이상 | 승률은 중간이지만 레인지 높이 = 측정 가능한 목표로 R:R 우수 |
| **4위** | Case 6 — 레인지/박스 | 60~70% | 1:1~1.5:1 | 극단 구간 승률은 높지만 박스 폭이 목표의 한계 — R:R이 구조적으로 제한됨 |
| **5위** | Case 5-B — 진짜 바닥 전환 | 40~55% | 3:1 이상 | R:R은 최상급이나 5-A(함정)와의 구분이 어려워 실전 승률이 낮아짐 |
| **6위** | Case 3 — 상승→전환 초기 | 35~45% | 2:1 이상 | 이론상 R:R은 좋지만 추세 재개 시 손절 빈번. "SL이 생명" 구간 |
| **7위** | Case 2 — 과열/피로 | — | 불리 | 신규 진입 금지 구간. 기존 보유자의 분할 익절 타이밍으로만 유효 |

```
EV(기대값) 순서:

Case 1 ≈ Case 4  >  Case 7  >  Case 6  >  Case 5-B  >  Case 3  >>  Case 2
   (추세 추종)       (돌파)      (레인지)    (반전)         (전환)     (진입 금지)
```

## 케이스 1. 정상 상승 추세

> **"추세 추종의 교과서" — 가장 높은 확률의 롱 구간**

| 구분 | 조건 |
|------|------|
| MA | 단기 > 중기 > 장기 정렬 + 장기 MA 우상향 |
| 가격 | 단기/중기 MA 위에서 눌림 후 재상승 |
| RSI | 50선 위 유지, 조정 때 40~50에서 지지 후 반등 |

### 진입 시점

- **1차 진입**: 눌림이 중기 MA(20 or 60) 부근에서 멈추고 캔들이 양전환 확인
- **2차 추가**: 고점 돌파 + RSI가 50~60 이상 재확인 ("엔진 재점화" 신호)

### TP/SL

| 구분 | 조건 |
|------|------|
| **SL** | 눌림 저점 이탈 *또는* 중기 MA 종가 이탈 + RSI 50 하향 **동시** 충족 |
| **TP 1** | 직전 스윙 고점 부근 분할 청산 (30~50%) |
| **TP 2** | RSI 70~80에서 꺾이는데 가격이 신고점 못 내면 (약한 다이버전스) 추가 정리 |
| **TP 3** | 추세 유지형이면 중기 MA 기반 **추적 손절**이 수익 극대화에 유리 |

---

## 케이스 2. 상승 추세의 과열/피로

> **"방향은 살아있지만, 기대수익/리스크 비율이 나빠진 상태"**

| 구분 | 조건 |
|------|------|
| MA | 정렬은 상승이지만, 가격이 단기 MA에서 과도하게 이격 |
| RSI | 75~85 고레벨 반복 + 베어리시 다이버전스 출현 (가격 HH, RSI LH) |

**중요 해석**: 추세가 꺾인 "확정"이 아니라, 지속 확률은 있지만 **신규 진입의 매력이 크게 줄어든 상태**입니다.

### 실전 운영

| 포지션 | 대응 |
|--------|------|
| **신규 진입** | 보수적 대기 (눌림 기다리기) — 추격 매수 금지 구간 |
| **기존 보유자** | 이격/RSI 과열 구간에서 분할 익절 시작 |
| **SL 조정** | 단기 MA 이탈이 아닌 **중기 MA 이탈**을 트리거로 (과열 구간은 단기 흔들림이 극심) |

## 케이스 3. 상승 추세 → 분배/전환 초기 신호

> **"경고에서 확인으로 — 단계적으로 읽어야 실수를 줄인다"**

전환은 한 번에 오지 않습니다. **4단계로 순서대로 확인**해야 합니다.

| 단계 | 신호 |
|------|------|
| 1단계 | RSI 베어리시 다이버전스 / 실패스윙 조짐 |
| 2단계 | 가격 고점 돌파 실패 (더블탑 유사 구조) |
| 3단계 | 단기 MA 기울기 둔화 → 단기/중기 데드크로스 가능성 |
| 4단계 | **RSI 50선 하향 이탈** ← "전환 확인"의 가장 중요한 신호 |

### 진입 시점

- **숏/헤지**: RSI 실패스윙 완성 + 50선 하향 + 단기 < 중기 첫 확인 시
- **기존 롱 보유자**: "새 롱 중단 + 보유분 리스크 축소"가 먼저. 바로 숏 전환은 성급함

### TP/SL

전환 매매에서는 **SL이 생명**입니다.

| 구분 | 조건 |
|------|------|
| **SL** | 마지막 구조적 고점 상향 돌파 시 즉시 손절 |
| **TP** | 장기 MA(120/200) 또는 이전 박스 하단까지 단계적 목표 |

## 케이스 4. 정상 하락 추세

> **"추세 추종 vs 역추세 반등매매를 반드시 구분해야 한다"**

| 구분 | 조건 |
|------|------|
| MA | 단기 < 중기 < 장기 정렬 + 장기 MA 하향 |
| RSI | 50선 아래 유지, 반등 때 60 근처에서 막히고 재하락 |

### 전략 분기

| 전략 | 진입 조건 |
|------|-----------|
| **추세 추종 (숏/현금 비중 확대)** | 반등이 중기 MA에서 막힐 때 진입 |
| **역추세 바닥잡기** | RSI 30 이하만으로는 **절대 부족**. 케이스 5-B 조건 모두 충족 시에만 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **숏 TP** | 이전 저점 / RSI 20~30 과매도 구간에서 분할 청산 |
| **숏 SL** | "중기 MA 회복 + RSI 50 회복" 동시 조건 — **레짐 변화 기반 청산** |

## 케이스 5. 하락 추세 속 반등 — 함정 vs 바닥 전환

> **"가장 헷갈리는 구간 — RSI+MA 조합이 가장 빛나는 케이스"**

### 5-A. 데드캣 바운스 (반등 함정)

| 항목 | 상태 |
|------|------|
| MA | 장기 하향, 배열 여전히 하락 정렬 |
| RSI | 반등해도 50을 못 넘고 55~60에서 꺾임 |
| 가격 | 중기 MA(60) 아래에서 반등 종료 |

→ **"추세 내 조정" 가능성 높음**. 롱 진입 자제.

### 5-B. 진짜 바닥 전환 후보

| 항목 | 상태 |
|------|------|
| RSI | 불리시 다이버전스 (가격 LL, RSI HL) + 이후 **50선 상향 회복** |
| MA | 단기 MA가 평평→상향 전환, 가격이 중기 MA를 회복하고 지지 |
| 배열 | 완전 전환 전이라도 **"단기 > 중기"가 먼저** 발생 |

→ 이 조합이 나오면 **"진짜 전환 후보"로 격상**해서 관찰합니다.

### TP/SL (전환 롱은 SL을 타이트하게)

| 구분 | 조건 |
|------|------|
| **SL** | 다이버전스를 만든 마지막 저점 이탈 시 즉시 손절 |
| **TP 1차** | 중기/장기 MA 구간 (첫 저항) |
| **TP 이후** | MA 배열이 상승 전환되면 케이스 1 (추세 추종)으로 운영 방식 변경 |

## 케이스 6. 레인지/박스 장

> **"RSI 단독으로 가장 잘 먹히는 구간 — MA 크로스는 잡음으로 취급"**

| 구분 | 조건 |
|------|------|
| MA | 장기 MA 평평, 단기/중기 MA가 서로 엉킴 (배열이 자주 뒤집힘) |
| RSI | 40~60 또는 30~70 사이 왕복 반복 |

### 진입 시점

| 방향 | 조건 |
|------|------|
| **박스 하단 매수** | RSI 30~40 + 가격이 단기 MA 회복 확인 |
| **박스 상단 매도** | RSI 60~70 + 가격이 단기 MA 이탈 확인 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **TP** | 박스 반대편 + RSI가 50 도달 시 일부 청산 (욕심 금지) |
| **SL** | 박스 종가 이탈 + RSI가 50을 **한 방향으로 지속 유지** (레짐 변화 신호) |

> 레인지 장에서 MA 크로스 신호는 잡음이 많습니다. **RSI 평균 회귀 전략이 우선**입니다.

## 케이스 7. 돌파 장 (레인지 → 추세 레짐 전환)

> **"레인지 탈출 — 리테스트 확인이 핵심"**

### 돌파 판별 조건

| 단계 | 상태 |
|------|------|
| **돌파 전** | MA 평평/엉킴 + RSI 중립권 (45~55) |
| **돌파 발생** | 가격이 중기/장기 MA 위에 안착 + RSI가 50 돌파 후 60~70 테스트 |
| **결정적 확인** | 눌림(리테스트)에서 **RSI가 40~50에서 지지** ← 핵심 |

### 진입 시점

| 스타일 | 조건 |
|--------|------|
| **보수적** | 돌파 후 첫 눌림 (리테스트)에서 중기 MA 지지 확인 후 진입 |
| **공격적** | 돌파 캔들 + 거래량 조건 충족 시 즉시 진입 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **SL** | 리테스트 실패 (돌파 레벨 하향 이탈) + RSI 50 하향 동시 |
| **TP 1차** | 직전 레인지 높이만큼 측정 목표 |
| **TP 이후** | 돌파 확인되면 케이스 1 (정상 상승 추세) 운영으로 전환 |

## TP/SL 규칙화 요약

| 방식 | 적합 케이스 | 설명 |
|------|------------|------|
| **구조 기반** (스윙 저점/고점 + RSI 50 확인) | 케이스 1, 3, 5 | 추세 매매에서 가장 일관적 |
| **MA 기반** (중기 MA 종가 이탈 + RSI 밴드) | 케이스 1, 4, 7 | 휩소 감소, 추세 유지 조건화 |
| **과열 기반** (RSI 70/30 + 이격) | 케이스 2, 6 | **TP에만 유리**, SL 단독 사용 비추 |

## 레짐 분류 빠른 판단 트리

```
장기 MA 방향은?
├─ 우상향 → 상승 레짐
│       RSI 50 위? → 케이스 1 (추세 추종)
│       RSI 과열 + 다이버전스? → 케이스 2/3 (과열/전환 주의)
│
├─ 하향 → 하락 레짐
│       반등 RSI 50 못 넘음? → 케이스 5-A (함정)
│       불리시 다이버전스 + RSI 50 회복? → 케이스 5-B (전환 후보)
│
└─ 횡보/평평 → 레인지 레짐
        RSI 왕복? → 케이스 6 (레인지 매매)
        RSI 50 돌파 + MA 정렬 시작? → 케이스 7 (돌파 대기)
```
