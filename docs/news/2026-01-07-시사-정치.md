오늘 정치권의 가장 큰 이슈는 국민의힘 내부 갈등이었습니다. 장동혁 대표의 비상계엄 사과가 논란의 중심에 섰습니다. 장 대표는 계엄 선포 자체보다는 '상황에 맞지 않는 잘못된 **수단**'이었다고 사과했는데, 이를 두고 '꼼수 사과'라는 비판이 제기되었습니다 <a href="#FsXZ3xtmjww">[1]</a>. 이 사과는 즉각 극우 지지층의 거센 반발을 샀습니다. 역사 강사 전한길 씨를 필두로 한 지지자들은 "사과하라고 지지한 게 아니다"라며 분노했고, 일부는 장 대표를 '**중공 간첩**'이라 비난하며 팬카페를 탈퇴하는 등 '멘붕' 상태에 빠졌습니다 <a href="#FsXZ3xtmjww">[1]</a>.

반면, 당내 다른 세력은 이 사과를 **한동훈 전 대표**를 겨냥한 정치적 행보로 해석했습니다. 사과의 진정성을 위해서는 계엄의 배후로 지목되는 **윤석열 세력과의 절연**이 선행되어야 하는데, 이 부분이 빠졌다는 지적입니다 <a href="#2RniWsMSSgQ">[3]</a>. 특히 '누구와도 함께 하겠다'는 쇄신안 발표에도 불구하고, 실제로는 한동훈 전 대표를 징계하기 위한 명분 쌓기라는 비판이 제기되었습니다 <a href="#2RniWsMSSgQ">[3]</a>, <a href="#LOrNf1jM3kA">[9]</a>. 새로 구성된 윤리위원회 역시 사법적 판단보다는 한 전 대표를 축출하기 위한 **정치적 판단**을 할 것이라는 우려가 큽니다 <a href="#LOrNf1jM3kA">[9]</a>. 신임 윤리위원장이 과거 "개딸들의 이재명 사랑은 김건희 여사에 대한 질투"와 같은 편향된 발언을 한 사실도 논란을 키웠습니다 <a href="#y5ovlVtnQ9Q">[12]</a>. 친한계는 부당한 징계 시 강력히 대응할 것을 예고하며 <a href="#2RniWsMSSgQ">[3]</a>, 당 지도부와 중진 및 일반 당원 간의 생각 차이가 뚜렷해지고 있음이 드러났습니다 <a href="#LOrNf1jM3kA">[9]</a>.

2. 주요 인물 논란 가열: 이혜훈·김병기·김건희 의혹

여러 주요 정치인을 둘러싼 의혹도 하루 종일 뜨거웠습니다.

**이혜훈** 중소벤처기업부 장관 후보자는 지명 배경부터 여러 논란에 휩싸였습니다. 대통령실은 그의 지명을 **중도보수 확장**과 **부산 선거** 등을 고려한 다목적 카드로 분석했지만 <a href="#F3ypWFWTcw8">[5]</a>, 지명 직후부터 '막말 갑질' 의혹이 불거졌습니다 <a href="#jxAlr7KZ40c">[10]</a>. 시민단체는 이 후보자의 폭언을 협박형, 인격 말살형 등으로 분류하며 문제를 제기했습니다 <a href="#jxAlr7KZ40c">[10]</a>. 또한, 장남의 논문 작성 과정에서 '**아빠 찬스**'를 활용했다는 의혹도 제기되었습니다 <a href="#y5ovlVtnQ9Q">[12]</a>. 민주당은 대통령의 인사권을 존중해 청문회까지는 가야 한다는 입장이지만 <a href="#sNHW4lkbBXg">[11]</a>, 도덕성뿐 아니라 정책 비전까지 철저히 검증하겠다고 밝혔습니다 <a href="#F3ypWFWTcw8">[5]</a>.

**김병기** 더불어민주당 의원의 공천 헌금 의혹은 단순 실수가 아닌 '**휴먼 크라임(human crime)**'이라는 강도 높은 비판을 받았습니다 <a href="#maqkMFPsfnE">[7]</a>. 이 문제는 여야를 떠나 나타나는 구조적 문제라는 지적이 나왔습니다 <a href="#maqkMFPsfnE">[7]</a>. 김 의원이 "알리바이를 만들어야겠다"고 말하는 것을 들었다는 보좌진의 증언이 나오며 '**거짓 해명**' 의혹까지 더해졌습니다 <a href="#y5ovlVtnQ9Q">[12]</a>. 당내에서는 이를 공천 시스템의 문제가 아닌 '개인의 문제'로 선을 긋고 있지만 <a href="#sNHW4lkbBXg">[11]</a>, 당에 지속적인 부담으로 작용하고 있습니다 <a href="#F3ypWFWTcw8">[5]</a>.

**김건희** 여사와 관련된 의혹도 새로 추가되었습니다. 서희건설로부터 **반 클리프 앤 아펠** 목걸이 등 8천만 원 상당의 명품을 **서초동 아크로비스타**에서 수수한 뒤, "회사에 도와줄 일이 없느냐"고 직접 연락한 정황이 드러났습니다 <a href="#w5EyB-OjP2Y">[4]</a>. 이는 김영삼 전 대통령의 친필 사인이 담긴 책들을 분리수거한 사실과 대비되며 비판을 받았습니다 <a href="#w5EyB-OjP2Y">[4]</a>. 또한, 양평고속도로 종점이 김건희 일가 땅이 있는 곳으로 변경되는 것을 사전에 인지했다는 정황이 담긴 **문건**이 확보되기도 했습니다 <a href="#y5ovlVtnQ9Q">[12]</a>.

최근 열린 한중정상회담에 대한 긍정적 평가가 나왔습니다. 중국 현지 언론은 이번 회담을 대대적이고 **우호적**으로 보도하며, 양국 관계가 정상 궤도에 재진입했다고 평가했습니다 <a href="#asrdCP1ZSj8">[8]</a>. 특히 이번 회담의 최대 성과로는 '**전략적 협력 동반자 관계 복원**'이 꼽혔습니다 <a href="#asrdCP1ZSj8">[8]</a>. 서해 구조물 문제에 대해 한국이 먼저 중국의 원칙인 '**분쟁 유보, 공동 개발**'을 원용해 해법을 제시한 것이 '**실사구시**'적 접근이라며 높은 평가를 받았습니다 <a href="#asrdCP1ZSj8">[8]</a>. 시진핑 주석의 '역사의 **올바른 편**에 서야 한다'는 발언 역시 특정 국가에 대한 줄서기 요구가 아닌, 정의로운 선택을 하자는 원론적 의미로 해석되었습니다 <a href="#asrdCP1ZSj8">[8]</a>. 반면, 이재명 대통령이 쿠팡 개인정보 유출 사건 범인이 중국인이라는 지적에 '어쩌라고'라고 반응한 사례는 그의 직설적 외교 스타일을 보여주는 단면으로 언급되었습니다 <a href="#FsXZ3xtmjww">[1]</a>.
