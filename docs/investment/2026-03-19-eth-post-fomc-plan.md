## dry powder 소스: ETH 매도 vs BTC 전환

dry powder 복원이 필요하다면, **무엇을 팔 것인가**가 핵심 의사결정입니다.

### ETH 매도의 문제점

| 문제 | 설명 |
|------|------|
| **역사적 저평가 자산을 매도** | MVRV 0.85, 7/7 바닥 신호 = ETH 역사상 가장 강한 저평가 구간 |
| **상대적 약세 자산을 매도** | ETH/BTC 0.032~0.034 (14개월 최저) = ETH가 BTC 대비 역대급 할인 중 |
| **반등 시 기회비용 극대화** | ETH 반등폭 > BTC 반등폭 (고베타) — 저평가 자산의 반등을 놓침 |
| **7/7 신호와 모순** | 바닥 신호가 "매수" 를 가리키는데, 같은 자산을 "매도"하는 것은 논리적 비일관 |

### BTC 일부 전환의 장점

| 장점 | 설명 |
|------|------|
| **상대적 고평가 자산 매도** | BTC -4% vs ETH -6.48% = BTC가 덜 빠짐 → 더 높은 가격에 매도 |
| **ETH/BTC 리밸런싱** | BTC 줄이고 ETH dry powder 확보 = ETH/BTC 반등에 베팅하는 구조 |
| **7/7 신호는 ETH 전용** | 바닥 신호 7개는 모두 ETH 온체인 지표 — BTC에는 해당 없음 |
| **헷지 역할 변화** | FOMC 이벤트 통과 → BTC의 "이벤트 헷지" 역할 종료 |
| **비중 과대** | BTC 72%는 이미 과도 — 5~7% 줄여도 65~67%로 여전히 최대 비중 |

### BTC 매도의 리스크

| 리스크 | 평가 |
|--------|------|
| 매크로 악화 시 BTC 방어력 약화 | **낮음** — <span>$</span>3~5K(7~10%) 매도로 헷지 기능 실질 영향 미미 |
| BTC <span>$</span>70K 이탈 가능성 | 이 경우 ETH도 동반 하락 → 어떤 자산을 팔았든 동일 문제 |
| "BTC 매도 금지" 원칙 | 기존 원칙은 **FOMC 대비 헷지 목적** → FOMC 종료 후 전제 소멸 |

### 결론: BTC 일부 전환이 논리적으로 우월

**ETH는 역사적 저평가(MVRV 0.85, 7/7 신호)**이고 **BTC는 상대적 고평가(ETH/BTC 14개월 최저)**. 저평가 자산을 팔아서 dry powder를 만드는 것보다, 상대적 고평가 자산을 줄여서 dry powder를 만드는 것이 합리적입니다.
