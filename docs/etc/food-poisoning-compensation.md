## 300만원이 Max인가?

**아닙니다.** 300만원은 판례에서 "위자료만" 청구한 금액이고, 실제 배상은 여러 항목의 합산입니다.

### 대표 판례: 분당 김밥 프랜차이즈 사건 (2021)

살모넬라균 식중독으로 121명 피해가 발생한 사건에서:

| 구분 | 청구액 | 판결액 |
|------|--------|--------|
| 입원+통원 치료자 위자료 | 1인당 300만원 | **200만원** |
| 통원 치료만 받은 자 위자료 | 1인당 300만원 | **100만원** |

**핵심**: 위자료 300만원 청구 → 100~200만원 인정이 판례 수준입니다. 하지만 **위자료는 전체 배상의 일부**에 불과합니다.

---

## 청구 가능한 항목 전체

실제 받을 수 있는 배상은 아래 항목의 **합산**입니다.

| 항목 | 설명 | 비고 |
|------|------|------|
| **치료비** | 병원비, 약값, 검사비 전액 | 영수증 필수 |
| **일실수익** | 일 못한 기간의 소득 손실 | 급여명세서/사업소득 증빙 |
| **교통비** | 병원 통원 교통비 | 택시 영수증 등 |
| **간병비** | 입원 시 간병인 비용 | 해당 시 |
| **위자료** | 정신적 고통에 대한 배상 | 통원 100만, 입원 200만 수준 |
| **향후 치료비** | 후유증 치료 예상 비용 | 의사 소견서 필요 |

### 현실적 합산 시나리오

**경증 (통원 3~5일)**:
- 치료비 10~30만원 + 일실수익 20~50만원 + 위자료 100만원
- **합계: 약 130~180만원**

**중증 (입원 3~7일)**:
- 치료비 50~150만원 + 일실수익 50~200만원 + 위자료 200만원 + 간병/교통비
- **합계: 약 350~600만원+**

**후유증 동반 (만성 장질환 등)**:
- 위 금액 + 향후 치료비 + 추가 위자료
- **합계: 500만원~1,000만원+** (사안에 따라 상향)

## 변호사 활용 전략: 비용 대비 효과

### 변호사 비용 구조

| 항목 | 금액 |
|------|------|
| 착수금 | 200~330만원 (민사 기준) |
| 성공보수 | 배상액의 10~15% |
| 내용증명 작성만 | 20~50만원 |
| 법률 상담 | 무료~5만원 |

### 전략별 비용 효과 분석

#### 1. 변호사 없이 직접 합의 (배상 규모 200만원 이하)

- 비용: 0원
- 장점: 비용 없음
- 단점: 상대방이 시간 끌기 가능, 서류 작성 번거로움

#### 2. 내용증명 + 합의 대행만 의뢰 (추천)

- 비용: **50~100만원**
- 변호사가 해주는 것: 내용증명 발송, 합의 협상 대행, 합의서 작성
- **귀찮은 일 최소화 + 비용 효율 최적**
- 변호사 이름이 들어간 내용증명만으로도 상대방 태도가 달라짐

#### 3. 민사소송까지 풀 대리 (배상 규모 500만원 이상)

- 비용: 착수금 200~330만원 + 성공보수
- 소송 기간: 6개월~1년
- 배상액이 충분히 크지 않으면 역마진

### 결론: 최적 전략

> **합의 기반 + 변호사 내용증명/합의대행**이 가장 효율적입니다.
>
> - 변호사 비용 50~100만원으로 귀찮은 과정 위임
> - 변호사 명의 내용증명으로 압박 효과
> - 합의가 안 되면 소송 전환 여부를 그때 판단

## 배상 극대화를 위한 증거 확보 체크리스트

합의든 소송이든, **인과관계 입증**이 핵심입니다. 아래를 빠짐없이 확보하세요.

### 즉시 해야 할 것 (24시간 이내)

- [ ] **병원 진료** — 증상 발현 즉시 초진 기록 남기기 (식중독 질병코드 확보)
- [ ] **음식점 영수증** 보관 (결제 내역, 카드 명세)
- [ ] **보건소 신고** — 관할 보건소에 식중독 의심 신고 (☎ 1399)
- [ ] **증상 사진/영상** 촬영 (구토, 설사 등)
- [ ] **동행자 확보** — 같이 먹은 사람이 있으면 동일 증상 여부 확인

### 보건소 신고가 중요한 이유

보건소에 신고하면 **역학조사**가 진행됩니다:
- 해당 식당 현장 조사 (위생 상태, 식재료, 조리기구)
- 보존식·식재료 검체 채취 및 검사
- 환자 인체 가검물 검사
- **조사 결과 보고서** = 인과관계 입증의 가장 강력한 증거

> 보건소 역학조사 결과가 있으면 합의 협상에서 압도적으로 유리합니다.

### 치료 과정에서 확보할 것

- [ ] 모든 **진료비 영수증 + 세부내역서**
- [ ] **진단서** (질병코드 포함)
- [ ] 입원 시 **입퇴원 확인서**
- [ ] 일 못한 기간의 **급여명세서** 또는 소득 증빙
- [ ] 통원 **교통비 영수증**
- [ ] 후유증 있으면 **의사 소견서**

## 합의 진행 절차 (타임라인)

### Step 1: 증거 확보 (Day 1~3)

병원 진료 + 보건소 신고 + 영수증 확보

### Step 2: 음식점 배상책임보험 확인 (Day 3~7)

음식점에 연락하여 **음식물배상책임보험** 가입 여부 확인:
- **가입 O** → 보험사에 보험금 청구 (치료비 + 위로금 20~30만원 수준)
- **가입 X** → 직접 합의 또는 변호사 선임

> 음식물배상책임보험의 보험금은 **매우 적습니다** (1인당 20~30만원 수준). 이것과 별도로 추가 배상을 청구하는 것이 핵심입니다.

### Step 3: 변호사 상담 + 내용증명 발송 (Day 7~14)

변호사에게 사건 개요 전달 후:
1. **청구 금액 산정** (치료비 + 일실수익 + 위자료 합산)
2. **내용증명** 발송 — 14일 이내 배상 요구
3. 상대방 반응 대기

### Step 4: 합의 협상 (Day 14~30)

- 상대방이 응하면 → 변호사가 합의서 작성, 배상금 수령
- 상대방이 무시/거부하면 → 소송 전환 여부 판단

### Step 5: (필요시) 한국소비자원 분쟁조정 또는 민사소송

- **소비자원 분쟁조정**: 무료, 1~3개월 소요
- **민사소송**: 6개월~1년, 변호사 착수금 필요

## 실손보험 별도 청구

음식점 배상과 별개로, 본인의 **실손보험**으로도 청구 가능합니다.

| 항목 | 실손보험 | 음식점 배상 |
|------|----------|------------|
| 치료비 | O (중복 청구 가능) | O |
| 위자료 | X | O |
| 일실수익 | X | O |

> 실손보험 청구는 음식점 배상 청구와 **독립적**으로 진행하세요. 둘 다 받을 수 있습니다.

**실손보험 필요 서류**: 질병코드 포함 진단서, 진료비 영수증, 진료비 세부내역서

## 소멸시효

| 유형 | 시효 |
|------|------|
| 불법행위 손해배상 | 피해 안 날로부터 **3년** |
| 제조물 책임 | 손해 안 날로부터 **3년**, 공급일로부터 10년 |

서두를 필요는 없지만, 증거 확보는 빠를수록 좋습니다.

## 요약: 최대 터치 전략

```
1. 즉시: 병원 + 보건소 신고 + 영수증 확보
2. 1주 내: 변호사 상담 (내용증명 + 합의대행 의뢰, 50~100만원)
3. 변호사가 청구 금액 산정: 치료비 + 일실수익 + 위자료 합산
4. 내용증명 발송 → 합의 협상
5. 합의 결렬 시: 소비자원 분쟁조정 (무료) 또는 소송 전환 판단
```

**현실적 기대 금액**:
- 경증 (통원): **130~200만원**
- 중증 (입원): **350~600만원+**
- 300만원은 위자료만의 상한이 아니라, **판례에서 청구한 위자료 금액**일 뿐
- 치료비·일실수익까지 합산하면 그 이상 충분히 가능

> 변호사 내용증명 한 통이 직접 전화 열 통보다 효과적입니다.
