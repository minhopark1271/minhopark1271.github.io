# 임신부 식중독 — 빠른 합의 실전 가이드
{:.no_toc}

쿠팡에서 구매한 식품으로 식중독이 발생했을 때, 변호사 없이 직접 합의로 빠르게 마무리하되, 태아 후유증 추가 청구권만은 확실히 유보하는 전략을 정리합니다.

## 상황

- 임신 중인 배우자가 쿠팡에서 주문한 유명 브랜드 샌드위치 섭취
- 섭취 약 30분 내 극심한 구역감, 두통, 복통, 구토
- 설사는 즉시 발현되지 않았으나, **추후 설사 발생 시 입원 치료 필요**하다는 진단
- 수액 처방. **태아에게 치명적일 수 있다는 진단**
- 실물(샌드위치, 포장재) 폐기 상태. 주문내역·진료기록 보유
- **보호자(남편)**: 스트레스 및 불면증으로 **정신과 방문 예정**

## STEP 2. 쿠팡 클레임 접수

### 접수 채널 — 전화만 가능

쿠팡 전문부서(상품 피해 담당)는 **채팅, 문자, 이메일 모두 불가**이며 **유선 통화만** 운영합니다.

> 전문부서에서는 상세한 상담을 위해 채팅 상담을 운영하고 있지 않습니다. 정보 제공 동의 및 제조사 연결 등 여러 절차에 있어 텍스트 상담으로는 어렵기에 양해 말씀을 드립니다. — 쿠팡 전문부서 답변 (2026-03-13)

**기록 확보 전략**:

| 방법 | 설명 |
|------|------|
| **통화 녹음** | 한국법상 통화 당사자가 녹음하는 것은 **상대 동의 없이도 합법** (통신비밀보호법 제3조 제1항 단서). 굳이 허락을 구할 필요 없음 |
| **통화 후 문자 확인** | 통화에서 합의된 내용을 문자로 "방금 통화 내용 정리하면 ~입니다. 맞으시죠?" 형태로 발송 → 기록화 |
| **앱 채팅 병행** | 일반 고객센터 채팅(마이쿠팡 → 고객센터)으로 "전문부서 통화 내용 확인" 요청 → 텍스트 기록 보조 |

### 현재 진행 상황

- **전문부서 담당자 정희수** 배정됨
- 재연락 예정: **2026-03-13 (오늘) 18시**

### 아이폰 통화 녹음 — 고지 안내 주의

iOS 18 기본 녹음 기능(통화 중 좌측 상단 녹음 버튼)은 **상대에게 "이 통화는 녹음됩니다"라고 자동 안내**가 갑니다. 상대가 경계할 수 있으므로:

| 방법 | 고지 여부 | 방법 |
|------|-----------|------|
| **스피커폰 + 다른 기기 녹음** (추천) | 안내 안 감 | 스피커폰 켜고, 옆에 다른 폰/태블릿/노트북의 **음성 메모** 앱으로 녹음 |
| iOS 18 기본 녹음 | 안내 감 | 통화 중 좌측 상단 녹음 버튼 → 메모 앱에 저장됨 |

> **법적으로는 둘 다 합법**입니다. 한국법상 통화 당사자 녹음은 상대 동의 없이도 합법 (통신비밀보호법 제3조 제1항 단서). 고지 안내는 Apple 정책일 뿐 법적 요건이 아닙니다.

### 추천 톤: "협조적이지만 단호한 피해자"

아래 4가지 선택지 중 **가장 유리한 조합**:

| 선택지 | 판단 |
|--------|------|
| ~~업무 중이라 실시간 통화 어려움, 변호사 통해 이야기~~ | ❌ 과잉 대응. 변호사는 합의 결렬 시의 카드로 남겨둘 것 |
| ~~녹음해도 되냐고 물어보기~~ | ❌ 불필요. 한국법상 당사자 녹음은 동의 불요. 물어보면 상대가 경계만 함 |
| **분쟁으로 가고 싶지 않다** | ✅ 사용. 양측 모두 빠른 합의를 원한다는 공감대 형성 |
| **적절한 보상안 제시해 주시면 검토하겠다** | ✅ 사용. 상대가 먼저 금액을 꺼내게 유도하는 핵심 문장 |

### 대화 스크립트

**오프닝** (차분하고 짧게):

> "안녕하세요. 연락 주셔서 감사합니다. 현재 아내가 임신 중인 상황에서 식중독이 발생해서, 솔직히 저희도 걱정이 많은 상태입니다."

**상황 설명 요청 시**:

> "○○ 샌드위치 섭취 후 30분 내에 극심한 구토, 복통이 발생했고, 응급 진료에서 수액 처방을 받았습니다. 담당 의사가 임신 중 식중독은 태아에 치명적일 수 있다고 했고, 설사 발생 시 입원이 필요하다는 소견입니다."

**핵심 전달** (반드시 포함):

> "저희도 분쟁 상황으로 가고 싶지 않습니다. 다만 임신 중 식중독이라는 게 일반적인 식중독과는 무게가 다르다 보니, **적절한 보상안을 제시해 주시면 검토하겠습니다.**"

### 방어해야 할 것 — 상대의 예상 수법

#### 1. 인과관계 흔들기 — "정말 우리 제품 때문인지 모르잖아요"

| 상대 | 대응 |
|------|------|
| "인과관계를 확인해야 합니다" | "확인 절차는 진행하시되, **보상 논의도 병행**해 주세요. 저희는 진료기록과 주문내역 보유하고 있습니다" |
| "다른 음식 드신 건 없으세요?" | "**해당 샌드위치 섭취 후 30분 이내에 증상이 발생**했습니다. 시간적 인과관계가 명확합니다" |
| "임신 중이라 입덧이나 다른 원인일 수도..." | "담당 의사가 **식중독 의심 진단**을 내렸고, 수액 처방까지 했습니다. 의료 기록에 명확히 남아 있습니다" |

> **원칙**: 의학적 판단은 의사가 이미 내렸다. 쿠팡 상담원이 의료 소견을 부정하게 두지 않기.

#### 2. 실물 없음 공략 — "제품이 없으면 확인이 어렵다"

| 상대 | 대응 |
|------|------|
| "실물 있으세요?" | "없습니다." (짧게 끝내고 즉시 전환) → "다만 **섭취-발병 30분 이내 + 병원 진료기록**이 있습니다" |
| "실물 없으면 확인이 어렵습니다" | "**쿠팡 쪽에서 유통 기록이나 같은 로트 제품으로 확인하실 수 있지 않나요?** 저희는 주문내역과 진료기록을 갖고 있습니다" |
| "제품 검사를 해야 하는데..." | "같은 로트 제품이 쿠팡이나 제조사에 있을 텐데, **그쪽에서 확인 가능하시잖아요**" |
| 계속 물고 늘어지면 | "실물 여부와 관계없이 **진료기록, 주문내역, 발병 시점**이 확보되어 있고, 소비자원 분쟁조정에서도 충분한 수준입니다. **보상 논의를 진행해 주세요.**" |

> **원칙**: 실물 보관은 소비자 의무가 아니다. 변명하지 말고 있는 증거로 전환. 3회 이상 반복하면 끊기.

#### 3. 시간 끌기 — "확인 후 연락드리겠습니다"

| 상대 | 대응 |
|------|------|
| "제조사 확인 후 연락드리겠습니다" | "확인해 주세요. 다만 **쿠팡 직매입(로켓배송) 상품이면 쿠팡도 유통 책임**이 있습니다. **며칠까지 연락 주실 수 있나요?**" |
| "내부 검토가 필요합니다" | "언제까지 회신 주실 수 있는지 **구체적 일정**을 알려주세요" |
| 기한 없이 "확인되는 대로" | "**3영업일 이내로 회신** 부탁드립니다. 그때까지 연락 없으면 소비자원에 피해구제 접수하겠습니다" |

> **원칙**: "확인 후 연락"은 시간 끌기의 시작. 반드시 구체적 기한을 잡고, 기한 넘기면 소비자원 언급.

#### 4. 책임 떠넘기기 — "저희는 플랫폼일 뿐입니다"

| 상대 | 대응 |
|------|------|
| "제조사에 연락해 보셔야 합니다" | "**로켓배송 상품은 쿠팡이 직접 매입·보관·배송**한 거잖아요. 유통 책임이 있습니다" |
| "저희는 중개만 했습니다" | "주문내역에 **쿠팡 직매입으로 표시**되어 있습니다. 제조물책임법상 유통업자도 책임 주체입니다" |
| "제조사와 협의해서 알려드리겠다" | "쿠팡과 제조사 사이의 문제는 쿠팡 내부에서 정리해 주세요. **저희 창구는 쿠팡**입니다" |

> **원칙**: 로켓배송이면 쿠팡이 유통 책임을 진다. 제조사와의 협의는 쿠팡 내부 문제이지 소비자가 관여할 사안이 아님.

#### 5. 금액 최소화 — "치료비 보전해 드리겠습니다"

| 상대 | 대응 |
|------|------|
| "치료비를 보전해 드리겠습니다" | "치료비는 기본이고, **임신부 식중독의 정신적 피해와 태아 추적관찰 비용**도 고려되어야 합니다" |
| "○○만원 정도 어떠세요?" (저가 제시) | "감사합니다만, **임신 중 식중독의 심각성이 반영된 금액은 아닌 것 같습니다.** 재검토 부탁드립니다" |
| "고객님이 원하시는 금액이 있으세요?" | "저희가 산정하기보다, **쿠팡에서 검토하신 보상안을 먼저 제시**해 주시면 검토하겠습니다" |
| "다른 분들은 보통 이 정도에 합의하십니다" | "다른 사례와 저희 상황은 다릅니다. **임신 중 식중독은 일반 식중독과 동일선상에서 비교하기 어렵습니다**" |

> **원칙**: 절대 먼저 금액을 말하지 않는다. 상대가 제시하면 "감사하지만 부족" → 상대가 올리게 만들기.

#### 6. 빠른 종결 유도 — "빨리 마무리하시죠"

| 상대 | 대응 |
|------|------|
| "합의서 보내드릴 테니 서명해 주세요" | "합의서 내용을 **먼저 검토한 후 연락**드리겠습니다" |
| "오늘 중으로 결정해 주시면 빠르게 처리" | "**중요한 사안이라 검토 시간이 필요합니다.** 며칠 후에 답변드리겠습니다" |
| "이 금액은 오늘까지만 유효합니다" | "시한부 제안은 받아들이기 어렵습니다. **정당한 보상이면 시간에 상관없이 합의 의사**가 있습니다" |
| 전화에서 구두 합의를 유도 | "**구두로 합의하기보다 합의서 문서를 검토한 후** 결정하겠습니다" |

> **원칙**: 전화에서 절대 "합의하겠다"고 말하지 않는다. 합의서를 문서로 받아서 검토하겠다고만 하기.

#### 7. 정보 캐내기 — "혹시 ~하셨나요?"

| 상대 | 대응 |
|------|------|
| "보건소 신고는 하셨나요?" | 했으면 → "네." / 안 했으면 → "**검토 중입니다**" (안 했다고 말하면 레버리지 상실) |
| "변호사 상담은 받으셨나요?" | "**필요하면 진행할 예정입니다**" (있든 없든 이렇게 답) |
| "다른 곳에도 접수하셨나요?" | "**지금은 쿠팡과 직접 해결하고 싶습니다**" (다른 루트의 존재를 암시하면서 협조적 태도) |
| "증상이 지금은 괜찮으세요?" | "아직 경과 관찰 중이고, **태아 영향은 더 지켜봐야 합니다**" (괜찮다고 하면 심각성 감소) |

> **원칙**: 없는 것은 "없다"고 하지 말고 "검토/예정/관찰 중"으로. 있는 것은 핵심만 짧게.

#### 8. 감정 흔들기 — "정말 안타깝습니다"

| 상대 | 대응 |
|------|------|
| 과도한 공감 + "저도 부모로서 마음이..." | 감사 인사 후 본론으로 → "감사합니다. 그래서 **구체적인 보상안**은 어떻게 되나요?" |
| "고객님 입장 충분히 이해합니다" 반복 | "이해해 주셔서 감사합니다. 그러면 **그에 맞는 보상안을 제시**해 주세요" |
| "저도 최대한 도와드리고 싶은데 권한이..." | "권한이 있으신 분과 **직접 통화할 수 있을까요?**" |

> **원칙**: 공감에 감사하되 감정에 끌려가지 않기. 공감 표현이 나올 때마다 "그래서 보상안은?"으로 돌아오기.

### 통화 중 절대 하지 말 것

- ❌ 구체적 금액을 먼저 언급
- ❌ "이 정도면 됩니다" 같은 수용 시그널
- ❌ 증거 목록을 상세히 나열 ("진료기록 보유" 정도만)

- ❌ "괜찮아졌다" / "별일 아니다" 같은 상태 호전 시그널
- ❌ "보건소 신고 안 했다" / "변호사 없다" 같은 레버리지 포기 발언

## 합의 불발 시

쿠팡이 성의 없는 금액을 고집하거나 태아 조항을 끝까지 거부하면:

1. **한국소비자원** 피해구제 신청 (consumer.go.kr, 무료) — 기업에 심리적 압박
2. **식약처 1399** 신고 — 행정조사 기록이 협상 레버리지로 활용
3. 그래도 안 되면 **변호사 상담** (법률구조공단 무료 상담 가능)

---

## 참고

- **실손의료보험**: 치료비 별도 청구 가능. 합의금과 무관
- **배상 청구 근거**: 쿠팡 직매입(로켓배송)이면 판매·유통 책임. 마켓플레이스면 제조사에 병행 청구 가능 (제조물책임법)

## 핵심 요약

1. **월요일**: 산모 경과관찰 + 남편 정신과 진료 → 진단서 확보
2. **청구 4축**: 산모 신체 + 산모 정신 + 태아 + 보호자 정신
3. **합의 타겟**: 치료비 전액 + 위자료 **400~500만원** (보호자 위자료·태아 추적 포함)
4. **합의서 최우선**: "태아 후유증 추가 청구권 유보" — 합의금보다 중요
5. **변호사는 최후 카드**: 합의 결렬 시에만 언급
