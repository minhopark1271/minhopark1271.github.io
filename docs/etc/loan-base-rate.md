# 대출 기준금리 종류와 금리 연동 구조
{:.no_toc}

대출 상품을 비교할 때 가장 먼저 마주치는 것이 "기준금리"입니다. 신잔액 COFIX, CD금리, 금융채 등 다양한 기준금리가 있는데, 각각의 차이와 실제 대출금리에 어떻게 연동되는지 정리합니다.

## 기준금리란

은행이 대출 상품의 이자율을 산정할 때 기초로 사용하는 지표금리입니다. 실제 대출금리는 다음 공식으로 결정됩니다.

```
실제 대출금리 = 기준금리 + 가산금리 - 우대금리
```

같은 기준금리를 사용하더라도 은행마다 가산금리와 우대금리가 다르기 때문에, **최종 대출금리는 은행·상품별로 차이**가 납니다.

---

## 기준금리 종류

### 1. COFIX (코픽스)

**자금조달비용지수**(Cost of Fund Index). 국내 8개 주요 시중은행의 자금조달 비용을 가중평균하여 산출합니다. 은행연합회가 **매월 15일 15시**에 공시하며, 16일부터 한 달간 적용됩니다.

4가지 세부 유형이 있습니다.

| 유형 | 특징 | 변동성 |
|------|------|--------|
| **신규취급액 COFIX** | 해당 월 신규 조달자금 기준. 시장금리 반영 빠름 | 높음 |
| **잔액 COFIX** | 누적 전체 자금 기준. 반영 느림, 안정적 | 낮음 |
| **신잔액 COFIX** | 기타예수금·결제성자금(이자 ~0%) 포함. 금리 수준 가장 낮음 | 가장 낮음 |
| **단기 COFIX** | 매주 공시. 단기 자금조달 비용 반영 | 중간 |

> **신잔액 COFIX가 낮은 이유**: 요구불예금(보통예금 등 이자가 거의 없는 자금)을 포함하여 계산하기 때문에 절대 수준이 낮습니다. 다만 은행이 이를 반영하여 가산금리를 더 높게 책정하는 경우가 있습니다.

### 2. CD금리 (91일물)

**양도성예금증서**(Certificate of Deposit) 91일물 유통수익률입니다. 금융투자협회가 **하루 2회** 고시합니다.

- 한국은행 기준금리 변동이 **가장 빠르게** 반영되는 지표
- 주로 **신용대출**의 기준금리로 활용
- 금리 인하기에 가장 빠르게 하락

### 3. 금융채 (은행채 AAA)

은행이 발행하는 **무담보 채권**의 유통금리입니다. 만기별(3개월, 6개월, 1년, 3년 등)로 금리가 다릅니다.

- **일단위 고시**, 기준금리 변동 신속 반영
- 신용대출 및 주담대 혼합형 상품의 기준금리로 사용
- 장기물(3~5년)은 **고정금리 대출**의 기준으로 활용

### 4. MOR (시장조달금리)

Market Opportunity Rate. CD, 금융채, KORIBOR 금리의 **직전 3영업일 단순평균**으로 산출합니다.

- **주 단위** 업데이트 (COFIX보다 빠른 반영)
- KB국민은행 등 일부 은행이 자체 기준금리로 채택
- 시장 실시간 조건을 더 정확히 반영

### 5. KORIBOR

Korea InterBank Offered Rate. **은행 간 무담보 원화 대차 호가금리**입니다. 은행연합회가 매 영업일 11시 공시합니다.

- 대출 기준금리로 직접 사용되기보다 MOR 산출의 구성요소
- 3개월, 6개월, 12개월물 존재

## 대출금리 산출 구조

### 전체 구조

```
기준금리 (COFIX, CD 등)
    +
가산금리 (은행이 결정)
    -
우대금리 (조건 충족 시 감면)
    =
━━━━━━━━━━━━━━━━
실제 대출금리
```

### 가산금리의 구성

은행이 기준금리 위에 붙이는 마진으로, 7가지 요소로 구성됩니다.

| 구성 요소 | 내용 |
|----------|------|
| 리스크 프리미엄 | 금리 변동에 따른 위험 보전 |
| 유동성 프리미엄 | 자금 재조달 비용 |
| **신용 프리미엄** | 차주의 신용등급, 담보 종류 → **가장 큰 변수** |
| 자본 비용 | 예상 손실 대비 적립 비용 |
| 업무원가 | 인건비, 전산 비용 |
| 법적 비용 | 보증기관 출연료, 세금 |
| 기대이익률 | 은행 수익 마진 |

> **핵심**: 기준금리가 낮아도 가산금리가 높으면 최종 금리는 높아집니다. 신잔액 COFIX(2.47%)가 CD금리(2.83%)보다 0.36%p 낮지만, 은행이 가산금리를 그만큼 높게 붙이는 경우가 많습니다.

### 기준금리와 자금조달 원천의 관계

기준금리가 다르다고 해서 은행이 대출에 투입하는 재원이 달라지는 것은 **아닙니다**. 은행은 다양한 원천에서 조달한 자금을 **하나의 풀(pool)로 통합 관리**합니다.

```
예금 (정기예금, 보통예금, 적금 등)
CD (양도성예금증서) 발행
금융채 (은행채) 발행            →  [ 자금 풀 ]  →  대출 실행
은행간 차입 (콜머니)
한국은행 차입
```

CD금리 연동 대출이라고 해서 CD로 조달한 돈이 그 대출에 투입되는 것이 아니고, COFIX 연동 대출이라고 해서 별도 재원이 배정되는 것도 아닙니다. 기준금리는 대출 이자율 산정의 **참조 지표** 역할만 합니다.

이 때문에 은행은 기준금리별로 가산금리를 다르게 책정합니다. 기준금리가 낮으면 가산금리를 높이고, 기준금리가 높으면 가산금리를 낮춰서 **최종 대출금리가 비슷한 수준으로 수렴**하도록 조정합니다. 은행 입장에서 중요한 것은 어떤 지표를 쓰느냐가 아니라, 자금조달 평균비용 대비 **충분한 마진이 확보되는가**입니다.

### 우대금리 조건 (예시)

| 조건 | 감면 폭 (예시) |
|------|---------------|
| 급여/연금 이체 (월 50만원 이상) | -0.1~0.3%p |
| 신용카드 실적 (최근 3개월) | -0.1~0.2%p |
| 자동이체 3건 이상 | -0.1%p |
| 모바일뱅킹 이용 | -0.1%p |
| 적금 잔액 30만원 이상 | -0.1%p |

{: .warning }
> **주의**: 대부분의 은행은 **대출 실행 3개월 이후부터 매월 우대금리를 재산정**합니다. 급여이체를 중단하거나 카드 실적이 미달하면 우대 혜택이 소멸되어 금리가 올라갑니다.

### 실제 은행 대출금리 사례

2026년 3월 기준, 5대 시중은행 변동금리 주택담보대출 금리 범위입니다.

| 은행 | 신규취급액 COFIX 기준 | 신잔액 COFIX 기준 |
|------|---------------------|------------------|
| KB국민 | 4.15% ~ 5.55% | 4.14% ~ 5.54% |
| 우리 | 4.37% ~ 5.57% | 4.21% ~ 5.41% |
| 5대은행 전체 | 3.66% ~ 6.06% 범위 | - |

→ 기준금리(2.47~2.82%)에 가산금리 약 1.3~3.2%p가 붙어 최종 금리가 형성됩니다.

## 변동금리 vs 고정금리

### 금리 유형별 비교

| 구분 | 변동금리 | 고정금리 | 혼합금리 |
|------|---------|---------|---------|
| 금리 변동 여부 | 3/6/12개월 주기로 재조정 | 만기까지 고정 | 초기 3~10년 고정, 이후 변동 |
| 사용 기준금리 | COFIX, CD, 금융채 (단기물) | 금융채(장기물), 국고채 | 초기: 고정금리 / 이후: 변동 |
| 유리한 시기 | 금리 인하기 | 금리 인상기 | 중간 성격 |
| 초기 금리 수준 | 상대적으로 낮음 | 상대적으로 높음 | 중간 |

### 금리변동 주기

변동금리 대출을 선택할 때 금리 재조정 주기도 중요합니다.

| 주기 | 특징 |
|------|------|
| **3개월** | 시장 변화 가장 빠르게 반영. 금리 하락기에 유리하나 불확실성 높음 |
| **6개월** | 중간 수준. 가장 보편적 |
| **12개월** | 상대적으로 안정적. 금리 하락 시 반영 느림 |

## 기준금리 선택 가이드

### 금리 인하기 (현재 상황)

한국은행 기준금리 2.50% 동결 중이나, 추가 인하 전망이 있는 상황입니다.

- **CD금리/금융채 연동**이 유리 → 기준금리 인하가 매일 반영
- COFIX는 월 1회 반영이라 인하 효과가 1개월 이상 지연

### 금리 인상기

- **신잔액 COFIX**가 유리 → 변동성 가장 낮아 인상 속도 완만
- 혹은 **고정금리**로 확정하여 리스크 차단

### 장기 안정성 중시

- **잔액 COFIX** 또는 **신잔액 COFIX** → 변동폭 작아 예측 가능

### 핵심 체크리스트

{: .important }
> 1. 기준금리만 비교하지 말고 **최종 대출금리**를 비교하세요
> 2. 신잔액 COFIX가 가장 낮지만, 가산금리가 높게 책정될 수 있습니다
> 3. 우대금리 조건을 장기간 유지할 수 있는지 확인하세요
> 4. 금리 인하기에는 변동금리 + 짧은 재조정 주기가, 인상기에는 고정금리가 유리합니다
