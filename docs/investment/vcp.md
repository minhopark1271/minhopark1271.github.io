# VCP (Volatility Contraction Pattern)
{:.no_toc}

마크 미너비니(Mark Minervini)가 체계화한 변동성 수축 패턴. 가격 변동폭과 거래량이 점진적으로 줄어들며 매도 공급이 소진되는 과정을 포착해, 저위험 고수익 브레이크아웃 진입점을 제공하는 패턴이다.

## 1. VCP란 무엇인가

VCP(Volatility Contraction Pattern, 변동성 수축 패턴)는 **강한 상승 추세에 있는 주식이 조정 과정에서 점점 작아지는 되돌림(contraction)을 반복하며, 거래량도 함께 줄어드는 현상**을 패턴화한 것이다.

핵심 전제는 단순하다:

> "매번 되돌림이 작아진다는 것은 매도하려는 사람(공급)이 점점 사라지고 있다는 뜻이다. 공급이 소진되면, 작은 수요만으로도 가격이 크게 움직인다."

이 개념은 마크 미너비니가 저서 *Trade Like a Stock Market Wizard*와 *Think and Trade Like a Champion*에서 체계화했으며, 윌리엄 오닐(William O'Neil)의 컵 앤 핸들(Cup & Handle) 패턴과 근본 원리를 공유한다. 미너비니는 이 전략으로 전미 투자 대회(U.S. Investing Championship)에서 2021년 334% 이상의 검증된 수익률을 기록하며 우승했다.

## 2. VCP의 구조

### 2.1 기본 형태

VCP는 2~6개의 연속적인 **수축(contraction)**으로 구성된다. 각 수축은 이전보다 **되돌림 폭이 작고**, **거래량도 감소**한다.

```
가격 ▲
     │    ╭──╮
     │   ╭╯  ╰╮     ╭─╮
     │──╯     ╰╮   ╭╯ ╰╮  ╭╮
     │         ╰──╯    ╰─╯╰──── → 브레이크아웃
     │
     │  ┌─1차─┐ ┌─2차─┐ ┌3차┐
     │   -25%     -12%    -5%    ← 되돌림 폭 축소
     │
     └────────────────────────→ 시간
```

| 구성 요소 | 설명 |
|----------|------|
| **1차 수축** | 가장 큰 되돌림 (15~35%). 직전 상승 후 차익 실현 매물 |
| **2차 수축** | 중간 되돌림 (8~15%). 남은 매도 물량 소화 |
| **3차+ 수축** | 작은 되돌림 (3~8%). 매도 공급 거의 소진 |
| **피벗 라인** | 최종 수축의 고점. 브레이크아웃 기준선 |

### 2.2 시간 프레임

전형적인 VCP 베이스는 **3~12주**에 걸쳐 형성된다. 너무 짧으면(1주 미만) 충분한 공급 소진이 일어나지 않고, 너무 길면(6개월 이상) 추세 자체가 약화될 수 있다.

### 2.3 수축 단계별 체크리스트

각 수축을 식별할 때 확인해야 할 요소:

| 단계 | 가격 행동 | 거래량 | 의미 |
|------|----------|--------|------|
| **수축 시작** | 고점에서 하락 시작 | 평균 이상 가능 | 차익 실현/매도 압력 유입 |
| **수축 저점** | 이전 수축 저점보다 높은 저점 형성 | 평균 이하로 감소 | 매도 의사 약화 |
| **수축 종료** | 이전 고점 근처까지 반등 | 여전히 평균 이하 | 공급 흡수 완료 |
| **다음 수축** | 더 작은 폭의 조정 | 더 적은 거래량 | 잔여 공급 소진 중 |

## 3. 거래량 축소의 두 가지 얼굴: 매도 압력 감소 vs 관심 실종

VCP의 핵심은 거래량 감소인데, **모든 거래량 감소가 긍정적인 것은 아니다**. 건강한 VCP 형성(매도 압력 감소)과 단순한 관심 실종(dead stock)을 구분하는 것이 가장 중요한 스킬이다.

### 3.1 매도 압력 감소 (건강한 수축)

**"파는 사람이 사라지는 것"**

- 강한 선행 상승 이후 조정 과정에서 발생
- 약한 손(weak hands)이 조정 초기에 매도를 완료
- 이후 수축마다 매도할 사람이 줄어 거래량 자연 감소
- **기관 투자자가 조용히 매집(accumulation)하는 흔적**이 남음

특징:
- 되돌림 폭이 **점진적으로 축소** (예: 25% → 12% → 5%)
- 하락일 거래량 < 상승일 거래량
- 가격이 주요 이동평균선(50일, 150일, 200일) **위에** 유지
- 상대강도(RS) 70 이상 유지
- OBV(On Balance Volume)가 하락하지 않거나 소폭 상승

### 3.2 관심 실종 (Dead Stock)

**"사는 사람도, 파는 사람도 없는 것"**

- 상승 추세가 이미 무너졌거나 존재하지 않는 상태
- 시장 참여자 전반의 무관심
- 기관의 매집 흔적 없음
- **브레이크아웃이 발생해도 거래량이 따라오지 않음**

특징:
- 가격이 이동평균선 **아래**에서 횡보
- 상대강도(RS) 50 이하로 약함
- 되돌림 폭이 줄어들지 않고 불규칙
- 상승일과 하락일의 거래량 차이 없음
- OBV가 횡보하거나 하락 추세

### 3.3 구분 체크리스트

| 판단 기준 | 매도 압력 감소 (VCP) | 관심 실종 (Dead Stock) |
|----------|---------------------|----------------------|
| **선행 추세** | 강한 상승 추세 존재 | 추세 불명확 또는 하락 |
| **이동평균 관계** | 가격 > 50일 > 150일 > 200일 | 이동평균선 아래 또는 역배열 |
| **상대강도 (RS)** | 70 이상 | 50 이하 |
| **되돌림 패턴** | 점진적 축소 (규칙적) | 불규칙, 축소 패턴 없음 |
| **52주 고점 대비** | 25% 이내 | 50% 이상 하락 |
| **하락일 vs 상승일 거래량** | 하락일 거래량 < 상승일 | 차이 없거나 하락일이 더 큼 |
| **OBV 방향** | 유지 또는 상승 | 횡보 또는 하락 |
| **브레이크아웃 시도** | 거래량 급증 동반 | 거래량 변화 없음 |
| **기관 흔적** | Pocket Pivot, VDU 관찰됨 | 없음 |

## 4. VDU와 Pocket Pivot: 기관의 발자국

VDU(Volume Dry Up)와 Pocket Pivot은 Chris Kacher와 Gil Morales가 개발한 개념으로, VCP 내에서 기관 투자자의 활동을 포착하는 핵심 도구이다.

### 4.1 VDU (Volume Dry Up)

**공급 소진의 신호**

VDU는 주식이 지지 영역(10일, 20일, 50일 이동평균선 또는 이전 베이스 상단)으로 되돌아올 때, **거래량이 평균의 50% 이하**로 극단적으로 줄어드는 현상이다.

- 일일 거래량이 50일 평균 거래량의 **35~50% 이하**
- 조정 중 가격이 주요 이동평균선 근처에서 지지
- 매도할 사람이 거의 남지 않았다는 직접적 증거

### 4.2 Pocket Pivot

**수요 확인의 신호**

VDU가 "공급이 없다"는 것을 보여준다면, Pocket Pivot은 "수요가 있다"는 것을 확인해 준다.

조건:
- **양봉(green candle)** 마감
- 해당 일의 거래량이 **직전 10일 이내 모든 음봉일의 거래량보다 큼**
- 10일 또는 20일 이동평균선 근처에서 발생

이는 수요가 최근의 모든 공급보다 강하다는 것을 수치적으로 증명한다.

### 4.3 VDU + Pocket Pivot의 활용

```
[조정 구간]
     거래량 극도로 줄어듦 → VDU 확인
          ↓
     양봉 + 최근 10일 음봉 거래량 초과 → Pocket Pivot 확인
          ↓
     "공급 소진 + 수요 유입" → 매수 시그널
```

VCP의 3차 이후 수축 구간에서 VDU와 Pocket Pivot이 연속으로 나타나면, 브레이크아웃 임박의 강력한 신호로 해석한다.

## 5. 미너비니의 트렌드 템플릿: VCP 전제 조건

VCP는 아무 차트에서나 찾는 것이 아니다. **반드시 강한 상승 추세(Stage 2)에 있는 주식에서만** 유효하다. 미너비니는 8가지 트렌드 템플릿 기준을 제시했다.

### 5.1 8가지 기준

| # | 기준 | 조건 |
|---|------|------|
| 1 | 가격 위치 (장기) | 현재 가격 > 150일 이평선 **AND** 200일 이평선 |
| 2 | 이평선 정배열 (중기) | 150일 이평선 > 200일 이평선 |
| 3 | 200일선 방향 | 200일 이평선이 **최소 1개월 이상** 상승 추세 (이상적으로 4~5개월) |
| 4 | 이평선 정배열 (단기) | 50일 이평선 > 150일 이평선 **AND** 200일 이평선 |
| 5 | 가격 위치 (단기) | 현재 가격 > 50일 이평선 |
| 6 | 52주 저점 대비 | 현재 가격이 52주 저점 대비 **최소 25~30% 이상** |
| 7 | 52주 고점 대비 | 현재 가격이 52주 고점 대비 **25% 이내** (신고가에 가까울수록 좋음) |
| 8 | 상대강도 (RS) | RS 랭킹 **70 이상** (시장의 상위 30%) |

### 5.2 이평선 정배열의 의미

```
가격  ────────── (현재가)
50일  ────────
150일 ──────
200일 ─────     ← 모두 상승 기울기

이 정배열이 유지되는 상태에서만 VCP가 유효
```

8가지 기준을 모두 충족하는 주식은 **기관이 매집하고 있는 Stage 2 상승 추세** 안에 있다는 것이 확인된 상태이며, 이 위에서 형성되는 VCP만이 신뢰도 높은 매매 셋업이다.

## 6. 함께 봐야 하는 기술적 지표

VCP는 단독 패턴이 아니라 여러 지표와 함께 종합적으로 판단해야 한다.

### 6.1 거래량 지표

| 지표 | 역할 | VCP에서의 활용 |
|------|------|---------------|
| **RVOL (Relative Volume)** | 현재 거래량 / 평균 거래량 | 수축 중 RVOL < 0.5 → 공급 소진 신호. 브레이크아웃 시 RVOL > 1.5 → 수요 확인 |
| **OBV (On Balance Volume)** | 거래량의 누적 방향 | VCP 형성 중 OBV 하락 없이 유지/상승 → 기관 매집 중 |
| **CMF (Chaikin Money Flow)** | 매수/매도 압력 측정 | 0선 위 유지 → 매수 압력 우세. 0선 아래 하락 → 매도 압력 (경고) |
| **A/D Line (Accumulation/Distribution)** | 종가의 고저 범위 내 위치 × 거래량 | 가격 횡보 + A/D 상승 → 숨은 매집 |

### 6.2 변동성 지표

| 지표 | 역할 | VCP에서의 활용 |
|------|------|---------------|
| **BBW (Bollinger Band Width)** | 볼린저 밴드 폭 측정 | BBW 지속 감소 → 변동성 수축 확인. 볼린저 스퀴즈와 VCP 형성이 겹치면 신뢰도 높음 |
| **ATR (Average True Range)** | 평균 변동폭 | ATR 감소 추세 → 수축 진행 확인. 손절 설정 시 1.5~3 × ATR 활용 |

### 6.3 추세/강도 지표

| 지표 | 역할 | VCP에서의 활용 |
|------|------|---------------|
| **RS (Relative Strength)** | 시장 대비 상대 성과 | RS 70 이상 유지 필수. RS가 VCP 형성 중 하락하면 셋업 무효화 |
| **RS Line** | 주가 / 지수 비율 차트 | RS Line이 신고가 근처 → 시장보다 강함 (최상의 셋업) |
| **이동평균선 (50/150/200일)** | 추세 확인 | 정배열 유지 확인. 50일선이 지지선 역할을 하는지 확인 |
| **ADX** | 추세 강도 | ADX 25 이상 → 추세 존재. VCP는 추세가 있는 상태에서 형성되어야 함 |

### 6.4 지표 활용 종합 예시

```
VCP 셋업 검증 체크리스트:

✅ 트렌드 템플릿 8가지 통과
✅ 2회 이상 수축, 각 수축 폭 감소
✅ 거래량 수축 확인 (RVOL < 0.5)
✅ OBV 유지 또는 상승
✅ BBW / ATR 감소 추세
✅ RS 70 이상 유지
✅ VDU 또는 Pocket Pivot 관찰 (보너스)

→ 피벗 라인 돌파 + 거래량 급증 시 진입
```

## 7. 브레이크아웃 진입과 리스크 관리

### 7.1 진입 조건

| 조건 | 기준 |
|------|------|
| **가격** | 마지막 수축 구간의 고점(피벗 라인) 돌파 |
| **거래량** | 50일 평균 거래량의 **1.5배 이상** |
| **타이밍** | 장 시작 30분~1시간 이내 돌파가 이상적 |

### 7.2 손절 설정

- **마지막 수축의 저점** 바로 아래에 손절 설정
- VCP가 잘 형성되었다면 이 손절 거리는 매우 좁음 (3~7%)
- ATR 기반: 피벗 라인 - (1.5 × ATR)

### 7.3 수익 관리

- 진입 후 첫 목표: 직전 상승폭만큼의 확장 (measured move)
- 후행 손절(trailing stop)을 10일 또는 21일 이동평균선에 설정
- 3:1 이상의 위험/보상 비율 확보가 이상적

## 8. VCP의 실패 패턴과 주의사항

### 8.1 흔한 실패 유형

| 실패 유형 | 설명 | 경고 신호 |
|----------|------|----------|
| **거래량 없는 돌파** | 피벗 돌파 시 거래량 미동반 | RVOL < 1.0에서 돌파 |
| **4차+ 수축 후 돌파** | 수축이 너무 많으면 에너지 소진 | 5~6차 수축까지 가면 실패 확률 증가 |
| **RS 하락 중 형성** | 시장보다 약해지는 주식 | RS가 VCP 형성 중 70 아래로 하락 |
| **이평선 역배열 전환** | 추세 자체가 무너짐 | 50일선이 150일선 아래로 교차 |
| **폭이 줄지 않는 수축** | 각 되돌림이 비슷하거나 커짐 | 공급 소진 아닌 분배(distribution) 가능성 |

### 8.2 자주 하는 실수

1. **트렌드 템플릿 무시**: VCP 형태만 보고 추세 확인을 건너뜀
2. **너무 이른 진입**: 피벗 돌파 전에 "예상 매수" → 손절 위치가 넓어짐
3. **거래량 확인 생략**: 형태는 맞지만 거래량 패턴이 동반되지 않은 가짜 VCP
4. **관심 실종 주식을 VCP로 착각**: 섹션 3의 체크리스트로 반드시 검증

## 9. 요약

VCP는 **공급 소진의 수학적 증거**이다. 매번 되돌림이 작아지고 거래량이 줄어드는 것은 "팔고 싶은 사람이 거의 없다"는 것을 의미하며, 이 상태에서 수요가 유입되면 폭발적인 상승이 발생한다.

핵심 원칙:

1. **추세가 먼저**: 트렌드 템플릿 8가지 기준을 통과한 종목에서만 VCP를 찾을 것
2. **수축의 질**: 되돌림 폭 축소 + 거래량 감소가 반드시 동반되어야 함
3. **관심 실종과 구분**: RS, 이동평균 관계, OBV로 건강한 수축인지 검증
4. **확인 후 진입**: 피벗 돌파 + 거래량 급증이라는 두 가지 확인 후에만 진입
5. **좁은 손절**: VCP의 최대 장점은 마지막 수축 저점이 가까워 위험/보상 비율이 우수하다는 것
