# 가격방향성 (ADX, DI)
{:.no_toc}

ADX(14)는 +DI와 -DI의 차이(우위)를 정규화한 DX를 만든 뒤, 그 DX를 Wilder Smoothing(14)으로 평활한 값입니다. **ADX는 방향이 아니라 "추세 강도"**를 나타냅니다.

## 전체 계산 흐름

```
고가/저가/종가
    │
    ├─→ True Range (TR)         ── 변동성 측정
    │
    ├─→ +DM / -DM               ── 방향성 원재료
    │       │
    │       ▼
    │   Wilder Smoothing (14)    ── 평활화
    │       │
    │       ▼
    ├─→ +DI / -DI               ── 방향성 지수 (0~100)
    │       │
    │       ▼
    │      DX                    ── 우위 정규화 (0~100)
    │       │
    │       ▼
    └─→  ADX(14)                 ── 추세 강도 (0~100)
```

---

## 입력 데이터

각 시점 t에 대해 필요한 데이터:

| 데이터 | 설명 |
|--------|------|
| H_t | 고가 (High) |
| L_t | 저가 (Low) |
| C_t | 종가 (Close) |
| H_(t-1), L_(t-1), C_(t-1) | 전일 고가, 저가, 종가 |

기간 파라미터: **n = 14** (Wilder의 원래 설정)

---

## 1단계: True Range (TR) — 변동성 측정

갭(전일 종가와 당일 시가 사이의 차이)까지 포함한 **실제 변동폭**을 측정합니다.

```
TR_t = max( H_t - L_t,  |H_t - C_(t-1)|,  |L_t - C_(t-1)| )
```

세 가지 후보 중 가장 큰 값을 선택합니다:

| 후보 | 의미 |
|------|------|
| H_t - L_t | 당일 고저 차이 (갭 없는 경우) |
| \|H_t - C_(t-1)\| | 전일 종가 대비 당일 고가 (위쪽 갭) |
| \|L_t - C_(t-1)\| | 전일 종가 대비 당일 저가 (아래쪽 갭) |

**의미**: 해당 봉에서의 실질적인 변동폭으로, 이후 방향성 지표를 정규화할 때 분모 역할을 합니다.

---

## 2단계: Directional Movement (+DM, -DM) — 방향성 원재료

전일 대비 고가/저가가 얼마나 **확장**했는지를 비교합니다.

```
UpMove_t   = H_t - H_(t-1)     (고가 확장폭)
DownMove_t = L_(t-1) - L_t     (저가 확장폭)
```

둘 중 **더 큰 방향만 채택**하여 노이즈를 억제합니다:

### +DM (상승 방향성)

```
+DM_t = UpMove_t    (UpMove_t > DownMove_t 이고 UpMove_t > 0 일 때)
+DM_t = 0           (그 외)
```

### -DM (하락 방향성)

```
-DM_t = DownMove_t  (DownMove_t > UpMove_t 이고 DownMove_t > 0 일 때)
-DM_t = 0           (그 외)
```

**핵심 직관**:

- **+DM**: 위로 새 고가를 만든 힘 (상방 확장)
- **-DM**: 아래로 새 저가를 만든 힘 (하방 확장)
- 같은 봉에서 +DM과 -DM이 동시에 커지는 것을 막기 위해 **한 방향만 인정**합니다
- 고가도 저가도 확장하지 않았다면 둘 다 0입니다

---

## 3단계: Wilder Smoothing (n=14) — 표준 평활화

ADX 계열은 일반적인 SMA가 아니라 **Wilder 방식의 재귀형 평활(EMA와 유사)**을 사용합니다.

### 초기값: 첫 14개의 합

```
SmTR_14     = TR_1 + TR_2 + ... + TR_14
Sm(+DM)_14  = +DM_1 + +DM_2 + ... + +DM_14
Sm(-DM)_14  = -DM_1 + -DM_2 + ... + -DM_14
```

### 이후 갱신 (t > 14)

```
SmTR_t     = SmTR_(t-1)    - SmTR_(t-1)/14    + TR_t
Sm(+DM)_t  = Sm(+DM)_(t-1) - Sm(+DM)_(t-1)/14 + +DM_t
Sm(-DM)_t  = Sm(-DM)_(t-1) - Sm(-DM)_(t-1)/14 + -DM_t
```

**직관**: 이전 누적치에서 1/14씩 서서히 빼고, 오늘 값을 더해서 부드럽게 만듭니다. 이 방식은 EMA의 decay factor `(1 - 1/n)`과 동일한 효과를 가지며, 오래된 데이터가 자연스럽게 소멸합니다.

---

## 4단계: +DI / -DI — 방향성 지수 (0~100)

TR(변동성)로 나누어 스케일을 맞춘 **방향성 지수**입니다.

```
+DI_t = 100 × Sm(+DM)_t / SmTR_t
-DI_t = 100 × Sm(-DM)_t / SmTR_t
```

| 조건 | 의미 |
|------|------|
| +DI가 높다 | 최근 14기간 기준, 상승 방향 확장(새 고가)이 변동성 대비 우세 |
| -DI가 높다 | 최근 14기간 기준, 하락 방향 확장(새 저가)이 변동성 대비 우세 |

**방향(상승/하락) 판단은 DI로 합니다.** ADX가 아닌 +DI와 -DI의 상대적 크기가 방향을 결정합니다.

---

## 5단계: DX — 우위 정규화 (0~100)

+DI와 -DI의 차이를 총량 대비로 정규화한 값입니다.

```
DX_t = 100 × |+DI_t - -DI_t| / (+DI_t + -DI_t)
```

| 구성 | 역할 |
|------|------|
| 분자: \|+DI - -DI\| | 우세 격차 (얼마나 벌어졌는가) |
| 분모: +DI + -DI | 방향성 총량 (두 힘의 합) |

**직관**: "한쪽이 얼마나 압도적인지"를 0~100으로 표현합니다. 두 DI가 비슷하면 DX는 낮고, 한쪽이 크게 우세하면 DX는 높습니다.

---

## 6단계: ADX(14) — 추세 강도

DX의 변동을 줄이기 위해 **다시 Wilder Smoothing을 적용**한 최종 지표입니다.

### 첫 ADX: 초기 14개 DX의 평균

```
ADX_첫값 = (DX_1 + DX_2 + ... + DX_14) / 14
```

### 이후 갱신

```
ADX_t = (ADX_(t-1) × 13 + DX_t) / 14
```

**참고**: +DI/-DI를 만들려면 최소 14개 봉이 필요하고, DX도 그 이후부터 나오므로 실무적으로 ADX는 **대략 28번째 봉 근처부터** 안정적으로 형성됩니다.

---

## 해석 가이드

### ADX는 "방향"이 아니라 "강도"

ADX 자체는 상승/하락 여부를 알려주지 않습니다. 오직 **추세의 강도**만 나타냅니다.

| ADX 변화 | 의미 |
|----------|------|
| ADX 상승 | 추세 강도 강화 (한쪽 우위가 더 뚜렷해지는 중) |
| ADX 하락 | 추세 강도 약화 (우위 축소, 횡보화 가능성) |

> 가격이 오르는데 ADX가 떨어질 수 있습니다.
> 이는 오르긴 하지만 "압도성(추세 강도)"이 줄어드는 중이라는 의미입니다.

### 방향은 +DI / -DI로 판단

| 조건 | 해석 |
|------|------|
| +DI > -DI | 상승 우세 |
| -DI > +DI | 하락 우세 |

단타에서는 DI 교차가 잦아 휩쏘(whipsaw)가 발생할 수 있어, 진입 신호보다는 **필터**로 사용하는 경우가 많습니다.

### 자주 쓰이는 ADX 레벨 감각

| ADX 레벨 | 일반적 해석 |
|----------|------------|
| < 15~20 | 추세 약함 (레인지/횡보 가능성) |
| 20~25 이상 | 추세 존재 가능성 |
| 40 이상 | 강한 추세 (과열/연장 동반 가능) |

**주의**: 자산과 타임프레임마다 최적 임계값이 다르므로 고정된 진리로 사용해서는 안 됩니다. 백테스트를 통해 해당 자산에 맞는 기준을 찾아야 합니다.

## 실전 활용

### 환경 필터로서의 ADX

ADX는 진입 트리거보다 **추세/비추세 환경 판별**에 활용할 때 기대값이 좋아집니다.

| 시나리오 | 해석 |
|----------|------|
| ADX 낮다가 상승 전환 + (+DI 우세) | 새로운 상승 추세 시작 가정 |
| ADX 낮다가 상승 전환 + (-DI 우세) | 새로운 하락 추세 시작 가정 |
| ADX 높았다가 꺾이며 하락 | 추세 약화, 레인지 전환 가능성 |
| ADX 지속 상승 중 | 기존 추세 지속, 추세 추종 전략 유리 |

### 다른 지표와의 조합

- **ADX + RSI**: ADX로 추세 존재 여부를 확인 후, RSI 과매수/과매도 신호 필터링
- **ADX + 이동평균 교차**: ADX가 낮은 구간에서의 이동평균 교차는 무시 (횡보 구간의 휩쏘)
- **ADX + 볼린저밴드**: ADX 낮은 구간에서 밴드 수축 → 추후 추세 발생 대비

### 주의사항

- ADX는 **후행 지표**입니다. Wilder Smoothing 공식 `ADX_t = (ADX_(t-1) × 13 + DX_t) / 14`에서 오늘의 DX는 전체의 1/14(약 7%)만 반영되고 나머지 93%는 이전 누적값입니다. 이 평활을 2번 거치므로 가격 급변 시 ADX가 따라오려면 여러 봉이 누적되어야 합니다
- 따라서 추세 시작 시점을 포착하기보다는 **추세가 이미 진행 중인지** 확인하는 데 유용합니다
- 단독 사용보다 다른 지표와 조합하여 **확인(confirmation) 용도**로 활용하는 것이 효과적입니다
