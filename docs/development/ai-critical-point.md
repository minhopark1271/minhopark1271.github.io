# AI는 정말 임계점을 넘었는가
{:.no_toc}

2024년 12월을 전후로, AI 업계 안팎에서 "임계점(critical point)을 지났다"는 목소리가 쏟아지고 있습니다. 단순한 마케팅이 아니라, 숫자와 사례가 뒷받침하는 구조적 전환입니다. 이 글에서는 분야별로 **파괴적이고 실감나는 사례**를 모아 정리합니다.

## 왜 "임계점"인가

AI의 발전은 점진적이지 않았습니다. 2024년 말~2025년에 걸쳐 여러 지표가 **비선형적 도약**을 보여줍니다.

| 지표 | 2023~2024 초 | 2025 | 변화폭 |
|------|-------------|------|--------|
| SWE-bench (코딩 벤치마크) | 4.4% | 71.7% | +67pp |
| AI 코딩 시장 규모 | <span>$</span>550M | <span>$</span>4B | 7.3배 |
| Cursor ARR | <span>$</span>100M | <span>$</span>2B | 20배 |
| FDA AI 의료기기 승인 | 6건 (2015) | 223건 (2023) | 37배 |
| AI 신약개발 VC 투자 | - | <span>$</span>3.3B (2024) | - |

Geoffrey Hinton은 Agentic AI의 등장을 **"step change"**라 표현했고, Andrej Karpathy는 2026년 1월 "나는 이제 대부분 영어로 프로그래밍한다"며 **소프트웨어 엔지니어링의 상전이(phase shift)**를 선언했습니다.

## 1. 코딩: 4.4%에서 71.7%로

### SWE-bench의 폭발적 상승

SWE-bench는 실제 GitHub 이슈를 AI가 자율적으로 해결하는 벤치마크입니다. 불과 1년 만에 정답률이 **4.4% → 71.7%**로 뛰었습니다. 12개월 전에는 사람만 할 수 있던 작업을 이제 AI가 일상적으로 처리합니다.

2025년 IMO에서 금메달 수준을 달성한 모델들은 SWE-bench Verified에서도 75~80% 이상을 기록합니다. OpenAI가 "벤치마크가 포화됐다"며 SWE-bench Verified 평가를 중단할 정도입니다.

### Cursor: 역대 최고속 SaaS 성장

AI 코드 에디터 Cursor는 **역대 SaaS 기업 중 가장 빠른 성장 속도**를 기록했습니다.

- 2024년: ARR <span>$</span>100M
- 2025년: ARR <span>$</span>1.2B (전년 대비 1,100% 성장)
- 2026년 3월: ARR **<span>$</span>2B 돌파** (3개월 만에 2배)
- Fortune 500 기업의 **절반 이상**이 사용
- OpenAI, Midjourney, Perplexity, Shopify 개발자들이 주요 사용층

Wiz, Deel, Ramp 등 기존 최고속 SaaS 기업의 기록을 모두 경신했습니다.

### Devin: 자율 코딩 에이전트

Cognition Labs의 Devin은 **"최초의 AI 소프트웨어 엔지니어"**를 표방합니다.

- 코드 마이그레이션: 사람 30~40시간 → Devin 3~4시간 (**10배**)
- Goldman Sachs: 12,000 개발자 + Devin → **14,400명 수준 산출** (20% 효율 향상)
- Nubank: 엔지니어링 시간 **12배**, 비용 **20배** 절감
- 2025년 4월 가격 인하: <span>$</span>500/월 → **<span>$</span>20/월** (진입장벽 대폭 하락)

복잡한 창의적 작업에는 아직 한계가 있지만, 마이그레이션·테스트 작성·취약점 수정 등 **반복적이고 명확한 작업**에서는 이미 실전 배치 수준입니다.

### Vibe Coding: 코드 모르는 사람이 앱을 만든다

Karpathy가 2025년 2월 명명한 **"Vibe Coding"** — 자연어로 의도를 설명하면 AI가 코드를 생성하는 방식입니다.

- Anthropic 공식: Claude Code가 자체 코드의 **약 90%를 직접 작성**
- Scientific American이 특집으로 다룰 정도로 비개발자의 앱 개발이 현실화
- Sam Altman의 예측 "1인 10억 달러 기업"이 수학적으로 확실해지는 시대
- Peter Steinberger: Claude Code로 만든 OpenClaw → OpenAI에 **수억 달러 추정 인수**

50명이 필요하던 엔지니어링 팀이 **5명 이하**로 축소되는 사례가 나타나기 시작했습니다.

## 2. 영상·이미지: 스튜디오 없이 영화를 만든다

### AI 영상 생성의 도약

| 모델 | 출시 | 핵심 역량 |
|------|------|----------|
| **Sora 2** (OpenAI) | 2025.09 | 20초 영상, 네이티브 싱크 오디오, 물리 법칙 준수 |
| **Veo 3** (Google) | 2025.05 | **4K 해상도**, 1분 영상, 시네마토그래피 자동 적용 |

Sora 2는 OpenAI가 **"영상 생성의 GPT-3.5 모먼트"**라고 부른 전환점입니다. 공이 백보드에 맞으면 실제 물리 궤적을 따르고, 대화 립싱크가 자동으로 맞춰집니다.

Veo 3는 텍스트 하나로 **4K 해상도 영상 + 동기화된 오디오**를 동시에 생성합니다. 트래킹 샷, 피사계 심도 등 영화학교에서 배울 기법을 자동 적용합니다.

대규모 스튜디오 예산 없이도 **시네마 퀄리티 영상**을 만들 수 있는 시대가 열렸습니다.

### 이미지 생성: 실사와 구분 불가

2025년 현재, AI 생성 이미지와 실제 사진의 경계가 사실상 사라졌습니다.

- **Flux 2 Pro**: 피사계 심도, 렌즈 왜곡, 색수차, 필름 그레인까지 카메라 광학 특성을 재현
- **Midjourney**: 예술적 스타일에서 여전히 최강
- 손가락, 눈동자 등 기존 AI 이미지의 약점이 대부분 해결

## 3. 수학·추론: 올림피아드 금메달

### IMO 2025 금메달

2025년 국제수학올림피아드(IMO)에서 AI가 **금메달 수준**을 달성했습니다.

- **OpenAI**: 범용 언어 모델이 6문제 중 5문제 정답, 42점 만점 중 **35점**
- **Google DeepMind**: Gemini Deep Think가 자연어로 직접 수학 증명을 생성, **4.5시간** 제한 시간 내 완료

특히 OpenAI의 성과가 주목할 만합니다. DeepMind의 2024년 은메달은 수학 전용 시스템(AlphaProof, AlphaGeometry)으로 달성했지만, OpenAI는 **범용 모델**로 금메달을 달성했습니다.

### AIME 벤치마크

미국 수학 경시대회 AIME에서 2025년 초 AI 시스템이 **최상위 인간을 일상적으로 능가**합니다. 2022~2023년에 초등 수준 수학(GSM8K)이 포화되더니, 2년 만에 올림피아드급까지 도달한 것입니다.

## 4. 과학·의료: 신약 18개월, 단백질 구조 수 초

### AlphaFold 3: 생명과학의 게임 체인저

2024년 5월 출시된 AlphaFold 3는 단백질·핵산·소분자·이온의 3D 구조와 상호작용을 예측합니다.

- 사람이 수 년 걸리던 단백질 구조 파악 → **수 초**
- 약물 결합 부위와 상호작용 에너지를 정확히 예측
- 2026년에는 **"AlphaFold 4"**급 전용 AI가 Nature지에 소개됨

### AI 신약 개발

- **Insilico Medicine**: AI 설계 섬유증 치료제가 **18개월** 만에 임상 시험 진입 (기존: 4년)
- Generate:Biomedicines의 **<span>$</span>1B** Novartis 파트너십
- Isomorphic Labs의 **<span>$</span>600M+** AlphaFold 약물 설계 확장
- FDA: AI 의료기기 **1,016건** 승인 (2024.10 기준), 2025년 12월 FDA 내부에 에이전틱 AI 플랫폼 가동

## 5. AI 에이전트: 도구에서 동료로

### MCP: AI의 USB-C

Anthropic이 2024년 말 공개한 **Model Context Protocol(MCP)**은 AI 에이전트가 데이터베이스, 검색 엔진, API 등 외부 도구와 연결되는 **표준 프로토콜**입니다. "AI의 USB-C"로 불리며, 에이전트가 실제 시스템에 접속하는 마찰을 획기적으로 줄였습니다.

### 에이전트의 실전 배치

- **Gartner 전망**: 엔터프라이즈 앱의 AI 에이전트 내장률 — 2025년 초 5% 미만 → 2026년 중반 **40%**
- **고객 서비스**: 하위 분위 상담원이 자율 에이전트로 대체, 서비스 품질과 전환율 동시 향상
- **에이전트 스웜**: 단일 에이전트 → 여러 전문 에이전트가 협업 (리서치·작성·검증 분업)
- **2028년 전망**: 조직의 38%가 인간 팀 내에 AI 에이전트를 팀원으로 운영 (Gartner)

2025년, AI의 정의 자체가 바뀌었습니다. "인지·추론·행동하는 시스템"에서 **"소프트웨어 도구를 사용하고 자율적으로 행동하는 LLM"**으로.

## 6. 오픈소스의 추격

폐쇄형 모델과 오픈소스의 격차도 급속히 좁혀졌습니다.

- Chatbot Arena 성능 격차: **8.04%** (2024.01) → **1.70%** (2025.02)
- Meta Llama, DeepSeek 등이 GPT-4급 성능을 무료로 제공
- 누구나 로컬에서 강력한 AI를 돌릴 수 있는 시대

## 숫자가 말하는 것

이 사례들을 관통하는 패턴은 명확합니다.

1. **비선형 도약**: 점진적 개선이 아니라, 1~2년 사이에 수십 배 성능 향상
2. **범용화**: 전문 시스템이 하던 일을 범용 모델이 대체 (IMO 금메달을 범용 LLM이 달성)
3. **접근성 폭발**: <span>$</span>500/월 → <span>$</span>20/월, 비개발자도 앱 개발 가능
4. **자율성 확대**: 보조 도구 → 자율 에이전트 → 에이전트 간 협업

Geoffrey Hinton은 2026년을 **"대규모 AI 일자리 대체의 원년"**으로 예측했고, 전직 OpenAI 연구원은 초지능(superintelligence)의 도달 시점을 **2027년(최빈값), 2028년(중앙값)**으로 전망합니다.

임계점을 넘었는지 아닌지는 더 이상 논쟁의 대상이 아닙니다. 문제는 **그 다음에 무엇을 할 것인가**입니다.
