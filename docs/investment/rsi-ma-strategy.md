# RSI + MA 배열 활용 장세 분류 & 매매 케이스
{:.no_toc}

MA로 레짐(장세 방향)을 판별하고, RSI로 타이밍을 잡는 결합 전략을 7가지 케이스로 정리합니다.

## 공통 관측 프레임워크

매매 판단 전, 아래 두 축을 항상 동시에 확인합니다.

### MA 배열 체크포인트

| 항목 | 확인 내용 |
|------|-----------|
| **정렬** | 단기 > 중기 > 장기 (상승 추세) / 반대 (하락 추세) |
| **기울기** | 장기 MA가 우상향인지 → "진짜 추세"의 핵심 조건 |
| **이격** | 가격이 단기 MA에서 얼마나 떨어졌는지 → 과열/되돌림 위험 판단 |

### RSI 체크포인트

| 항목 | 확인 내용 |
|------|-----------|
| **50선 위치** | 상승 압력 우위 (50 위) vs 하락 압력 우위 (50 아래) |
| **추세장 밴드** | 상승장에선 RSI 바닥이 주로 40 근처, 하락장에선 천장이 60 근처 |
| **다이버전스/실패스윙** | 모멘텀 구조 변화의 조기 신호 |

> **핵심 원칙**: MA는 "어느 방향의 장인지(레짐)"를 알려주고, RSI는 "그 장 안에서 지금 어느 위치인지(타이밍)"를 알려줍니다. 둘을 분리해서 쓰면 절반짜리 정보가 됩니다.

## 케이스 1. 정상 상승 추세

> **"추세 추종의 교과서" — 가장 높은 확률의 롱 구간**

| 구분 | 조건 |
|------|------|
| MA | 단기 > 중기 > 장기 정렬 + 장기 MA 우상향 |
| 가격 | 단기/중기 MA 위에서 눌림 후 재상승 |
| RSI | 50선 위 유지, 조정 때 40~50에서 지지 후 반등 |

### 진입 시점

- **1차 진입**: 눌림이 중기 MA(20 or 60) 부근에서 멈추고 캔들이 양전환 확인
- **2차 추가**: 고점 돌파 + RSI가 50~60 이상 재확인 ("엔진 재점화" 신호)

### TP/SL

| 구분 | 조건 |
|------|------|
| **SL** | 눌림 저점 이탈 *또는* 중기 MA 종가 이탈 + RSI 50 하향 **동시** 충족 |
| **TP 1** | 직전 스윙 고점 부근 분할 청산 (30~50%) |
| **TP 2** | RSI 70~80에서 꺾이는데 가격이 신고점 못 내면 (약한 다이버전스) 추가 정리 |
| **TP 3** | 추세 유지형이면 중기 MA 기반 **추적 손절**이 수익 극대화에 유리 |

## 케이스 2. 상승 추세의 과열/피로

> **"방향은 살아있지만, 기대수익/리스크 비율이 나빠진 상태"**

| 구분 | 조건 |
|------|------|
| MA | 정렬은 상승이지만, 가격이 단기 MA에서 과도하게 이격 |
| RSI | 75~85 고레벨 반복 + 베어리시 다이버전스 출현 (가격 HH, RSI LH) |

**중요 해석**: 추세가 꺾인 "확정"이 아니라, 지속 확률은 있지만 **신규 진입의 매력이 크게 줄어든 상태**입니다.

### 실전 운영

| 포지션 | 대응 |
|--------|------|
| **신규 진입** | 보수적 대기 (눌림 기다리기) — 추격 매수 금지 구간 |
| **기존 보유자** | 이격/RSI 과열 구간에서 분할 익절 시작 |
| **SL 조정** | 단기 MA 이탈이 아닌 **중기 MA 이탈**을 트리거로 (과열 구간은 단기 흔들림이 극심) |

## 케이스 3. 상승 추세 → 분배/전환 초기 신호

> **"경고에서 확인으로 — 단계적으로 읽어야 실수를 줄인다"**

전환은 한 번에 오지 않습니다. **4단계로 순서대로 확인**해야 합니다.

| 단계 | 신호 |
|------|------|
| 1단계 | RSI 베어리시 다이버전스 / 실패스윙 조짐 |
| 2단계 | 가격 고점 돌파 실패 (더블탑 유사 구조) |
| 3단계 | 단기 MA 기울기 둔화 → 단기/중기 데드크로스 가능성 |
| 4단계 | **RSI 50선 하향 이탈** ← "전환 확인"의 가장 중요한 신호 |

### 진입 시점

- **숏/헤지**: RSI 실패스윙 완성 + 50선 하향 + 단기 < 중기 첫 확인 시
- **기존 롱 보유자**: "새 롱 중단 + 보유분 리스크 축소"가 먼저. 바로 숏 전환은 성급함

### TP/SL

전환 매매에서는 **SL이 생명**입니다.

| 구분 | 조건 |
|------|------|
| **SL** | 마지막 구조적 고점 상향 돌파 시 즉시 손절 |
| **TP** | 장기 MA(120/200) 또는 이전 박스 하단까지 단계적 목표 |

## 케이스 4. 정상 하락 추세

> **"추세 추종 vs 역추세 반등매매를 반드시 구분해야 한다"**

| 구분 | 조건 |
|------|------|
| MA | 단기 < 중기 < 장기 정렬 + 장기 MA 하향 |
| RSI | 50선 아래 유지, 반등 때 60 근처에서 막히고 재하락 |

### 전략 분기

| 전략 | 진입 조건 |
|------|-----------|
| **추세 추종 (숏/현금 비중 확대)** | 반등이 중기 MA에서 막힐 때 진입 |
| **역추세 바닥잡기** | RSI 30 이하만으로는 **절대 부족**. 케이스 5-B 조건 모두 충족 시에만 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **숏 TP** | 이전 저점 / RSI 20~30 과매도 구간에서 분할 청산 |
| **숏 SL** | "중기 MA 회복 + RSI 50 회복" 동시 조건 — **레짐 변화 기반 청산** |

---

## 케이스 5. 하락 추세 속 반등 — 함정 vs 바닥 전환

> **"가장 헷갈리는 구간 — RSI+MA 조합이 가장 빛나는 케이스"**

### 5-A. 데드캣 바운스 (반등 함정)

| 항목 | 상태 |
|------|------|
| MA | 장기 하향, 배열 여전히 하락 정렬 |
| RSI | 반등해도 50을 못 넘고 55~60에서 꺾임 |
| 가격 | 중기 MA(60) 아래에서 반등 종료 |

→ **"추세 내 조정" 가능성 높음**. 롱 진입 자제.

### 5-B. 진짜 바닥 전환 후보

| 항목 | 상태 |
|------|------|
| RSI | 불리시 다이버전스 (가격 LL, RSI HL) + 이후 **50선 상향 회복** |
| MA | 단기 MA가 평평→상향 전환, 가격이 중기 MA를 회복하고 지지 |
| 배열 | 완전 전환 전이라도 **"단기 > 중기"가 먼저** 발생 |

→ 이 조합이 나오면 **"진짜 전환 후보"로 격상**해서 관찰합니다.

### TP/SL (전환 롱은 SL을 타이트하게)

| 구분 | 조건 |
|------|------|
| **SL** | 다이버전스를 만든 마지막 저점 이탈 시 즉시 손절 |
| **TP 1차** | 중기/장기 MA 구간 (첫 저항) |
| **TP 이후** | MA 배열이 상승 전환되면 케이스 1 (추세 추종)으로 운영 방식 변경 |

## 케이스 6. 레인지/박스 장

> **"RSI 단독으로 가장 잘 먹히는 구간 — MA 크로스는 잡음으로 취급"**

| 구분 | 조건 |
|------|------|
| MA | 장기 MA 평평, 단기/중기 MA가 서로 엉킴 (배열이 자주 뒤집힘) |
| RSI | 40~60 또는 30~70 사이 왕복 반복 |

### 진입 시점

| 방향 | 조건 |
|------|------|
| **박스 하단 매수** | RSI 30~40 + 가격이 단기 MA 회복 확인 |
| **박스 상단 매도** | RSI 60~70 + 가격이 단기 MA 이탈 확인 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **TP** | 박스 반대편 + RSI가 50 도달 시 일부 청산 (욕심 금지) |
| **SL** | 박스 종가 이탈 + RSI가 50을 **한 방향으로 지속 유지** (레짐 변화 신호) |

> 레인지 장에서 MA 크로스 신호는 잡음이 많습니다. **RSI 평균 회귀 전략이 우선**입니다.

## 케이스 7. 돌파 장 (레인지 → 추세 레짐 전환)

> **"레인지 탈출 — 리테스트 확인이 핵심"**

### 돌파 판별 조건

| 단계 | 상태 |
|------|------|
| **돌파 전** | MA 평평/엉킴 + RSI 중립권 (45~55) |
| **돌파 발생** | 가격이 중기/장기 MA 위에 안착 + RSI가 50 돌파 후 60~70 테스트 |
| **결정적 확인** | 눌림(리테스트)에서 **RSI가 40~50에서 지지** ← 핵심 |

### 진입 시점

| 스타일 | 조건 |
|--------|------|
| **보수적** | 돌파 후 첫 눌림 (리테스트)에서 중기 MA 지지 확인 후 진입 |
| **공격적** | 돌파 캔들 + 거래량 조건 충족 시 즉시 진입 |

### TP/SL

| 구분 | 조건 |
|------|------|
| **SL** | 리테스트 실패 (돌파 레벨 하향 이탈) + RSI 50 하향 동시 |
| **TP 1차** | 직전 레인지 높이만큼 측정 목표 |
| **TP 이후** | 돌파 확인되면 케이스 1 (정상 상승 추세) 운영으로 전환 |

## TP/SL 규칙화 요약

| 방식 | 적합 케이스 | 설명 |
|------|------------|------|
| **구조 기반** (스윙 저점/고점 + RSI 50 확인) | 케이스 1, 3, 5 | 추세 매매에서 가장 일관적 |
| **MA 기반** (중기 MA 종가 이탈 + RSI 밴드) | 케이스 1, 4, 7 | 휩소 감소, 추세 유지 조건화 |
| **과열 기반** (RSI 70/30 + 이격) | 케이스 2, 6 | **TP에만 유리**, SL 단독 사용 비추 |

## 레짐 분류 빠른 판단 트리

```
장기 MA 방향은?
├─ 우상향 → 상승 레짐
│       RSI 50 위? → 케이스 1 (추세 추종)
│       RSI 과열 + 다이버전스? → 케이스 2/3 (과열/전환 주의)
│
├─ 하향 → 하락 레짐
│       반등 RSI 50 못 넘음? → 케이스 5-A (함정)
│       불리시 다이버전스 + RSI 50 회복? → 케이스 5-B (전환 후보)
│
└─ 횡보/평평 → 레인지 레짐
        RSI 왕복? → 케이스 6 (레인지 매매)
        RSI 50 돌파 + MA 정렬 시작? → 케이스 7 (돌파 대기)
```

## 데이트레이딩 타임프레임 구성 상세

위 7가지 케이스를 데이트레이딩에 적용할 때의 타임프레임 3계층 구성을 다룹니다.

### 기본 구조

```
레짐 레이어   (일봉 MA)     → 오늘 어느 방향으로 거래할 수 있는가
      ↓ 방향 허가
타이밍 레이어 (1시간봉 RSI) → 지금 진입할 위치인가
      ↓ 위치 확인
실행 레이어   (15분봉 캔들) → 정확히 어느 캔들에서 들어가는가
```

> 데이트레이딩에서 이 구조가 중요한 이유: 하루 안에 진입/청산이 끝나기 때문에 레짐 판단 없이 단기 RSI만 보면 추세 역행 거래를 반복하게 됩니다. 일봉 MA가 "오늘 하루의 방향 허가증" 역할을 해야 합니다.

### 레이어 1. 레짐 판단 — 일봉 MA

**MA 설정**: MA(20), MA(60) 기본 / MA(120) 추가 확인용

#### 장세 분류 기준

**상승 레짐** (롱만 허가)

| 조건 | 설명 |
|------|------|
| MA(20) > MA(60) 정렬 | 단기 추세 우위 |
| MA(60) 기울기 우상향 | 중기 추세 방향 확인 |
| 가격이 MA(20) 위 또는 부근 지지 | 눌림이 구조 내에 있음 |

- 오늘 허가: **롱 진입만 탐색**
- 오늘 금지: 숏 진입 (RSI 과매수여도 숏 신규 진입 금지)

**하락 레짐** (숏만 허가, 현물이면 현금 대기)

| 조건 | 설명 |
|------|------|
| MA(20) < MA(60) 정렬 | 단기 추세 하락 우위 |
| MA(60) 기울기 우하향 | 중기 추세 하락 확인 |
| 가격이 MA(20) 아래 또는 부근 저항 | 반등이 구조적 저항에 걸림 |

- 오늘 허가: **숏 진입만 탐색** (현물이면 신규 매수 금지)
- 오늘 금지: 롱 진입

**레인지 레짐** (박스 경계 매매만 허가)

| 조건 | 설명 |
|------|------|
| MA(20)과 MA(60) 수렴 또는 평평 | 방향성 없음 |
| 가격이 MA(20)/MA(60) 위아래 반복 | 박스 구간 |

- 오늘 허가: 박스 하단 롱 / 박스 상단 숏 (경계 매매)
- 오늘 금지: 중간 지점 신규 진입

**애매한 레짐** (당일 거래 자제)

| 조건 | 설명 |
|------|------|
| MA(20)과 MA(60) 크로스 직전/직후 | 방향 미확정 |
| MA(60) 평평 + MA(20) 위아래 교차 중 | 레짐 불명확 |

→ 거래량을 줄이거나 **관망**

> 매일 시장 열리기 전 루틴: 일봉 MA 배열을 먼저 확인하고 "오늘 롱만/숏만/박스/관망" 중 하나를 결정한 뒤 하위 TF를 열 것. 하위 TF부터 열면 레짐 판단이 흐려집니다.

### 레이어 2. 타이밍 판단 — 1시간봉 RSI

**RSI 설정**: RSI(14) 기본 / RSI(9) 민감 신호 병행 가능

#### 상승 레짐에서 1시간봉 RSI 활용

| 항목 | RSI 구간 | 해석 |
|------|----------|------|
| **진입 탐색** | 40~50 | 추세 내 눌림이 "지지권"에 도달한 상태 |
| **진입 확인** | 45 이상으로 방향 전환 + 50선 회복 시도 | "모멘텀 재점화" 신호 |
| **익절 목표** | 65~70 | 과열 전 1차 익절 구간. 70 이상은 당일 기준으로 욕심 |
| **손절** | 40 이탈 (종가 기준) | 추세 내 눌림이 아닌 추세 약화. 일봉 MA(20) 이탈 동시 시 즉시 손절 |
| **추가 진입** | 50~60 유지 + 가격 신고점 돌파 | 모멘텀이 살아있는 상태에서의 추가 |

#### 하락 레짐에서 1시간봉 RSI 활용

| 항목 | RSI 구간 | 해석 |
|------|----------|------|
| **진입 탐색** | 50~60 | 하락 추세 내 반등이 "저항권"에 도달한 상태 |
| **진입 확인** | 55 이하로 방향 전환 + 50선 하향 이탈 | 반등 실패 확인, 하락 재개 신호 |
| **익절 목표** | 30~35 | 과매도 전 1차 익절 구간 |
| **손절** | 60 돌파 + 일봉 MA(20) 상향 회복 | 하락 레짐 약화 신호, 즉시 손절 |

#### 레인지 레짐에서 1시간봉 RSI 활용

| 항목 | 조건 |
|------|------|
| **박스 하단 매수** | RSI 30~40 진입 + RSI 반등 전환 확인 |
| **박스 상단 매도** | RSI 60~70 진입 + RSI 하락 전환 확인 |
| **TP** | RSI 50 도달 시 절반 청산, 반대편 박스 도달 시 나머지 |
| **SL** | 박스 이탈 종가 확정 시 즉시 손절 |

> 레인지 중간(RSI 45~55)에서 신규 진입 금지. 크로스나 50선 돌파만으로 진입하면 잡음에 당하기 쉽습니다.

#### 1시간봉 RSI 다이버전스 활용

| 유형 | 조건 | 활용 |
|------|------|------|
| **베어리시 다이버전스** | 가격 HH + RSI LH | 상승 레짐에서 **당일 TP 신호**. 신규 롱 진입 금지 + 기존 보유분 익절 트리거 |
| **불리시 다이버전스** | 가격 LL + RSI HL | 하락 레짐에서 **당일 숏 커버 신호**. 신규 숏 진입 금지 + 기존 숏 익절 트리거 |

> 다이버전스는 단독 진입 신호로 쓰지 않습니다. 레짐 전환의 "경고"이지 "확인"이 아닙니다.

### 레이어 3. 실행 판단 — 15분봉 캔들

15분봉은 딱 한 가지만 담당합니다: **"1시간봉 RSI가 진입 조건을 충족했을 때, 정확히 어느 캔들에서 들어가는가"**

15분봉에서 보지 않아야 할 것:
- 레짐 판단 (일봉 MA가 담당)
- RSI 신호 (1시간봉이 담당)
- 큰 추세 방향 (상위 TF가 담당)

#### 롱 진입 실행 캔들 (상승 레짐 + 1시간봉 RSI 40~50 지지)

| 패턴 | 진입 시점 |
|------|-----------|
| **핀바 양전환** — 아래 꼬리 긴 캔들 + 다음 캔들 양봉 확인 | 핀바 고점 돌파 시 |
| **장대양봉** — RSI 지지 구간에서 강한 양봉 | 장대양봉 종가 or 다음 캔들 시가 |
| **연속 양봉 (3봉 반등)** — 3캔들 연속 양봉으로 방향 전환 확인 | 3번째 양봉 종가 (보수적) |

#### 숏 진입 실행 캔들 (하락 레짐 + 1시간봉 RSI 50~60 저항)

| 패턴 | 진입 시점 |
|------|-----------|
| **위 꼬리 핀바** — 위 꼬리 긴 음봉 + 다음 캔들 음봉 확인 | 핀바 저점 이탈 시 |
| **장대음봉** — RSI 저항 구간에서 강한 음봉 | 장대음봉 종가 or 다음 캔들 시가 |
| **연속 음봉 (3봉 하락)** — 3캔들 연속 음봉으로 방향 전환 확인 | 3번째 음봉 종가 (보수적) |

### SL/TP 설정 — 레이어 조합 기준

#### SL 설정 원칙

| 유형 | 롱 (상승 레짐) | 숏 (하락 레짐) |
|------|---------------|---------------|
| **구조적 SL** (주된 기준) | 15분봉 스윙저점 이탈 or 1시간봉 MA(20) 종가 이탈 | 15분봉 스윙고점 이탈 or 1시간봉 MA(20) 종가 이탈 |
| **RSI 기반 SL** | 1시간봉 RSI 40 이탈 (지지 실패) | 1시간봉 RSI 60 돌파 (저항 실패) |
| **레짐 기반 SL** (최우선) | 일봉 MA(20) 종가 이탈 → 전량 청산 | 일봉 MA(20) 종가 회복 → 전량 청산 |

우선순위: 구조적 SL 또는 RSI SL 중 **먼저 도달하는 쪽** 적용. 일봉 MA 이탈은 어느 것보다 우선하여 즉시 손절.

#### TP 설정 원칙

| 단계 | 조건 (상승 레짐 롱 기준) |
|------|--------------------------|
| **1차 TP** (50% 물량) | 1시간봉 RSI 65~70 도달 → 절반 청산 후 SL을 진입가로 이동 (본전 SL) |
| **2차 TP** (나머지) | 다음 중 먼저 도달: (1) 당일 예상 고점 (직전 스윙고점 or 일봉 MA(60) 이격) / (2) 1시간봉 RSI 베어리시 다이버전스 출현 / (3) 장 마감 30분 전 |

> **당일 청산 원칙**: 데이트레이딩은 당일 청산이 기본. 단, 일봉 MA 레짐이 강하고 포지션이 크게 수익 중이면 다음날 SL을 타이트하게 설정하고 일부 보유 허용 (예외적 스윙 전환).

### 당일 매매 흐름 예시

> **사례: 상승 레짐 + 눌림 진입**

| 시간 | 상황 | 판단 |
|------|------|------|
| **장 시작 전** | 일봉 MA(20) > MA(60), MA(60) 우상향 확인 | 오늘 롱만 탐색, 숏 금지 결정 |
| **9시~10시** | 1시간봉 RSI가 55에서 눌리며 46으로 하락 중 | 아직 진입 구간 미도달, 대기 |
| **10시~11시** | 1시간봉 RSI가 43까지 내려왔다가 상승 전환 시작 | 진입 탐색 구간(40~50) 진입, 15분봉 관찰 시작 |
| **11시 15분** | 15분봉에서 핀바 + 다음 캔들 양봉 확인 | 핀바 고점 돌파 시 롱 진입. SL: 핀바 저점 이탈 |
| **오후 1시** | 1시간봉 RSI 67 도달 | 1차 TP 실행 (50% 청산). SL을 진입가로 상향 |
| **오후 2시 30분** | 1시간봉 RSI 72에서 베어리시 다이버전스 조짐 | 나머지 50% 전량 청산 |
| **결과** | 1차 TP + 다이버전스 2차 TP로 당일 청산 완료 | |

### 핵심 규칙 체크리스트

**레짐 확인:**
- 일봉 MA(20)/MA(60) 배열 방향 확인했는가
- 오늘 허가된 방향(롱/숏/박스)을 결정했는가
- 레짐이 애매하면 관망을 선택했는가

**타이밍 확인:**
- 1시간봉 RSI가 레짐에 맞는 진입 구간에 있는가
- RSI 방향 전환이 확인됐는가 (하락에서 상승 or 반대)
- 다이버전스 여부를 체크했는가

**실행 확인:**
- 15분봉에서 진입 캔들 패턴이 출현했는가
- SL 위치를 진입 전에 정했는가
- 1차/2차 TP 목표를 사전에 설정했는가

**금지 사항:**
- 일봉 MA 레짐과 반대 방향으로 진입하려 하지 않는가
- 1시간봉 RSI 조건 미충족 상태에서 15분봉만 보고 진입하지 않는가
- 당일 청산 원칙을 지키고 있는가
