# 손익계산서 핵심 이익 지표 총정리
{:.no_toc}

매출에서 비용을 어디까지 차감하느냐에 따라 이익의 이름이 달라집니다. 각 지표가 무엇을 측정하고, 투자 판단에 어떻게 쓰이는지 정리합니다.

## 손익계산서 흐름 한눈에 보기

손익계산서(Income Statement)는 위에서 아래로 비용을 단계적으로 차감하는 구조입니다.

```
매출액 (Revenue)
 ─ 매출원가 (COGS)
━━━━━━━━━━━━━━━━━━━━
= 매출총이익 (Gross Profit)
 ─ 판매관리비 (SG&A)
━━━━━━━━━━━━━━━━━━━━
= 영업이익 (Operating Profit)
 + 감가상각비 (D&A) 더하면
━━━━━━━━━━━━━━━━━━━━
= EBITDA
 ─ 영업외손익, 이자, 세금
━━━━━━━━━━━━━━━━━━━━
= 당기순이익 (Net Income)
```

아래로 갈수록 더 많은 비용이 반영되므로, **위쪽 지표는 본업의 효율성**, **아래쪽 지표는 최종 수익성**을 보여줍니다.

## 매출총이익 (Gross Profit)

| 항목 | 내용 |
|------|------|
| **공식** | 매출액 − 매출원가(COGS) |
| **영문** | Gross Profit |
| **마진율** | 매출총이익률 = 매출총이익 ÷ 매출액 × 100 |

**매출원가(COGS)**는 제품을 만들거나 서비스를 제공하는 데 직접 들어간 비용입니다.
- 제조업: 원재료비, 직접 인건비, 공장 감가상각비
- 소매업: 상품 매입비
- SaaS: 서버 비용, 호스팅 비용

### 의미

- **제품/서비스 자체의 수익성**을 측정합니다.
- 매출총이익률이 높을수록 원가 대비 마진이 크다는 뜻입니다.
- 같은 업종 내 기업 간 비교에 유용합니다.

### 업종별 참고 범위

| 업종 | 매출총이익률 |
|------|-------------|
| 소프트웨어/SaaS | 70~90% |
| 제약/바이오 | 60~80% |
| 소비재/유통 | 20~40% |
| 제조업 | 15~30% |

## 공헌이익 (Contribution Margin)

| 항목 | 내용 |
|------|------|
| **공식** | 매출액 − 변동비(Variable Costs) |
| **영문** | Contribution Margin |
| **마진율** | 공헌이익률 = 공헌이익 ÷ 매출액 × 100 |

### 매출총이익과의 차이

공헌이익과 매출총이익은 비슷해 보이지만 **비용 분류 기준이 다릅니다**.

| 구분 | 매출총이익 | 공헌이익 |
|------|-----------|---------|
| 차감 비용 | 매출원가(COGS) | 모든 변동비 |
| 비용 분류 | 기능별 (제조 vs 판관비) | 성격별 (변동 vs 고정) |
| 포함 범위 | 제조 관련 비용만 | 변동 판매비·배송비 등 포함 |

- **매출총이익**: "만드는 데 얼마 들었나"
- **공헌이익**: "한 단위 더 팔면 고정비 충당에 얼마 기여하나"

### 의미

- 고정비를 충당하고 이익을 내는 데 **각 제품이 얼마나 기여하는지** 측정합니다.
- **손익분기점(BEP) 분석**의 핵심 변수입니다.
  - 손익분기 매출 = 고정비 ÷ 공헌이익률
- 스타트업에서 "공헌이익 흑자 달성"은 변동비는 커버한다는 의미입니다.

### 예시

온라인 쇼핑몰에서 티셔츠 1장을 30,000원에 판매:

| 비용 항목 | 금액 | 비용 성격 |
|----------|------|----------|
| 상품 매입비 | 12,000원 | 변동비 |
| 배송비 | 3,000원 | 변동비 |
| 결제 수수료 | 900원 | 변동비 |
| **변동비 합계** | **15,900원** | |
| **공헌이익** | **14,100원** | |
| 공헌이익률 | 47% | |

→ 1장 팔 때마다 14,100원이 임대료·인건비 등 고정비를 충당하는 데 기여합니다.

## 영업이익 (Operating Profit / EBIT)

| 항목 | 내용 |
|------|------|
| **공식** | 매출총이익 − 판매관리비(SG&A) |
| **영문** | Operating Profit, Operating Income |
| **마진율** | 영업이익률 = 영업이익 ÷ 매출액 × 100 |

**판매관리비(SG&A)**에는 다음이 포함됩니다:
- 임직원 급여, 복리후생비
- 마케팅·광고비
- 사무실 임차료
- 연구개발비(R&D)
- 감가상각비·무형자산상각비

### EBIT과의 차이

| 구분 | 영업이익 (K-IFRS) | EBIT |
|------|-------------------|------|
| 정의 | 매출총이익 − 판관비 | 세전이익 + 이자비용 |
| 영업외손익 | 미포함 | 포함 |
| 활용 | 한국 기업 실적 발표 | 글로벌 비교 |

한국 회계기준(K-IFRS)의 영업이익은 **본업의 성과만** 반영합니다. EBIT은 영업외수익·비용(자산 매각, 외환 손익 등)까지 포함하므로, 엄밀히 다릅니다. 다만 실무에서는 혼용되는 경우가 많습니다.

### 의미

- **기업 본업의 수익 창출 능력**을 가장 직접적으로 보여줍니다.
- 재무 구조(부채 수준)와 세금 체계에 영향받지 않아 **기업 간 영업 효율 비교**에 적합합니다.
- 영업이익률이 꾸준히 개선되면 경영 효율이 높아지고 있다는 신호입니다.

## EBITDA

| 항목 | 내용 |
|------|------|
| **공식** | 영업이익 + 감가상각비(D) + 무형자산상각비(A) |
| **영문** | Earnings Before Interest, Taxes, Depreciation and Amortization |
| **마진율** | EBITDA 마진 = EBITDA ÷ 매출액 × 100 |

### 왜 감가상각비를 더하는가

감가상각비와 무형자산상각비는 **실제 현금이 나가지 않는 비용(비현금 비용)**입니다.

- 공장 설비를 100억에 샀다면, 매년 10억씩 10년간 비용 처리 (감가상각)
- 실제 현금 100억은 구매 시점에 이미 지출
- 이후 매년 잡히는 10억은 회계적 비용일 뿐, 현금 유출이 아님

따라서 감가상각비를 다시 더해주면 **영업활동에서 실제 벌어들이는 현금에 가까운 수치**가 됩니다.

### 활용

| 활용 분야 | 설명 |
|----------|------|
| **기업가치 평가** | EV/EBITDA 배수로 기업 가치 비교 |
| **M&A** | 인수 대상 기업의 현금 창출력 평가 |
| **업종 간 비교** | 감가상각 정책·세율 차이를 제거하고 비교 |
| **부채 상환 능력** | 순차입금/EBITDA로 재무 건전성 판단 |

### EV/EBITDA

주가 수준을 판단하는 대표적 밸류에이션 지표입니다.

```
EV/EBITDA = 기업가치(EV) ÷ EBITDA
```

- **EV**(Enterprise Value) = 시가총액 + 순차입금
- EV/EBITDA가 **낮을수록** 벌어들이는 현금 대비 기업이 저평가
- 일반적으로 EV/EBITDA **10배 이하**면 저평가로 판단 (업종별 차이 있음)

### 한계

- 설비투자(CAPEX)를 무시하므로, 자본 집약적 산업에서는 과대 평가될 수 있음
- 운전자본 변동을 반영하지 않음
- 워런 버핏: "EBITDA를 쓰는 경영진은 이의 요정도 믿는 것" — 실제 현금흐름(FCF)과 괴리가 있을 수 있음

## 당기순이익 (Net Income)

| 항목 | 내용 |
|------|------|
| **공식** | 영업이익 + 영업외수익 − 영업외비용 − 법인세 |
| **영문** | Net Income, Net Profit, Bottom Line |
| **마진율** | 순이익률 = 당기순이익 ÷ 매출액 × 100 |

### 영업이익에서 추가로 차감되는 항목

| 항목 | 예시 |
|------|------|
| 이자비용 | 차입금 이자 |
| 영업외비용 | 외환 손실, 자산 처분 손실 |
| 법인세 | 과세 소득에 대한 세금 |

영업외수익(이자수익, 자산 매각 이익 등)은 더해집니다.

### 의미

- 손익계산서의 **최종 결론**입니다.
- 주주에게 귀속되는 이익이며, **EPS(주당순이익)**의 기초가 됩니다.
  - EPS = 당기순이익 ÷ 발행주식수
- PER(주가수익비율) = 주가 ÷ EPS

### 주의점

- 일회성 항목(자산 매각, 소송 비용 등)에 의해 왜곡될 수 있음
- 순이익이 높아도 영업이익이 낮다면, 본업 외 요인이 크다는 의미
- 반대로 영업이익은 좋은데 순이익이 낮다면, 부채 이자나 세금 부담이 크다는 신호

## 전체 비교 요약

| 지표 | 공식 | 핵심 질문 | 주요 활용 |
|------|------|----------|----------|
| **매출총이익** | 매출 − 매출원가 | 제품 자체가 돈이 되는가? | 원가 관리, 업종 내 비교 |
| **공헌이익** | 매출 − 변동비 | 한 단위 더 팔면 얼마 남나? | 손익분기 분석, 가격 결정 |
| **영업이익** | 매출총이익 − 판관비 | 본업이 돈을 버는가? | 경영 효율 평가 |
| **EBITDA** | 영업이익 + 감가상각비 | 현금을 얼마나 만드는가? | 기업가치 평가, M&A |
| **당기순이익** | 모든 비용 차감 후 | 주주에게 얼마 남나? | EPS, PER, 배당 |

## 투자 분석에서의 활용 팁

### 1. 위에서 아래로 읽기

손익계산서를 위에서 아래로 따라가면서, **어느 단계에서 마진이 줄어드는지** 확인합니다.

- 매출총이익률은 높은데 영업이익률이 낮다 → **판관비(인건비, 마케팅) 과다**
- 영업이익은 좋은데 순이익이 낮다 → **부채 이자 부담 또는 일회성 손실**
- EBITDA는 높은데 영업이익이 낮다 → **감가상각비 비중이 큼** (자본 집약적)

### 2. 추세 확인

단일 분기보다 **3~5년 추세**가 중요합니다.

- 영업이익률이 매년 개선 → 경영 효율 향상
- 매출은 늘지만 공헌이익률이 하락 → 가격 경쟁 심화, 원가 상승

### 3. 스타트업 이익 지표 단계

적자 기업의 실적 개선은 보통 다음 순서를 따릅니다:

```
공헌이익 흑자 → EBITDA 흑자 → 영업이익 흑자 → 당기순이익 흑자
```

- **공헌이익 흑자**: 변동비는 커버 (팔수록 적자 줄어듦)
- **EBITDA 흑자**: 고정비까지 커버, 현금 창출 시작
- **영업이익 흑자**: 감가상각 포함 모든 영업 비용 충당
- **순이익 흑자**: 이자·세금까지 커버, 완전한 수익 기업

### 4. 지표별 대표 밸류에이션

| 이익 지표 | 밸류에이션 배수 | 용도 |
|----------|---------------|------|
| 당기순이익 | PER (P/E) | 가장 보편적 |
| EBITDA | EV/EBITDA | 업종 간 비교, M&A |
| 매출총이익 | EV/Gross Profit | 적자 SaaS 기업 |
| 매출액 | PSR (P/S) | 초기 성장 기업 |
