# 임신부 식중독 — 빠른 합의 실전 가이드
{:.no_toc}

쿠팡에서 구매한 식품으로 식중독이 발생했을 때, 변호사 없이 직접 합의로 빠르게 마무리하되, 태아 후유증 추가 청구권만은 확실히 유보하는 전략을 정리합니다.

## 상황

- 임신 중인 배우자가 쿠팡에서 주문한 유명 브랜드 샌드위치 섭취
- 섭취 약 30분 내 극심한 구역감, 두통, 복통, 구토
- 설사는 즉시 발현되지 않았으나, **추후 설사 발생 시 입원 치료 필요**하다는 진단
- 수액 처방. **태아에게 치명적일 수 있다는 진단**
- 실물(샌드위치, 포장재) 폐기 상태. 주문내역·진료기록 보유

## STEP 1. 증거 보강 (추후 진행)

> 초진 기록은 이미 병원에 있으므로 급하지 않음. 협상 시작 전에 아래 항목 확보.

### 확보할 것

| 항목 | 조치 | 상태 |
|------|------|------|
| 쿠팡 주문내역 | 스크린샷 저장 | |
| 병원 초진 기록 | PDF 확보 (병원에 이미 있음) | |
| 진단서/소견서 | **아래 문구 포함 요청** | |
| 증상 일지 | 시간대별 증상 메모 | |

### 진단서 보강 — 가장 중요

담당 의사에게 아래 내용이 명시된 **소견서** 요청:

- "○○ 샌드위치 섭취 후 식중독 의심 증상 발생"
- "임신 중 식중독으로 태아에 영향을 미칠 수 있음"
- "설사 발생 시 입원 치료 필요"
- "향후 추적관찰 필요"

이 문구의 유무가 합의금에 직접 영향을 줍니다.

## STEP 2. 쿠팡 클레임 접수 (오늘)

### 접수 채널 — 현실적인 선택지

쿠팡은 **문자(SMS)나 이메일을 통한 상담을 지원하지 않습니다.** 사용 가능한 채널은 두 가지입니다:

| 채널 | 방법 | 기록 |
|------|------|------|
| **앱 1:1 채팅** (1순위) | 쿠팡 앱 → 마이쿠팡 → 고객센터 → 1:1 채팅 | 텍스트 기록 자동 보존 |
| **전화** (2순위) | 1577-7011 (24시간) | **반드시 통화 녹음** 필수 |

**1순위: 앱 1:1 채팅**을 먼저 시도하세요. 텍스트 기록이 자동으로 남아 증거 확보에 유리합니다. 채팅 내용은 반드시 **스크린샷으로도 별도 저장**하세요.

**전화할 경우**: 통화 녹음을 반드시 켜고, 상담 시작 시 아래 5가지를 확인하세요:
1. 정확히 무엇을 조사하는지
2. 어떤 부서에서 처리하는지 (부서명)
3. 조사 결과가 실제 보상으로 이어지는지
4. 상담원에게 실행 권한이 있는지
5. 추후 같은 상담원 재연결이 가능한지

> 1~2회 전화로 해결이 안 되면 **상급자 또는 민원실(2차 센터) 연결**을 요청하세요. 민원실이 더 큰 권한을 가집니다.

### 접수 내용

아래 내용을 앱 채팅에 입력하거나, 전화 시 구두로 전달합니다:

> 안녕하세요. 심각한 피해 건으로 공식 접수 요청드립니다.
>
> **주문번호**: [주문번호]
> **제품명**: [샌드위치 정확한 제품명]
>
> 해당 제품 섭취 후 30분 이내에 극심한 구역감, 두통, 복통, 구토 증상이 발생하여 응급 진료를 받았습니다. 담당 의사로부터 추후 설사 발생 시 입원 치료가 필요하다는 소견을 받은 상태입니다.
>
> 특히 구매자는 현재 **임신 중**이며, 담당 의사로부터 **태아에 치명적 영향 가능성**이 있다는 진단을 받았습니다. 수액 처방을 받았습니다.
>
> 쿠팡 주문내역 및 병원 진료기록 보유 중입니다.
>
> 본인(아내)이 현재 몸 상태가 좋지 않아 남편 [이름]이 대리 처리합니다. 이후 모든 연락은 [전화번호]로 부탁드립니다.
>
> 정식 피해보상 절차 및 담당자 연결을 요청드립니다.

위임장 요구 시: 자필 위임장(아내 이름·서명, 남편 이름·연락처, "대리 처리 위임") 사진 첨부면 충분.

## STEP 3. 합의 협상

### 현실적 기대액

| 항목 | 범위 |
|------|------|
| 치료비 실비 | 전액 |
| 위자료 + 임신부 가산 | 200~500만원 |
| **빠른 합의 타겟** | **치료비 + 300만원 전후** |

쿠팡은 임신부 식중독 케이스에서 평판 리스크를 의식합니다. 빠른 합의를 원하는 건 양측 모두 같으므로, 적정선에서 합의가 성립할 가능성이 높습니다.

### 협상 원칙

| 하지 말 것 | 해야 할 것 |
|------------|------------|
| 먼저 금액 언급 | 상대가 먼저 제시하게 유도 |
| 증거 목록 상세 공개 | "진료기록과 주문내역 보유" 정도만 |
| 녹음 없이 전화 합의 | 앱 채팅 우선, 전화 시 반드시 녹음 (기록) |
| 없는 증거 언급 | 있는 증거만 강조, 없는 건 침묵 |

### 진행 흐름

**첫 연락** — 금액 얘기 꺼내지 말 것.

> "현재 아내 상태 지켜보는 중이며, 태아 영향 여부 추가 확인이 필요한 상황입니다."

**금액 제시 요청받으면** — 상대가 먼저 제시하게 유도.

> "치료비 실비 포함, 임신 중 식중독의 정신적 피해와 태아 영향 가능성을 고려한 금액을 먼저 제시해 주시면 검토하겠습니다."

**카운터는 1~2회로 마무리.**

| 상대 제시액 | 대응 |
|-------------|------|
| 100만원 이하 | "임신부 특수성이 반영 안 됨. 재검토 요청" |
| 100~200만원 | "태아 추적관찰 비용과 정신적 피해 감안하면 부족. 300만원으로 협의 원합니다" |
| 300만원 전후 | 태아 청구권 유보 조항 확보 후 수용 검토 |

## STEP 4. 합의서 — 이것만 지키면 됨

### 반드시 포함시킬 문구

> **"본 합의는 현재까지 확인된 산모의 피해에 대한 보상이며, 향후 태아에게 발생할 수 있는 건강상의 후유증 및 이로 인한 손해에 대해서는 별도로 청구할 권리를 유보한다."**

이 조항 없이 서명하면 나중에 태아 문제 생겨도 **추가 청구 불가**. 합의금보다 이 조항이 더 중요합니다.

거부 시: "이 조항 없이는 서명 불가. 소비자원 분쟁조정으로 진행하겠습니다." — 대부분 수용합니다.

### 서명 전 체크리스트

- [ ] **태아 후유증 추가 청구권 유보** 문구 포함
- [ ] **"본 합의로 모든 청구가 종결된다"는 포괄적 종결 문구** 삭제 또는 수정
- [ ] 합의금 지급 기한 명시 (보통 5~10 영업일)
- [ ] NDA(비밀유지) 조항 — 태아 청구권 유보와 충돌 여부 확인

## 합의 불발 시

쿠팡이 성의 없는 금액을 고집하거나 태아 조항을 끝까지 거부하면:

1. **한국소비자원** 피해구제 신청 (consumer.go.kr, 무료) — 기업에 심리적 압박
2. **식약처 1399** 신고 — 행정조사 기록이 협상 레버리지로 활용
3. 그래도 안 되면 **변호사 상담** (법률구조공단 무료 상담 가능)

## 참고

- **실손의료보험**: 치료비 별도 청구 가능. 합의금과 무관
- **배상 청구 근거**: 쿠팡 직매입(로켓배송)이면 판매·유통 책임. 마켓플레이스면 제조사에 병행 청구 가능 (제조물책임법)

## 핵심 요약

1. **오늘**: 진단서 보강(소견서) + 쿠팡 앱 1:1 채팅 클레임 접수 (전화 시 녹음 필수)
2. **협상**: 상대 먼저 제시 → 1~2회 카운터 → 치료비+300만원 전후 타겟
3. **합의서 최우선**: "태아 후유증 추가 청구권 유보" — 합의금보다 중요
4. **기대 기간**: 빠르면 2~3주
