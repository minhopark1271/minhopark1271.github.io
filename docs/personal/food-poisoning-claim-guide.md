# 임신부 식중독 — 배상 실전 가이드
{:.no_toc}

쿠팡에서 구매한 식품으로 식중독이 발생했을 때, 직접 합의 시도 → 합의서 검토 → 소비자원 → 소액심판까지의 전 과정을 정리합니다. 변호사 상담 결과를 반영한 현실적 기대치 포함.

## 상황

- 임신 중인 배우자가 쿠팡에서 주문한 유명 브랜드 샌드위치 섭취
- 섭취 약 30분 후 극심한 구역감, 두통, 복통, 구토
- 설사는 즉시 발현되지 않았으나, **추후 설사 발생 시 입원 치료 필요**하다는 진단
- 수액 처방. **태아에게 치명적일 수 있다는 진단**
- 실물(샌드위치, 포장재) 폐기 상태. 주문내역·진료기록 보유
- **보호자(남편)**: 스트레스 및 불면증으로 **정신과 방문 예정**

## STEP 2. 쿠팡 클레임 접수

### 접수 채널 — 전화만 가능

쿠팡 전문부서(상품 피해 담당)는 **채팅, 문자, 이메일 모두 불가**이며 **유선 통화만** 운영합니다.

> 전문부서에서는 상세한 상담을 위해 채팅 상담을 운영하고 있지 않습니다. 정보 제공 동의 및 제조사 연결 등 여러 절차에 있어 텍스트 상담으로는 어렵기에 양해 말씀을 드립니다. — 쿠팡 전문부서 답변 (2026-03-13)

**기록 확보 전략**:

| 방법 | 설명 |
|------|------|
| **통화 녹음** | 한국법상 통화 당사자가 녹음하는 것은 **상대 동의 없이도 합법** (통신비밀보호법 제3조 제1항 단서). 굳이 허락을 구할 필요 없음 |
| **통화 후 문자 확인** | 통화에서 합의된 내용을 문자로 "방금 통화 내용 정리하면 ~입니다. 맞으시죠?" 형태로 발송 → 기록화 |
| **앱 채팅 병행** | 일반 고객센터 채팅(마이쿠팡 → 고객센터)으로 "전문부서 통화 내용 확인" 요청 → 텍스트 기록 보조 |

### 아이폰 통화 녹음 — 고지 안내 주의

iOS 18 기본 녹음 기능(통화 중 좌측 상단 녹음 버튼)은 **상대에게 "이 통화는 녹음됩니다"라고 자동 안내**가 갑니다. 상대가 경계할 수 있으므로:

| 방법 | 고지 여부 | 방법 |
|------|-----------|------|
| **스피커폰 + 다른 기기 녹음** (추천) | 안내 안 감 | 스피커폰 켜고, 옆에 다른 폰/태블릿/노트북의 **음성 메모** 앱으로 녹음 |
| iOS 18 기본 녹음 | 안내 감 | 통화 중 좌측 상단 녹음 버튼 → 메모 앱에 저장됨 |

> **법적으로는 둘 다 합법**입니다. 한국법상 통화 당사자 녹음은 상대 동의 없이도 합법 (통신비밀보호법 제3조 제1항 단서). 고지 안내는 Apple 정책일 뿐 법적 요건이 아닙니다.

아래 4가지 선택지 중 **가장 유리한 조합**:

| 선택지 | 판단 |

| ~~녹음해도 되냐고 물어보기~~ | ❌ 불필요. 한국법상 당사자 녹음은 동의 불요. 물어보면 상대가 경계만 함 |
| **분쟁으로 가고 싶지 않다** | ✅ 사용. 양측 모두 빠른 합의를 원한다는 공감대 형성 |
| **적절한 보상안 제시해 주시면 검토하겠다** | ✅ 사용. 상대가 먼저 금액을 꺼내게 유도하는 핵심 문장 |

### 대화 스크립트

**오프닝** (차분하고 짧게):

> "안녕하세요. 연락 주셔서 감사합니다. 현재 아내가 임신 중인 상황에서 식중독이 발생해서, 솔직히 저희도 걱정이 많은 상태입니다."

**상황 설명 요청 시**:

> "○○ 샌드위치 섭취 후 약 30분 후 극심한 구토, 복통이 발생했고, 응급 진료에서 수액 처방을 받았습니다. 담당 의사가 임신 중 식중독은 태아에 치명적일 수 있다고 했고, 설사 발생 시 입원이 필요하다는 소견입니다."

**핵심 전달** (반드시 포함):

> "저희도 분쟁 상황으로 가고 싶지 않습니다. 다만 임신 중 식중독이라는 게 일반적인 식중독과는 무게가 다르다 보니, **적절한 보상안을 제시해 주시면 검토하겠습니다.**"

### 방어해야 할 것 — 상대의 예상 수법

#### 1. 인과관계 흔들기 — "정말 우리 제품 때문인지 모르잖아요"

| 상대 | 대응 |
|------|------|
| "인과관계를 확인해야 합니다" | "확인 절차는 진행하시되, **보상 논의도 병행**해 주세요. 저희는 진료기록과 주문내역 보유하고 있습니다" |
| "다른 음식 드신 건 없으세요?" | "**해당 샌드위치 섭취 후 30분 이내에 증상이 발생**했습니다. 시간적 인과관계가 명확합니다" |
| "임신 중이라 입덧이나 다른 원인일 수도..." | "담당 의사가 **식중독 의심 진단**을 내렸고, 수액 처방까지 했습니다. 의료 기록에 명확히 남아 있습니다" |

> **원칙**: 의학적 판단은 의사가 이미 내렸다. 쿠팡 상담원이 의료 소견을 부정하게 두지 않기.

#### 2. 실물 없음 공략 — "제품이 없으면 확인이 어렵다"

| 상대 | 대응 |
|------|------|
| "실물 있으세요?" | "없습니다." (짧게 끝내고 즉시 전환) → "다만 **섭취-발병 30분 이내 + 병원 진료기록**이 있습니다" |
| "실물 없으면 확인이 어렵습니다" | "**쿠팡 쪽에서 유통 기록이나 같은 로트 제품으로 확인하실 수 있지 않나요?** 저희는 주문내역과 진료기록을 갖고 있습니다" |
| "제품 검사를 해야 하는데..." | "같은 로트 제품이 쿠팡이나 제조사에 있을 텐데, **그쪽에서 확인 가능하시잖아요**" |
| 계속 물고 늘어지면 | "실물 여부와 관계없이 **진료기록, 주문내역, 발병 시점**이 확보되어 있고, 소비자원 분쟁조정에서도 충분한 수준입니다. **보상 논의를 진행해 주세요.**" |

> **원칙**: 실물 보관은 소비자 의무가 아니다. 변명하지 말고 있는 증거로 전환. 3회 이상 반복하면 끊기.

#### 3. 시간 끌기 — "확인 후 연락드리겠습니다"

| 상대 | 대응 |
|------|------|
| "제조사 확인 후 연락드리겠습니다" | "확인해 주세요. 다만 **쿠팡 직매입(로켓배송) 상품이면 쿠팡도 유통 책임**이 있습니다. **며칠까지 연락 주실 수 있나요?**" |
| "내부 검토가 필요합니다" | "언제까지 회신 주실 수 있는지 **구체적 일정**을 알려주세요" |
| 기한 없이 "확인되는 대로" | "**3영업일 이내로 회신** 부탁드립니다. 그때까지 연락 없으면 소비자원에 피해구제 접수하겠습니다" |

> **원칙**: "확인 후 연락"은 시간 끌기의 시작. 반드시 구체적 기한을 잡고, 기한 넘기면 소비자원 언급.

#### 4. 책임 떠넘기기 — "저희는 플랫폼일 뿐입니다"

| 상대 | 대응 |
|------|------|
| "제조사에 연락해 보셔야 합니다" | "**로켓배송 상품은 쿠팡이 직접 매입·보관·배송**한 거잖아요. 유통 책임이 있습니다" |
| "저희는 중개만 했습니다" | "주문내역에 **쿠팡 직매입으로 표시**되어 있습니다. 제조물책임법상 유통업자도 책임 주체입니다" |
| "제조사와 협의해서 알려드리겠다" | "쿠팡과 제조사 사이의 문제는 쿠팡 내부에서 정리해 주세요. **저희 창구는 쿠팡**입니다" |

> **원칙**: 로켓배송이면 쿠팡이 유통 책임을 진다. 제조사와의 협의는 쿠팡 내부 문제이지 소비자가 관여할 사안이 아님.

#### 5. 금액 최소화 — "치료비 보전해 드리겠습니다"

| 상대 | 대응 |
|------|------|
| "치료비를 보전해 드리겠습니다" | "치료비는 기본이고, **임신부 식중독의 정신적 피해와 태아 추적관찰 비용**도 고려되어야 합니다" |
| "○○만원 정도 어떠세요?" (저가 제시) | "감사합니다만, **임신 중 식중독의 심각성이 반영된 금액은 아닌 것 같습니다.** 재검토 부탁드립니다" |
| "고객님이 원하시는 금액이 있으세요?" | "저희가 산정하기보다, **쿠팡에서 검토하신 보상안을 먼저 제시**해 주시면 검토하겠습니다" |
| "다른 분들은 보통 이 정도에 합의하십니다" | "다른 사례와 저희 상황은 다릅니다. **임신 중 식중독은 일반 식중독과 동일선상에서 비교하기 어렵습니다**" |

> **원칙**: 절대 먼저 금액을 말하지 않는다. 상대가 제시하면 "감사하지만 부족" → 상대가 올리게 만들기.

#### 6. 빠른 종결 유도 — "빨리 마무리하시죠"

| 상대 | 대응 |
|------|------|
| "합의서 보내드릴 테니 서명해 주세요" | "합의서 내용을 **먼저 검토한 후 연락**드리겠습니다" |
| "오늘 중으로 결정해 주시면 빠르게 처리" | "**중요한 사안이라 검토 시간이 필요합니다.** 며칠 후에 답변드리겠습니다" |
| "이 금액은 오늘까지만 유효합니다" | "시한부 제안은 받아들이기 어렵습니다. **정당한 보상이면 시간에 상관없이 합의 의사**가 있습니다" |
| 전화에서 구두 합의를 유도 | "**구두로 합의하기보다 합의서 문서를 검토한 후** 결정하겠습니다" |

> **원칙**: 전화에서 절대 "합의하겠다"고 말하지 않는다. 합의서를 문서로 받아서 검토하겠다고만 하기.

#### 7. 정보 캐내기 — "혹시 ~하셨나요?"

| 상대 | 대응 |
|------|------|
| "보건소 신고는 하셨나요?" | 했으면 → "네." / 안 했으면 → "**검토 중입니다**" (안 했다고 말하면 레버리지 상실) |
| "변호사 상담은 받으셨나요?" | "**필요하면 진행할 예정입니다**" (있든 없든 이렇게 답) |
| "다른 곳에도 접수하셨나요?" | "**지금은 쿠팡과 직접 해결하고 싶습니다**" (다른 루트의 존재를 암시하면서 협조적 태도) |
| "증상이 지금은 괜찮으세요?" | "아직 경과 관찰 중이고, **태아 영향은 더 지켜봐야 합니다**" (괜찮다고 하면 심각성 감소) |

> **원칙**: 없는 것은 "없다"고 하지 말고 "검토/예정/관찰 중"으로. 있는 것은 핵심만 짧게.

#### 8. 감정 흔들기 — "정말 안타깝습니다"

| 상대 | 대응 |
|------|------|
| 과도한 공감 + "저도 부모로서 마음이..." | 감사 인사 후 본론으로 → "감사합니다. 그래서 **구체적인 보상안**은 어떻게 되나요?" |
| "고객님 입장 충분히 이해합니다" 반복 | "이해해 주셔서 감사합니다. 그러면 **그에 맞는 보상안을 제시**해 주세요" |
| "저도 최대한 도와드리고 싶은데 권한이..." | "권한이 있으신 분과 **직접 통화할 수 있을까요?**" |

> **원칙**: 공감에 감사하되 감정에 끌려가지 않기. 공감 표현이 나올 때마다 "그래서 보상안은?"으로 돌아오기.

### 통화 중 절대 하지 말 것

- ❌ 구체적 금액을 먼저 언급
- ❌ "이 정도면 됩니다" 같은 수용 시그널
- ❌ 증거 목록을 상세히 나열 ("진료기록 보유" 정도만)
- ❌ 감정적으로 흥분 (녹음 중이므로 차분하게)
- ❌ "합의하겠다"고 즉석에서 말하기 → "검토 후 연락드리겠다"
- ❌ "괜찮아졌다" / "별일 아니다" 같은 상태 호전 시그널
- ❌ "보건소 신고 안 했다" / "변호사 없다" 같은 레버리지 포기 발언

### 통화 후 즉시 할 것

1. **녹음 파일 백업** (클라우드에 저장)
2. **통화 내용 정리** → 문자로 담당자에게 발송:
   > "방금 통화 내용 정리합니다. [합의 내용/약속 사항]. 맞으시면 확인 부탁드립니다."
3. 상대가 보상안 제시했으면 → STEP 3으로 이동
4. "확인 후 연락"이면 → **콜백 기한 확인** (3영업일 이내 요청)

위임장 요구 시: 자필 위임장(아내 이름·서명, 남편 이름·연락처, "대리 처리 위임") 사진 첨부면 충분.

## STEP 2-3. 오늘(3/19) CS 관리부서 대응 전략 — 반박 논거 강화

### 전략 요약

쿠팡의 "내부 조사 이상 없음" 주장을 **3단계로 해체**한 뒤, 과거 품질 이력과 구조적 문제로 유통 관리 신뢰성 자체를 흔든다.

```
1단계: 조사 내용 구체적 공개 요구 — 주도권 확보
2단계: 내부 기록·제조사 답변 모두 이해당사자 자기 진술 — 객관성 부정
3단계: 과거 품질 이력 + 구조적 문제 — "완벽한 관리" 전제 붕괴
```

### 반박 논거 — 순서대로 전개

#### 1. 조사 내용 구체적 공개 요구

> 먼저 질문으로 주도권을 잡는다. "이상 없다"는 결론만 전달받았을 뿐, 조사 과정·범위·방법을 소비자가 확인한 것이 없다.

| 대응 |
|------|
| "조사 결과 이상 없다고 하셨는데, **구체적으로 어떤 항목을 어떤 방법으로 조사하셨는지** 알려주실 수 있나요?" |
| "해당 로트의 온도 기록, 프레시백 관리 기록, 제조사 검사 내역 등을 **열람하거나 서면으로 받아볼 수 있나요?**" |
| (답변이 부실하면) "조사 내용을 확인할 수 없는 상태에서 '이상 없다'는 결론만으로는 납득하기 어렵습니다" |

#### 2. 내부 기록·제조사 답변은 이해당사자의 자기 진술

> 쿠팡의 내부 기록도, 제조사의 답변도, 모두 이해관계가 있는 당사자의 자기 판단이다.

**2-a. 내부 기록의 한계**

| 상대 | 대응 |
|------|------|
| "온도 이탈·설비 이상 없었습니다" | "그것은 **쿠팡의 내부 기록에 따른 자체 판단**입니다. 제3자 검증이 이루어진 것이 아니며, 소비자가 그 기록을 신뢰해야 할 의무도 없습니다" |
| "물류센터에서 확인했습니다" | "내부 조사 결과를 부정하는 것이 아닙니다. 다만 **이해당사자의 자기 점검 결과만으로 소비자 피해를 부정하시는 것은 적절하지 않습니다**" |

**2-b. 제조사 답변의 구조적 한계**

| 상대 | 대응 |
|------|------|
| "제조사도 문제없다고 합니다" | "제조사가 문제없다고 답변한 것은 당연합니다. **자사 제품에 문제가 있다고 인정하면 리콜·손해배상·행정처분이 따르기 때문에, 구조적으로 인정할 수가 없습니다.** 제조사가 소비자 클레임 단계에서 자기 책임을 인정한 사례가 극히 드물다는 것은 잘 알려진 사실입니다" |
| "제조사 자체 검사에서도 이상이..." | "이해관계가 있는 당사자의 자기 진술을 객관적 근거로 삼아 소비자 보상을 거부하시는 것은 적절하지 않습니다" |

> **프레임**: "쿠팡 내부 기록도, 제조사 답변도, 모두 이해당사자의 자기 진술입니다. 피해자 입장에서 이를 전적으로 받아들여야 할 이유가 없습니다."

#### 3. 과거 품질 이력 — "완벽한 관리" 전제 붕괴

> 본인의 직접 경험으로 쿠팡 유통 관리가 완벽하지 않음을 증명한다.

**구체적 이력 (쿠팡 반품/환불 시스템에서 확인 가능한 건)**:

| 시기 | 품목 | 문제 |
|------|------|------|
| 2024-08-18 | 대패삼겹살 | 냉동/냉장 보관·배송 문제 |
| 2025-02-16 | 키위 | 품질 문제 |
| 2025-10-22 | 키위 | 품질 문제 |
| 2026-02-02 | 오이 | 변질·짓무름 |

| 상대 | 대응 |
|------|------|
| "유통 관리에 이상 없었습니다" | "저는 로켓프레시를 편리해서 자주 이용하는 고객입니다. 좋은 컨디션으로 배송되는 경우가 많지만, **냉동·냉장 상품에서 부패가 진행되거나 악취가 나는 상품을 받아본 적이 드물지 않습니다.** 반품 처리한 이력이 쿠팡 시스템에도 남아 있을 겁니다" |

> **주의**: 반품/환불 처리한 건 위주로 언급 (쿠팡 시스템에서 확인 가능). 기록 미보관 건(프레시백 오물 등)은 "그 외에도 여러 차례 있었다" 정도로 뭉뚱그려 말하고, 상대가 파고들면 "반품 처리를 안 한 건도 있다" 수준.

#### 4. 구조적 문제 — 쿠팡 시스템의 책임

> 개별 배송기사·작업자·소비자에게 전가할 수 없는 시스템 차원의 문제.

| 상대 | 대응 |
|------|------|
| "배송 후 소비자 보관 문제일 수 있습니다" | "새벽배송 후 아침에 수령해서 바로 섭취했습니다. 이 짧은 시간에 소비자 과실이 있었다는 것은 쿠팡이 입증하셔야 합니다" |
| (구조적 문제 언급 시) | "프레시백 청결 관리, 냉장 배송 품질 등은 **배송기사나 소비자가 아닌 쿠팡 시스템의 구조적 문제**입니다. 무료반품·전액환불 정책은 사후 처리이지, 사전 품질 관리를 대체하지 않습니다. 이 부분은 업계에서도 잘 알려진 사안입니다" |

#### 5. 임산부 감수성 — "다른 클레임 없음" 반박

| 정신과 진료기록 | 남편 불안·수면장애 | 보호자 위자료 근거이나, 법원 인정액 극히 소액 |

### 증거의 약점

**두 증거(카톡, 초진) 모두 결국 "본인이 샌드위치 때문이라고 말했다"는 것의 변형.** 샌드위치 자체가 오염되었다는 객관적 증거는 없음.

| 약점 | 쿠팡 측 반박 |
|------|-------------|
| 검체(실물) 없음 | "식품 오염을 증명할 방법 없음" |
| 1인 단건 | "1,600개 중 다른 클레임 없음" |
| 의사 소견 = "의심" | "확진이 아닌 환자 진술 기반" |
| 내부 조사 "이상 없음" | "유통 관리 정상, 소비자 보관 문제 가능성" |

**입증 책임은 원고(우리 쪽)에 있으며, 현재 증거로는 인과관계 입증이 불확실함.**

### 현실적 기대액 (변호사 상담 반영)

~~기존 합의 타겟 500만원은 비현실적.~~ 한국 법원의 위자료 기준:

| 항목 | 보수적 | 현실적 | 낙관적 |
|------|--------|--------|--------|
| 산모 치료비 실비 | 30만원 | 50만원 | 80만원 |
| 산모 위자료 | 50만원 | 100만원 | 150만원 |
| 남편 정신과 치료비 | 10만원 | 15만원 | 20만원 |
| 남편 위자료 | 0원 | 20만원 | 30만원 |
| 태아 추적관찰 | 0원 | 실비만 | 50만원 |
| **총합계** | **90만원** | **185만원** | **330만원** |

소액심판 갈 경우:

| 시나리오 | 확률 | 결과 |
|----------|------|------|
| 인과관계 불인정 → **패소** | 무시 못 함 | **0원** + 시간·스트레스 낭비 |
| 인과관계 인정 + 보수적 판결 | 가능 | **100~150만원** |

## STEP 3. 합의서 확인 및 태아 유보 조항 요청

> 쿠팡이 20만원 위로금을 제안한 상태. 금액은 수용하되, **합의서 내용이 핵심**.

### 전략

1. 쿠팡에 "합의서를 보내주시면 검토 후 결정하겠습니다" 요청
2. 합의서에 **태아 유보 조항** 삽입 요청
3. 결과에 따라 분기

### 합의서에서 확인할 것

대기업 표준 합의서에는 거의 확실히 아래 문구가 포함됨:

> "본 합의로 본 건에 관한 일체의 민·형사상 청구를 포기하며, 향후 어떠한 이의도 제기하지 않는다"

**이 문구가 있으면 서명 금지.** 태아에게 나중에 문제가 생겨도 추가 청구 불가.

### 태아 유보 조항 — 금액보다 중요

합의서에 아래 문구 삽입 요청:

> **"본 합의는 현재까지 확인된 산모의 피해에 대한 보상이며, 향후 태아에게 발생할 수 있는 건강상의 후유증 및 이로 인한 손해에 대해서는 별도로 청구할 권리를 유보한다."**

**왜 이게 핵심인가**: 태아에게 실제 후유장애가 발생하면 그때는 증거가 명확하고, 청구 금액도 수백~수천만원 규모. 20만원이라는 합의금 자체보다 이 조항이 진짜 보험.

### 분기

| 쿠팡 응답 | 대응 |
|-----------|------|
| 합의서에 태아 유보 조항 넣어줌 | 서명하고 20만원 수령 |
| 포괄적 권리 포기만 고수 | **서명 거부 → 소비자원 접수** |
| 합의서 없이 계좌이체만 | 받아도 됨 (서명한 게 없으므로 권리 포기 아님). 단, 이후 소비자원·소액심판 가능 |

### 서명 전 체크리스트

- [ ] **태아 후유증 추가 청구권 유보** 문구 포함 여부
- [ ] **"본 합의로 모든 청구가 종결된다"는 포괄적 종결 문구** — 있으면 서명 금지
- [ ] 합의금 지급 기한 명시 (보통 5~10 영업일)
- [ ] NDA(비밀유지) 조항 — 태아 청구권 유보와 충돌 여부 확인

## STEP 4. 진실 규명 + 시정 강제 — 소비자원·식약처 활용

> 목표를 **합의금 최대화**에서 **진실 규명 + 문제 시정**으로 전환.
> 쿠팡이 자체 조사 결과 문제 없다고 했고, 유사 클레임이 없다고 했고, 제조사도 문제 없다는 입장.
> 이것이 **제대로 조사된 것인지**, 유사 사례가 **정말 없는지**, 유통/배송 과정에 **실제로 문제가 없는지**를 제3자 행정기관을 통해 검증하고, 문제가 있다면 시정하게 만드는 것이 목표.

### 목표 전환 — 왜 방향을 바꾸는가

| 기존 목표 | 새 목표 |
|-----------|---------|
| 합의금 최대화 (500만원 → 현실 100~200만원) | 진실 규명 + 시정 강제 |
| 쿠팡과의 금전 합의가 핵심 | 쿠팡·제조사의 조사가 제대로 됐는지 **제3자 검증**이 핵심 |
| 식약처 신고 보류 (쿠팡에 유리해질 우려) | 식약처를 **행정조사 권한을 가진 제3자 검증 기관**으로 적극 활용 |

**핵심 질문 3가지**:
1. 쿠팡이 "유사 클레임 없다"고 했는데 → **쿠팡 CS 접수 외 다른 채널로 접수된 유사 사례는?**
2. 쿠팡이 "유통 관리 이상 없다"고 했는데 → **자기가 자기를 조사한 결과를 누가 검증했나?**
3. 제조사가 "문제 없다"고 했는데 → **이해당사자의 자기 진술을 제3자가 확인했나?**

→ 이 3가지를 **행정기관(식약처·소비자원)의 공식 절차**로 검증하게 만드는 것이 이번 단계의 목표.

---

### 4-1. 소비자원 — 피해구제 + 위해정보 동시 접수

소비자원에는 두 개의 별도 채널이 있으며, **동시에 접수** 가능.

| 채널 | 목적 | 핵심 기능 |
|------|------|-----------|
| **피해구제 신청** | 개인 보상 분쟁 조정 | 쿠팡에 합의 권고 (강제력 없음) |
| **위해정보 접수** | 제품 안전성 문제 제보 | **CISS(소비자위해감시시스템)에 등록** → 동일 제품/제조사/유통경로의 기존 신고 이력과 매칭 |

#### 위해정보가 중요한 이유

쿠팡이 "유사 클레임 없다"고 한 것은 **쿠팡 내부 CS 접수 기준**. 소비자원 CISS에는 쿠팡 외 채널(보건소, 직접 접수, 병원 신고 등)로 들어온 유사 사례가 집계되어 있을 수 있음.

**위해정보 접수 시 할 수 있는 것**:
- 해당 제조사(홍루이젠)/유통업자(쿠팡)에 대한 **기존 위해정보 누적 건수** 확인 요청
- "동일 제품 또는 동일 유통경로에 대한 유사 신고가 접수된 적 있는지" 공식 문의
- 일정 건수 이상 누적되면 소비자원이 **자체적으로 안전성 조사** 또는 식약처 이첩

#### 소비자원 접수 방법

| 채널 | 방법 |
|------|------|
| **피해구제** | [소비자24](https://www.consumer.go.kr) → 피해구제 신청, 또는 전화 1372 |
| **위해정보** | [소비자24](https://www.consumer.go.kr) → 위해정보 신고, 또는 전화 1372 |

#### 피해구제 신청 — 접수 후 흐름

```
피해구제 접수
    ↓
소비자원이 쿠팡에 합의 권고 → 쿠팡 거부 가능 (강제력 없음)
    ↓
분쟁조정위원회 회부 → 조정안 제시 → 쿠팡 거부 가능 (강제력 없음)
    ↓
끝. 다음은 법원뿐.
```

**강제력은 없지만 접수하는 이유**:
- 소액심판 갈 때 "성실하게 단계를 밟았다"는 기록
- 소비자원이 쿠팡에 **공식 서면으로 자료 요청** → 내부 CS와 다른 수준의 응대 압력
- 비용 0원, 부담 최소

| 단계 | 나의 행동 | 빈도 |
|------|-----------|------|
| 신청 | 온라인 서식 작성 + 증빙 첨부 | **1회** |
| 보완 | 소비자원이 추가 서류 요청 시 제출 | 0~1회 |
| 중간 확인 | 진행 상황 문의 (안 해도 됨) | 선택 |
| 결과 수령 | 조정안 수용/거부 결정 | **1회** |

**핵심: 신청서 한 번 잘 써서 내면, 그 뒤는 소비자원이 쿠팡과 알아서 주고받음.**

#### 피해구제 신청서 내용

**피해 경위**:

> 2026-03-12 쿠팡 로켓배송(직매입)으로 구매한 홍루이젠 샌드위치를 임신 중인 배우자가 섭취한 후 약 30분 후 극심한 구역감, 복통, 구토가 발생하였습니다. 응급 진료에서 수액 처방을 받았으며, 담당 의사로부터 "임신 중 식중독은 태아에 치명적일 수 있음", "설사 발생 시 입원 치료 필요", "향후 추적관찰 필요"라는 소견을 받았습니다. 보호자인 본인도 이 사건으로 인한 불안·수면장애로 2026-03-14 정신과 진료를 받았습니다.

**합의 시도 경과**:

> 2026-03-13부터 쿠팡 전문부서 및 CS 관리부서와 보상을 협의하였으나, 쿠팡은 "내부 조사 결과 이상 없으며 책임 없다"는 입장을 유지하면서 위로금 20만원만을 제안하였습니다. 2026-03-19 서면으로 합당한 보상안을 요청하였으나, 보상 증액 불가 입장이 확인되어 직접 합의가 결렬되었습니다.

**요청 사항**:

> 산모의 신체적·정신적 피해, 태아 영향에 대한 추적관찰 비용, 보호자의 정신적 피해를 포괄한 합당한 보상(치료비 전액 + 위자료)에 대한 분쟁조정을 요청합니다.

**첨부 자료**:
- 쿠팡 주문내역 스크린샷
- 산모 진료기록/영수증 (응급, 경과관찰)
- 남편 정신과 진료기록/영수증
- 쿠팡과의 서면 교신 내역 (3/19 서면)

### 4-2. 식약처 — 행정조사 권한으로 제3자 검증 강제

#### 기존 "보류" 판단의 재평가

기존에 식약처 신고를 보류한 이유는 "이상 없음 종결 시 쿠팡에 유리"였으나, 목표가 **진실 규명**이면 관점이 달라짐.

| 기존 우려 | 목표 전환 후 재평가 |
|-----------|---------------------|
| "이상 없음" 종결 시 쿠팡에 유리 | 목표가 금전 보상이 아니므로, **조사 과정 자체가 시정 압력**. 식약처가 공문을 보내는 것만으로도 쿠팡·제조사에 관리 감독 신호 |
| 단건이라 역학조사 안 됨 | 맞음. 하지만 **기록은 남음** → 이후 동일 제품 신고가 추가되면 누적 효과 |
| 철회 불가 | 진실 규명이 목표라면 오히려 장점 — 쿠팡이 압력으로 무마할 수 없음 |

#### 식약처가 할 수 있는 것 — 쿠팡·제조사가 거부 불가

| 조치 | 내용 | 강제력 |
|------|------|--------|
| **제조사에 HACCP 기록 제출 요구** | 해당 로트의 제조 공정 기록, 품질검사 결과, 원료 입고 기록 | **있음** (식품위생법) |
| **쿠팡에 유통 온도 기록 제출 요구** | 물류센터 입고~출고~배송 전 과정 온도 이력 | **있음** |
| **해당 제품 관련 기존 신고 이력 확인** | 동일 제조사·제품에 대한 식약처 접수 이력 | **행정 내부 조회** |
| **현장 점검** | 제조시설, 물류센터 점검 (건수/심각도에 따라) | 있음 (통상 단건으로는 드묾) |

**핵심**: 쿠팡이 "내부 조사 결과 이상 없다"고 한 것은 자기가 자기를 조사한 것. 식약처 신고를 하면, **제3자 행정기관이 공식적으로 자료 제출을 요구하고 검증하는 절차**가 작동. 쿠팡의 자체 조사와는 검증 수준이 다름.

#### 식약처 신고 방법

| 채널 | 방법 |
|------|------|
| **온라인** | [식품안전나라](https://www.foodsafetykorea.go.kr) → 신고/제보 |
| **전화** | 1399 |
| **모바일** | 식품안전나라 앱 |

> **주의**: 신고자가 철회해도 식약처 판단으로 조사를 계속할 수 있어 되돌릴 수 없음. 단, 진실 규명이 목표라면 이것은 장점.

### 4-3. 정보공개청구 — 공식 기록 확보

식약처 정보공개청구(정보공개법 기반)로 사전에 확인할 수 있는 것:

| 요청 내용 | 가능 여부 |
|-----------|-----------|
| 홍루이젠(제조사)에 대한 행정처분 이력 | **가능** (공개 대상) |
| 해당 제품에 대한 신고 접수 건수 (개인정보 제외 통계) | **가능** (비식별 통계) |
| 쿠팡 물류센터에 대한 점검/처분 이력 | **가능** |
| 신고자 개인정보, 구체적 신고 내용 | **불가** (개인정보) |

→ [정보공개포털](https://www.open.go.kr)에서 온라인 청구 가능. 처리기한 10일(+10일 연장 가능).

### 4-4. 실행 순서

```
1단계: 소비자원 위해정보 접수 + 피해구제 동시 신청
       → "동일 제품/유통경로 유사 신고 이력 있는지" 문의
       → CISS에 기록 남김

2단계: 식약처 정보공개청구
       → 홍루이젠 행정처분 이력
       → 해당 제품 관련 신고 건수
       → 쿠팡 물류센터 점검/처분 이력

3단계: 식약처 1399 신고
       → 2단계 결과를 보고 결정
       → 기존 처분/신고 이력이 있으면 → 즉시 신고 (누적 효과 극대화)
       → 없으면 → 신고하되 "행정조사를 통한 유통 과정 검증 요청" 취지로
       → 쿠팡의 자체 조사 결과를 제3자가 검증하게 만드는 효과

4단계: 신고 후 식약처에 조사 결과 공유 요청
       → 정보공개청구로 본인 신고건의 처리 결과 확인
```

#### 합의서(STEP 3)와의 관계

위 절차는 합의서 검토(STEP 3)와 **병행 가능**.

| 합의서 결과 | 행정 절차 |
|-------------|-----------|
| 태아 유보 조항 넣어줌 → 서명 | 그래도 진실 규명 목적의 행정 절차는 별도 진행 가능 (합의서가 행정 신고를 금지하지 않는 한) |
| 포괄적 권리 포기만 고수 → 서명 거부 | 행정 절차 + 소액심판 병행 |
| 합의서 없이 계좌이체만 | 받아도 됨 + 행정 절차 진행 (권리 포기 서명한 것이 아니므로) |

> **주의**: 합의서에 "행정기관 신고를 하지 않는다"는 조항이 있으면 서명 금지. 소비자의 행정 신고권은 공익적 성격이므로 이런 조항은 무효 주장 가능하나, 분쟁의 소지를 만들 필요 없음.

## STEP 5. 소액심판 검토

> 강제력 있는 유일한 수단. 정보공개청구·식약처 조사 결과 확인 후 최종 판단.
> **추천 판단 시기**: 2026년 4월 중순 (정보공개 결과 + 소비자원 피해구제 접수 완료 후)

### 절차 타임라인

```
[접수]  소장 작성 → 관할 법원 제출 (인지대 약 5,000~10,000원)
  │
  │    ← 약 1~2주 →
  │
[송달]  법원이 쿠팡에 소장 부본 송달
  │
  │    ← 약 30일 →
  │
[답변]  쿠팡이 답변서 제출 (법무팀 대응)
  │
  │    ← 약 1~2개월 →
  │
[기일]  변론기일 지정 → 법원 출석 (보통 1회로 종결)
  │     - 나: 증거 제출 + 구두 진술
  │     - 쿠팡: 답변서 기반 반박 (변호사 대리 가능)
  │     - 판사: 양측 심리 후 조정 시도 가능
  │
  │    ← 즉일 또는 1~2주 →
  │
[판결]  판결 선고
  │
  │    ← 2주 이내 →
  │
[확정]  항소 없으면 확정 → 쿠팡 지급 의무 발생
```

**총 소요**: 접수 → 판결 약 **2~4개월**. 내 부담: 소장 1회 작성 + 법원 1회 출석(평일). 인지대 약 1만원.

### 핵심 법리 — 제조물책임법 결함 추정 (제3조의2)

이 사건의 법적 구조에서 가장 중요한 조항. 원고가 3가지를 증명하면 **결함이 추정**되고 **입증 책임이 피고(쿠팡)에게 전환**됨.

| 요건 | 이 사건 적용 | 충족 여부 |
|------|-------------|-----------|
| 1. 정상 사용 중 손해 발생 | 샌드위치를 정상적으로 섭취 → 증상 발생 | ✅ |
| 2. 손해 원인이 제조·유통자의 지배영역에 속함 | 로켓배송 직매입 = 쿠팡이 매입·보관·배송 전 과정 관리 | ⚠️ 쿠팡은 "수령 후 소비자 보관" 주장 |
| 3. 결함 없이는 통상 발생하지 않는 손해 | 정상 식품 섭취 30분 후 급성 구토·복통은 통상 발생하지 않음 | ⚠️ 쿠팡은 "다른 원인 가능성" 주장 |

**이 추정이 적용되면**: 쿠팡이 "우리 제품에 결함이 없었다"를 증명해야 함. 원고가 오염을 증명할 필요가 줄어듦.
**현실적 한계**: 검체 없이 이 추정을 적용할지는 판사 재량. 아래 추가 증거가 판사의 판단을 좌우.

### 승소 확률 — 시나리오별 평가

#### 기본 (현재 증거만)

| 시나리오 | 확률 | 결과 |
|----------|------|------|
| 인과관계 불인정 → **패소** | 40~50% | 0원 + 시간·스트레스 |
| 인과관계 인정 + 보수적 판결 | 30~40% | **100~150만원** |
| 인과관계 인정 + 임산부 고려 | 10~20% | **150~250만원** |
| **기대값** | | **약 80만원** |

#### +정보공개·식약처 조사에서 유리한 정황이 나오면

| 발견 | 법적 효과 | 영향도 |
|------|-----------|--------|
| 홍루이젠 기존 행정처분 이력 | "결함 없는 제조사" 전제 붕괴. 판사가 결함 추정 적용할 동기 강화 | **높음** |
| 동일 제품 기존 신고 건수 존재 | 쿠팡의 "유사 클레임 없음" 주장 **직접 반박** | **매우 높음** |
| 쿠팡 물류센터 위생 점검 지적 이력 | "유통 관리 이상 없음" 주장 약화 | **중간~높음** |
| 식약처가 시정명령 등 조치 | 제3자 행정기관이 문제 인정 — 법정에서 매우 강력 | **매우 높음** |

| 시나리오 | 확률 | 결과 |
|----------|------|------|
| 패소 | 25~35% | 0원 |
| 승소 보수적 | 35~40% | **100~150만원** |
| 승소 유리 | 20~30% | **150~250만원** |
| **기대값** | | **약 130만원** |

#### +개인 반품 이력(연 2~3회) + 언론 자료

| 증거 | 판사 인정 가능성 | 쿠팡 반박 |
|------|-----------------|-----------|
| 반품 4건 (대패삼겹살, 키위×2, 오이) | **중간**. 같은 유통 시스템 반복 문제 패턴 | "다른 품목, 다른 시기, 무관" |
| 언론 보도 (프레시백 위생, 물류센터 제재) | **낮음~중간**. 업계 일반 이슈 | "이 건의 직접 증거 아님" |
| 쿠팡 2025년 프레시백 전면 교체 | **중간~높음**. 기존 위생 문제 자인 | "개선했지만 기존도 문제 없었다" (궁색) |

**한계**: 이 증거들은 "쿠팡 유통 관리가 완벽하지 않다는 배경 증거"이지 "이 샌드위치가 오염되었다는 직접 증거"는 아님. 단, 제조물책임법 결함 추정 적용 여부를 판사가 고민할 때 추정을 적용하는 쪽으로 기울게 만드는 역할.

#### 종합 (정보공개 유리 + 반품/언론 모두 제출)

| 시나리오 | 확률 | 결과 |
|----------|------|------|
| 패소 | 20~30% | 0원 |
| 승소 보수적 | 35~40% | **100~150만원** |
| 승소 유리 | 25~35% | **150~250만원** |
| **기대값** | | **약 160만원** |

### 금전 외 가치 — 진실 규명 관점

| 가치 | 설명 |
|------|------|
| **문서제출명령** | 법원을 통해 쿠팡에 내부 조사 자료(온도 기록, HACCP 기록 등) 제출 강제 가능 (판사 재량) |
| **공식 기록** | 판결문은 공적 기록으로 남음 |
| **조정 가능성** | 판사가 변론 전 조정 시도 — 쿠팡이 법정 밖에서보다 유연할 수 있음 |
| **시정 압력** | 소액심판이라도 쿠팡 법무팀 대응 필요. 유사 사건 누적 시 내부 프로세스 개선 압력 |

### 판단 기준

| 조건 | 판단 |
|------|------|
| 정보공개에서 유리한 정황 나옴 | **걸만함** — 증거 보강 + 진실 규명 가치 |
| 정보공개에서 아무것도 안 나옴 | **신중히** — 패소 리스크 상당, 이겨도 금액 크지 않음 |

소비자원 결과를 보고 소액심판 진행 여부를 최종 판단.

- 소비자원 조정 결렬 기록이 "성실한 합의 시도" 증거로 활용
- 소액심판 판결은 **법적 구속력**이 있어 쿠팡이 거부 불가
- 변호사 없이 직접 진행 가능 (소액사건심판법)
