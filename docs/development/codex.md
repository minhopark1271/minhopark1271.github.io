## 사용

- ChatGPT 답변을 VS Code로 옮기기 편함. (VS Code 활성 창 인식)
- 특히, 블로그로 내용 정리할 때

```
ChatGPT에서 프롬프팅하고 학습했던 주요 내용 블로그로 옮기기

>>> 암호화폐 15분봉, 1시간봉 및 여기에 조합하기 적절한 기술적 지표 조합으로 마진거래를 할 때 주문을 닫기에 적절한 시간텀과 TP/SL은?
>>> 위 답변 내용 그대로 마크다운 형태로 파일에 정리해줘. (헤더는 ###까지만 사용, 이모지 제거)
```

- Codex의 Coding Agent로서의 기능은 그닥. 너무 구형 모델 사용한다고 하고, 중요시하는 영역이 아닌듯
