# 기본적 분석 (Fundamental Analysis)
{:.no_toc}

기업의 내재가치를 평가하는 펀더멘털 분석 지표를 성격별로 정리한 가이드입니다. 기술적 분석(차트)이 가격·거래량을 보는 것과 달리, 기본적 분석은 **재무제표와 사업 실적**을 통해 기업의 본질적 가치를 판단합니다.

## 핵심 지표 요약

### TOP 5 핵심 지표

| 지표 | 설명 | 활용 |
|-----|------|------|
| **EPS** | 주당순이익. 기업이 주주 1주당 벌어들인 이익 | 실적의 기본, PER 계산의 기초 |
| **PER** | 주가수익비율. 이익 대비 주가 수준 | 저평가/고평가 판단의 핵심 |
| **ROE** | 자기자본이익률. 주주 돈으로 얼마나 버는가 | 경영 효율성, 복리 성장력 |
| **FCF** | 자유현금흐름. 실제 손에 남는 현금 | "진짜 돈"을 버는지 확인 |
| **부채비율** | 자기자본 대비 부채 수준 | 재무 안정성, 위기 대응력 |

### 주요 지표 +5

| 지표 | 설명 | 활용 |
|-----|------|------|
| **ROIC** | 투하자본이익률. 영업에 투입된 자본의 효율 | 기업의 "질" 판단 (워렌 버핏 선호) |
| **매출 성장률** | 전년/전분기 대비 매출 증가율 | 성장주 판단의 핵심 |
| **영업이익률** | 본업에서의 수익성 | 경쟁력·가격결정력 판단 |
| **EV/EBITDA** | 기업가치 대비 현금창출력 | M&A 밸류에이션, PER 보완 |
| **이자보상배율** | 영업이익으로 이자를 몇 배 갚는가 | 부채 상환 능력 |

## 1. 매출·이익 실적 지표

기업의 **영업 성과**를 직접적으로 보여주는 지표들입니다. 실적 발표 시즌에 가장 주목받습니다.

### 매출 (Revenue / Sales)

- 기업이 제품·서비스를 팔아 얻은 **총 수익**
- 모든 재무분석의 시작점
- "Top Line"이라고도 부름

### 매출총이익 / 매출총이익률 (Gross Profit / GPM)

```
매출총이익 = 매출 - 매출원가(COGS)
매출총이익률(GPM) = 매출총이익 / 매출 × 100
```

- 제품/서비스 자체의 **수익성**
- GPM이 높을수록 가격 결정력, 경쟁력이 강함
- 업종별 편차가 큼 (소프트웨어 70%+ vs 유통 20~30%)

### 영업이익 / 영업이익률 (Operating Income / OPM)

```
영업이익 = 매출총이익 - 판관비(SG&A)
영업이익률(OPM) = 영업이익 / 매출 × 100
```

- **본업**에서 벌어들인 이익
- 회사의 핵심 경쟁력을 나타냄
- 일회성 비용·수익 제외된 "순수 영업 성과"

### EBIT (Earnings Before Interest & Taxes)

```
EBIT = 영업이익 (+ 영업외이익 - 영업외비용)
```

- 이자·세금 차감 전 이익
- 자본구조(부채 수준)와 무관하게 **순수 영업 성과** 비교 가능
- 국가간·기업간 비교에 유용

### EBITDA (EBIT + Depreciation & Amortization)

```
EBITDA = EBIT + 감가상각비 + 무형자산상각비
```

- 현금 유출 없는 비용(감가상각)을 더해 **현금창출력** 근사치
- 설비 집약 산업에서 중요 (제조업, 통신, 인프라)
- 주의: CAPEX를 무시하므로 과대평가 가능

### 순이익 / 순이익률 (Net Income / Net Margin)

```
순이익 = EBIT - 이자비용 - 법인세
순이익률 = 순이익 / 매출 × 100
```

- 모든 비용 차감 후 **최종 이익** ("Bottom Line")
- 주주에게 귀속되는 이익
- 일회성 항목에 영향받을 수 있음

### EPS (Earnings Per Share) / 희석 EPS

```
EPS = 순이익 / 발행주식수
희석 EPS = 순이익 / (발행주식수 + 잠재적 주식)
```

- **주당순이익**: 1주가 벌어들인 이익
- 희석 EPS: 스톡옵션, 전환사채 등 행사 가정
- PER 계산의 기초, 실적 비교의 표준 단위

### 가이던스/컨센서스 대비 (Earnings Surprise)

```
서프라이즈(%) = (실제 EPS - 컨센서스 EPS) / |컨센서스 EPS| × 100
```

- 실적발표 때 **가장 중요한 요소**
- Beat(상회) → 주가 상승 / Miss(하회) → 주가 하락
- 절대 수치보다 "예상 대비" 결과가 주가에 더 큰 영향

## 2. 가치평가 (밸류에이션) 지표

현재 주가가 **비싼지 싼지** 판단하는 지표들입니다.

### PER (Price to Earnings Ratio)

```
PER = 주가 / EPS = 시가총액 / 순이익
```

- **가장 널리 쓰이는** 밸류에이션 지표
- "이익의 몇 배로 거래되는가"
- 낮을수록 저평가 (단, 성장률 고려 필요)
- 업종 평균, 과거 평균과 비교

| PER | 해석 |
|-----|------|
| 10 미만 | 저평가 또는 성장 정체 |
| 10~20 | 적정 수준 |
| 20~30 | 성장주 프리미엄 |
| 30 이상 | 고성장 기대 또는 과열 |

### PBR (Price to Book Ratio)

```
PBR = 주가 / 주당순자산(BPS) = 시가총액 / 자기자본
```

- 장부가치 대비 주가 수준
- **1 미만**이면 "청산가치보다 저렴" (극단적 저평가 또는 문제 기업)
- 금융주, 자산주 분석에 유용
- 기술주는 무형자산이 많아 PBR 높음

### PSR (Price to Sales Ratio)

```
PSR = 주가 / 주당매출 = 시가총액 / 매출
```

- 매출 대비 주가 수준
- **적자 기업** 밸류에이션에 유용 (PER 계산 불가 시)
- SaaS, 바이오 등 초기 성장주 분석에 활용
- 낮을수록 저평가 (업종별 편차 큼)

### PCR (Price to Cash Flow Ratio)

```
PCR = 주가 / 주당영업현금흐름
```

- 현금흐름 대비 주가 수준
- 감가상각 등 비현금 비용의 영향 제거
- PER보다 조작이 어려워 신뢰도 높음

### EV/EBITDA

```
EV(기업가치) = 시가총액 + 순차입금
EV/EBITDA = EV / EBITDA
```

- **M&A 밸류에이션의 표준**
- 자본구조(부채)와 세금 영향 제거
- 업종간 비교에 가장 적합
- 일반적으로 **6~10배**가 적정

### EV/EBIT, EV/Sales

```
EV/EBIT = EV / EBIT
EV/Sales = EV / 매출
```

- EV/EBIT: EBITDA보다 보수적 (감가상각 포함)
- EV/Sales: 적자 기업, 초기 성장주 분석용

### PEG (Price/Earnings to Growth)

```
PEG = PER / EPS 성장률(%)
```

- **성장률 대비 PER** 적정성 판단
- PEG < 1: 저평가 (성장 대비 싸다)
- PEG = 1: 적정
- PEG > 2: 고평가
- 피터 린치가 즐겨 사용

### 배당수익률 / 배당성향

```
배당수익률 = 주당배당금 / 주가 × 100
배당성향(Payout Ratio) = 배당금 / 순이익 × 100
```

- 배당수익률: 투자금 대비 **배당 수익**
- 배당성향: 이익 중 배당 비율 (30~50%가 일반적)
- 배당성향 100% 초과 시 지속 불가능 신호

## 3. 수익성 (효율) 지표

기업이 자원을 얼마나 **효율적으로** 사용하는지 측정합니다.

### ROE (Return on Equity)

```
ROE = 순이익 / 자기자본 × 100
```

- **자기자본이익률**: 주주 돈으로 얼마나 버는가
- 워렌 버핏이 가장 중시하는 지표 중 하나
- **15% 이상**이면 우량 기업
- 단, 부채 레버리지로 높아질 수 있음 → ROIC와 함께 확인

### ROA (Return on Assets)

```
ROA = 순이익 / 총자산 × 100
```

- **총자산이익률**: 전체 자산 활용 효율
- 부채 포함한 전체 자원 효율성
- 은행·금융업 분석에 특히 중요
- ROE보다 보수적인 수익성 지표

### ROIC (Return on Invested Capital)

```
ROIC = NOPAT / 투하자본 × 100
NOPAT = 영업이익 × (1 - 세율)
투하자본 = 자기자본 + 이자부 부채 - 초과현금
```

- **투하자본이익률**: 영업에 투입된 자본의 효율
- 기업의 **"질"**을 판단하는 핵심 지표
- ROIC > WACC(자본비용)이면 가치 창출
- **10% 이상**이면 양호, **15% 이상**이면 우수

> ROI는 정의가 회사/리포트마다 다르게 쓰입니다. 실무에서는 **ROI보다 ROIC/ROE를 더 표준적**으로 사용합니다.

### 마진율 비교

| 지표 | 공식 | 의미 |
|-----|------|------|
| 매출총이익률 | 매출총이익/매출 | 제품 경쟁력, 가격결정력 |
| 영업이익률 | 영업이익/매출 | 본업 수익성 |
| 순이익률 | 순이익/매출 | 최종 수익성 |
| EBITDA 마진 | EBITDA/매출 | 현금창출력 |

### 자산회전율

```
총자산회전율 = 매출 / 총자산
```

- 자산을 얼마나 효율적으로 매출로 전환하는가
- 높을수록 효율적 (유통업 높음, 제조업 낮음)
- ROA = 순이익률 × 자산회전율 (듀폰 분석)

## 4. 재무안정성 (부채/지급능력) 지표

기업의 **재무 건전성**과 위기 대응 능력을 측정합니다.

### 부채비율 / 순부채비율

```
부채비율 = 총부채 / 자기자본 × 100
순부채비율 = (총부채 - 현금성자산) / 자기자본 × 100
```

- **100% 이하**가 일반적으로 안전
- 업종별 편차 큼 (금융·유틸리티는 높음)
- 순부채비율이 더 실질적

### D/E Ratio (Debt to Equity)

```
D/E = 이자부 부채 / 자기자본
```

- 부채비율의 국제 표준 버전
- 이자 발생 부채만 포함 (더 보수적)

### 유동비율 / 당좌비율

```
유동비율 = 유동자산 / 유동부채 × 100
당좌비율 = (유동자산 - 재고자산) / 유동부채 × 100
```

- **단기 지급능력** 측정
- 유동비율 **200% 이상**, 당좌비율 **100% 이상** 권장
- 당좌비율이 더 보수적 (재고 제외)

### 이자보상배율 (Interest Coverage Ratio)

```
이자보상배율 = EBIT / 이자비용
```

- 영업이익으로 이자를 **몇 배** 갚을 수 있는가
- **3배 이상**이면 안전, **1배 미만**이면 위험
- 금리 상승기에 특히 중요

### 순차입금 (Net Debt)

```
순차입금 = 이자부 부채 - 현금 및 현금성자산
```

- 실질적인 부채 부담
- **음수(-)면 무차입 경영** (현금이 부채보다 많음)
- EV 계산에 사용

## 5. 현금흐름 지표 ("진짜 돈" 체크)

회계 이익이 아닌 **실제 현금**의 흐름을 확인합니다. "Cash is King"

### 영업현금흐름 (Operating Cash Flow, OCF)

- 본업에서 창출한 **실제 현금**
- 순이익 + 감가상각 ± 운전자본 변화
- 순이익보다 **조작이 어려움**
- OCF > 순이익이면 이익의 질이 좋음

### 자유현금흐름 (Free Cash Flow, FCF)

```
FCF = 영업현금흐름(OCF) - CAPEX(설비투자)
```

- **배당, 자사주 매입, 부채 상환**에 쓸 수 있는 현금
- 기업가치 평가(DCF)의 핵심
- 지속적인 **양(+)의 FCF**가 중요

### FCF 마진

```
FCF 마진 = FCF / 매출 × 100
```

- 매출 대비 자유현금흐름 비율
- **10% 이상**이면 우수한 현금창출력

### CAPEX (Capital Expenditure)

- **설비투자**: 공장, 장비, 부동산 등
- 성장투자 vs 유지투자 구분 필요
- CAPEX/매출 비율로 투자 강도 파악
- CAPEX 급증 → 향후 성장 기대 or 과잉투자 주의

### 운전자본 변화

```
운전자본 = 매출채권 + 재고자산 - 매입채무
```

- 운전자본 **증가** → 현금 유출 (묶인 돈 증가)
- 운전자본 **감소** → 현금 유입
- 매출 성장 시 운전자본도 증가하는 것이 정상

### 현금전환주기 (CCC, Cash Conversion Cycle)

```
CCC = 재고회전일수 + 매출채권회전일수 - 매입채무회전일수
```

- 원재료 투입 → 현금 회수까지 걸리는 **일수**
- **짧을수록** 현금 효율이 좋음
- 음수(-)면 매우 우수 (아마존, 코스트코)

## 6. 성장성 지표

기업의 **성장 속도와 잠재력**을 측정합니다.

### 매출 성장률 (Revenue Growth)

```
YoY 성장률 = (금년 매출 - 전년 매출) / 전년 매출 × 100
QoQ 성장률 = (금분기 매출 - 전분기 매출) / 전분기 매출 × 100
```

- 가장 기본적인 **성장 지표**
- YoY(전년 대비)가 계절성 제거에 유용
- 성장주: **20% 이상** YoY 성장 기대

### 영업이익 성장률

- 매출 성장과 함께 **수익성도 개선**되는지 확인
- 매출 성장 > 영업이익 성장: 마진 압박
- 영업이익 성장 > 매출 성장: 경영 효율 개선

### EPS 성장률

```
EPS 성장률 = (금년 EPS - 전년 EPS) / 전년 EPS × 100
```

- 주당 이익의 성장
- 자사주 매입 시 주식수 감소로 EPS 상승 가능
- PEG 계산에 사용

### CAGR (Compound Annual Growth Rate)

```
CAGR = (최종값/초기값)^(1/연수) - 1
```

- **연평균 성장률**: 복리 개념의 평균 성장률
- 5년 CAGR로 장기 성장 추세 파악
- 단순 평균보다 정확

### 산업별 KPI (Key Performance Indicators)

업종에 따라 중요한 성장 지표가 다릅니다:

| 업종 | 핵심 KPI |
|-----|---------|
| SaaS | ARR, MRR, 고객이탈률(Churn), LTV/CAC |
| 이커머스 | GMV, 활성고객수, 객단가 |
| 게임 | MAU/DAU, ARPU, 결제율 |
| 통신 | 가입자수, ARPU, 해지율 |
| 제조업 | 수주잔고(Backlog), 가동률 |
| 금융 | 예대마진(NIM), 충당금비율 |

## 7. 주주환원 / 자본정책

기업이 **주주에게 이익을 돌려주는** 방식입니다.

### 배당수익률 / 배당성향

```
배당수익률 = 주당배당금 / 주가 × 100
배당성향 = 총배당금 / 순이익 × 100
```

- 배당수익률: 투자금 대비 현금 수익
- 배당성향: 이익 중 배당 비율
- **배당성향 30~50%**가 지속 가능

### 자사주 매입/소각

- **매입**: 시장에서 자기 주식 구매 → 주가 지지
- **소각**: 매입한 주식 폐기 → 주당가치 상승
- 배당보다 세금 효율적 (양도세 vs 배당세)
- 소각 없는 매입은 효과 제한적

### 총주주수익률 (TSR, Total Shareholder Return)

```
TSR = (기말주가 - 기초주가 + 배당) / 기초주가 × 100
```

- **주가 상승 + 배당**의 총 수익률
- 투자 성과의 종합 지표
- 경영진 보상 기준으로 많이 사용

## 정리

기본적 분석은 **"좋은 기업을 적정 가격에 사는 것"**을 목표로 합니다.

1. **실적**: 돈을 잘 버는가? (매출, 영업이익, EPS)
2. **밸류**: 현재 가격이 적정한가? (PER, PBR, EV/EBITDA)
3. **수익성**: 효율적으로 경영하는가? (ROE, ROIC, 마진)
4. **안정성**: 재무적으로 건전한가? (부채비율, 이자보상배율)
5. **현금**: 진짜 돈을 버는가? (OCF, FCF)
6. **성장**: 앞으로도 성장하는가? (매출/EPS 성장률)

이 지표들을 **업종 평균**, **과거 추세**, **경쟁사**와 비교하여 종합적으로 판단하는 것이 핵심입니다.
