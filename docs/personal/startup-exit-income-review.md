# 스타트업 정리 중 사업소득 수령 가능 여부 검토
{:.no_toc}

코파운더(지분 9.5%, 비등기이사, 직원 신분)로 재직 중인 회사가 파산·폐업 예정인 상황에서, 다른 회사로부터 사업소득자(3.3%) 형태로 급여를 수령할 수 있는지 검토합니다.

## 현재 상황 요약

| 항목 | 내용 |
|------|------|
| 지분율 | 9.5% (희석 후 그 이하) |
| 법적 지위 | 비등기이사, 직원(근로계약) |
| 회사 상태 | 자본잠식, 기보 대출 15억, 폐업(파산) 예정 |
| 투자사 동의 | 모든 투자사 구두 동의 완료 |
| 일정 | 다음 달(2026년 4월경) 파산 절차 시작, 대표 제외 전원 퇴직 처리 |
| 투자계약서 | 주요 이해관계자 대상 경업·겸업 금지 조항 존재 |

## 시나리오 1: 퇴직 전 — 재직 중 사업소득 수령

### 결론: 조건부 가능 (리스크 낮음)

### 근로법 관점

- **근로기준법에 겸업금지 직접 규정 없음** — 취업규칙이나 근로계약에 겸업금지 조항이 있는 경우에만 제한
- 헌법상 직업선택의 자유가 보장되므로, 본업에 지장이 없는 범위에서 겸업은 원칙적으로 허용
- 실무적으로 대부분의 회사는 취업규칙에 겸업금지 조항을 두지만, 위반 시 징계 수준이며 형사 처벌 대상은 아님
- **회사가 폐업 준비 중이라면 겸업금지 조항을 엄격히 적용할 실익이 없음**

### 상법 관점 (주주·이사 의무)

- **비등기이사 → 상법 제397조(경업금지) 직접 적용 대상 아님**
- 경업금지의무는 "등기이사"에게 부과되는 의무이며, 단순 주주나 비등기임원에게는 상법상 직접 적용되지 않음
- 소수주주(9.5%)에게 충실의무나 경업금지의무를 묻는 것은 실무적으로 거의 없음

### 투자계약서상 경업·겸업 금지 조항

투자계약서에 주요 이해관계자(key man) 대상으로 경업금지·겸업금지 조항이 있는 경우:

- **동종 업종이 아닌 경우**: 경업금지는 통상 "동종 또는 유사 업종"에 대한 금지이므로, 다른 업종의 사업소득이라면 해당하지 않을 가능성이 높음
- **겸업 금지가 포함된 경우**: 업종 불문 겸업 자체를 금지하는 조항이라면 형식적으로는 위반 소지 있음
- **실질적 리스크 판단**:
  - 모든 투자사가 폐업에 구두 동의한 상태
  - 자본잠식 상태에서 투자사가 해당 조항 위반을 문제 삼을 실익이 거의 없음
  - 다만 구두 동의만으로는 법적 완전한 면책이 아니므로, **가능하다면 서면 동의(이메일이라도)를 확보**하는 것을 권장

### 세무 관점

- 근로소득(현 회사) + 사업소득(타 회사 3.3%) 동시 발생 → **합법**
- 다음 해 5월 종합소득세 신고 시 합산 신고
- 사업소득 지급 시 3.3% 원천징수 → 종합소득세에서 정산

### 종합 판단

| 쟁점 | 위험도 | 비고 |
|------|--------|------|
| 근로계약상 겸업금지 | 낮음 | 폐업 예정 회사에서 적극 문제 삼을 가능성 극히 낮음 |
| 상법상 경업금지 | 해당 없음 | 비등기이사, 소수주주이므로 적용 안됨 |
| 투자계약서 조항 | 낮음~중간 | 서면 동의 확보 권장 |
| 세무 | 없음 | 종합소득세 합산 신고하면 됨 |

## 시나리오 2: 퇴직 후, 폐업 전 — 사업소득 수령

### 결론: 가능 (실업급여 주의)

### 근로 관계 해소

- 퇴직 처리 완료 → 근로계약 종료 → **취업규칙상 겸업금지 조항 적용 불가**
- 퇴직 후에는 근로자 신분이 아니므로 회사의 인사규정에 구속되지 않음

### 주주 지위 유지

- 퇴직해도 주주 지위는 별도 — 지분(9.5%)은 그대로 유지
- 비등기이사이므로 상법상 경업금지 의무 여전히 해당 없음
- 자본잠식 상태이므로 주식의 실질 가치는 사실상 0원
- 파산 시 잔여재산 분배는 **모든 채권자 변제 후** 진행 → 15억 대출 + 자본잠식 상태에서 주주 배분 가능성 없음

### 투자계약서상 경업·겸업 금지 조항

- 주주 지위가 유지되는 한 형식적으로는 계약상 효력이 존속할 수 있음
- 그러나 **폐업에 모든 투자사가 동의한 상태**에서 경업금지 위반을 주장하는 것은 신의칙(신의성실 원칙)에 반할 가능성이 높음
- 파산 절차 개시 후 법인 청산이 완료되면 투자계약도 실효
- **권장 사항**: 투자사에 퇴직 후 타 회사 용역 제공 예정임을 사전 고지하고, 서면(이메일) 확인을 받아두면 가장 안전

### 실업급여 수급 시 주의사항

회사 폐업에 따른 비자발적 퇴직이므로 **실업급여(구직급여) 수급 자격**이 있을 수 있습니다. 그러나 사업소득 발생 시 주의해야 합니다.

| 항목 | 내용 |
|------|------|
| 수급 자격 | 비자발적 퇴직(회사 폐업) → 자격 충족 가능 |
| 사업소득 신고 | **필수** — 고용센터에 즉시 신고해야 함 |
| 미신고 시 | 부정수급 → 반환 명령 + 과징금(최대 5배) + 형사처벌 가능 |
| 소득 발생 시 영향 | 소액 → 해당일 실업급여 감액, 지속적 고액 → 지급 중단 |

**핵심**: 실업급여를 받으면서 프리랜서 3.3% 소득을 발생시키는 것 자체는 가능하지만, **반드시 신고**해야 하며 소득 규모에 따라 감액·중단될 수 있습니다. 실업급여와 사업소득 중 어느 쪽이 유리한지 비교 후 선택하는 것이 합리적입니다.

### 세무 관점

- 퇴직 후 사업소득만 발생 → 다음 해 5월 종합소득세 신고
- 퇴직 연도에 근로소득 + 사업소득 발생 시 합산 신고

### 종합 판단

| 쟁점 | 위험도 | 비고 |
|------|--------|------|
| 근로계약상 겸업금지 | 해당 없음 | 퇴직으로 근로관계 종료 |
| 상법상 경업금지 | 해당 없음 | 비등기이사이므로 적용 안됨 |
| 투자계약서 조항 | 낮음 | 폐업 동의 상태, 서면 확인 권장 |
| 실업급여 충돌 | 중간 | 사업소득 발생 시 신고 필수, 감액·중단 가능 |
| 세무 | 없음 | 종합소득세 신고하면 됨 |

## 체크리스트

### 퇴직 전 조치

- [ ] 투자계약서 경업·겸업 금지 조항 정확한 범위 확인 (동종 업종만인지, 전 업종인지)
- [ ] 투자사에 사업소득 수령 예정 사실 서면 고지 및 동의 확보
- [ ] 현 회사 근로계약서·취업규칙의 겸업금지 조항 확인
- [ ] 사업소득 지급 회사와 프리랜서 계약서 작성

### 퇴직 시 조치

- [ ] 퇴직 사유 확인서 수령 (회사 폐업에 의한 비자발적 퇴직 명시)
- [ ] 실업급여 신청 여부 결정 (사업소득 규모와 비교)
- [ ] 실업급여 신청 시, 사업소득 발생 사실 고용센터에 즉시 신고

### 폐업·청산 관련

- [ ] 투자사 구두 동의 → 서면(이메일) 확인 전환
- [ ] 퇴직금 수령 확인 (파산 시 임금·퇴직금은 재단채권으로 우선변제)
- [ ] 법인 청산 완료 시 투자계약 실효 확인
